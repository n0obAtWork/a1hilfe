# Meldung Meldebestände wurden unterschritten

<!-- source: https://amic.de/hilfe/meldungmeldebestndewurdenunter.htm -->

Wenn Meldebestände von Artikel unterschritten werden erscheint beim Start von A.eins ein Eintrag (Meldebestände wurden unterschritten) im Hauptmenü unter Favoriten.  
Mit einem Doppelklick auf diese Meldung ruft man einen Report auf, der alle betroffenen Artikel anzeigt.  
Wurde bei einem Artikel sogar der Mindestbestand unterschritten wird der Ist-Wert im Report rot gedruckt.
