# Verbotsliste

<!-- source: https://amic.de/hilfe/_ansch_verbotsliste.htm -->

Diese Funktionen sind exklusiv für das Compliancemodul von A.eins.

\- Verbotslistenprüfung: prüft die ausgewählte Anschrift.

\- Als Good-Guy definieren: Hier kann eine Ausnahme für die Verbotslistenprüfung hinzugefügt werden.

\- Definierte Anschriften prüfen: Abhängig von der Prozedur des SPA 1063 wird hier eine Verbotslistenprüfung ausgeführt.

\- Verbotsliste Ausnahme anzeigen: fall der Datensatz als Good-Guy definiert wurde, kann man die Begründung einsehen.
