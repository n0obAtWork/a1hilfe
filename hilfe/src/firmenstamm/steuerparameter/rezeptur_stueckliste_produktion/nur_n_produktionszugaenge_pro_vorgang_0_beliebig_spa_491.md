# Nur n Produktionszugänge pro Vorgang, 0=beliebig?(SPA 491)

<!-- source: https://amic.de/hilfe/_SPA_491.htm -->

Ein Wert ungleich 0 begrenzt die Anzahl der Produktionszugängen pro Vorgang auf den gewählten Wert. Der Wert 0 steht für eine unbegrenzte Anzahl.

Eine Trennung der Positionen erfolgt nur bei der Neuerfassung einer Produktion.
