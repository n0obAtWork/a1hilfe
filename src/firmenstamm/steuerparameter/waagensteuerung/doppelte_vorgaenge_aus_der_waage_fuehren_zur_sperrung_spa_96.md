# Doppelte Vorgänge aus der Waage führen zur Sperrung(SPA 964)

<!-- source: https://amic.de/hilfe/_SPA_964.htm -->

Wird dieser Steuerparameter auf Ja gestellt, so wird bei der Belegerzeugung abgeprüft, ob es schon ein Vorgang aus dem Waagensatz erzeugt wurde. Ist dies der Fall so werden die Belege werden dann gesperrt.
