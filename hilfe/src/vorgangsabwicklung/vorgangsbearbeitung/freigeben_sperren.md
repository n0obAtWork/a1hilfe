# Freigeben / Sperren

<!-- source: https://amic.de/hilfe/freigebensperren.htm -->

Belege können für bestimmte Bearbeitungsschritte gesperrt bzw. wieder freigegeben werden. Dies kann im Einzelfall für einen noch zu prüfenden Beleg erfolgen, oder aber eine generelle Belegflusssteuerung als Begründung haben. Hierzu ist dann auch die Möglichkeit zu beachten, bei der Formularzuordnung eine generelle Weiterverarbeitungssperre für eine Vorgangsklasse / -unterklasse einzuführen.
