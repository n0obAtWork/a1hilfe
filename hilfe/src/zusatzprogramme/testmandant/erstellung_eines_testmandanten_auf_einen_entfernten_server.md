# Erstellung eines Testmandanten auf einen entfernten Server

<!-- source: https://amic.de/hilfe/erstellungeinestestmandantenau.htm -->

Der Testmandant auf einen entfernten Server kann nur erstellt werden, wenn eine entsprechende Prozedur erstellt und unter Einrichterparameter eingetragen wurde.

Die Prozedur muss folgende Eingabeparameter enthalten.

in_wer: wird ermittelt

in_wohin: Inhalt des Feldes Testdatenbank

in_anwen: Inhalt des Feldes Mailempfänger

in_farbe: Inhalt des Feldes Farbe

Mit „Weiter“ wird der Testmandant auf den entfernten Server erstellt.
