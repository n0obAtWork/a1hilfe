# Dokumentenverwaltung- Multifunktionsleiste

<!-- source: https://amic.de/hilfe/dokumentenverwaltungmultifunkt.htm -->

Das Kontext-Menü und die Multifunktionsleiste ersetzen die sonst übliche Funktionsübersicht in A:eins.

Die Multifunktionsleiste ist in Abschnitte unterteilt, denen im Standard Funktionen wie folgt zugeordnet sind:

| Abschnitt | Funktion/[Tastatur]/Icon | Beschreibung |
| --- | --- | --- |
| **Dokument** | Rücktaste [ESC]<br>![nav\_left\_blue](../../../ImagesExt/image8_819.png "nav_left_blue") | Unterstützung verlassen der Maske auf Touchscreens |
| | Anzeigen [CF12]<br>![view](../../../ImagesExt/image8_820.png "view") | Ruft den externen Viewer auf, den Windows für die Anzeige vorgesehen hat. |
| | Anhänge … [CF11]<br>![](../../../ImagesExt/image8_821.png) | Sucht die zugehörigen Anhänge heraus, und stellt diese in einem extra Fenster dar. |
| | Senden an… [F9]<br>![mail\_forward](../../../ImagesExt/image8_822.png "mail_forward") | Formulararchiv Senden An-Dialog<br>[Archiv Mail Versand](../../archiv_manager/archiv_mail_versand/index.md) |
| | PDF Signieren<br>![](../../../ImagesExt/image8_823.png) | Unterstützt PDF-Signierung durch Signotect-System. |
| | Hinzufügen [F8]<br>![add](../../../ImagesExt/image8_824.png "add") | Formulararchiv-Stammdatenpfleger-Funktion Neu/Einfügen<br>[Archiv – Dokumente hinzufügen](../../archiv_dokumente_hinzufuegen.md) |
| | Drucken<br>![printer2\_pdf](../../../ImagesExt/image8_825.png "printer2_pdf") | PDF-Dokument drucken<br>Übergibt die selektierten Dokumente zum [PDF-Drucken](./pdf_drucken.md). |
| **Funktion** | Stammänderung [F5]<br>![edit](../../../ImagesExt/image8_826.png "edit") | Formulararchiv-Stammdatenpfleger-Funktion Ändern<br>[Archiv-Stammdatenpfleger](../../archiv_stammdatenpfleger.md) |
| | Stammdaten ansehen [F6]<br>![notebook\_view](../../../ImagesExt/image8_827.png "notebook_view") | Formulararchiv-Stammdatenpfleger-Funktion Ansehen<br>[Archiv-Stammdatenpfleger](../../archiv_stammdatenpfleger.md) |
| **Funktion**<br>**Funktionsgruppe Volltext** | Volltext anzeigen<br>![text\_view](../../../ImagesExt/image8_828.png "text_view") | Zeigt den Archivtext in einem Notepad an<br>[Volltext Funktionen](../../archiv_administration/anwendung_formulararchiv/archiv_aendern_ansehen.md) |
| | Volltext aktualisieren<br>![text\_refresh](../../../ImagesExt/image8_829.png "text_refresh") | Aktualisiert den Archivtext<br>[Volltext Funktionen](../../archiv_administration/anwendung_formulararchiv/archiv_aendern_ansehen.md) |
| | Volltext ändern<br>![text\_edit](../../../ImagesExt/image8_830.png "text_edit") | Öffnet den Archivtext in einem Notepad. Der Archivtext kann geändert werden und wird dann gespeichert.<br>[Volltext Funktionen](../../archiv_administration/anwendung_formulararchiv/archiv_aendern_ansehen.md) |
| | Volltext löschen<br>![text\_delete](../../../ImagesExt/image8_831.png "text_delete") | Löscht den Archivtext<br>[Volltext Funktionen](../../archiv_administration/anwendung_formulararchiv/archiv_aendern_ansehen.md) |
| **Funktionsgruppe „Anwendungen“** | OLAP<br>![olap\_neu\_transparent](../../../ImagesExt/image8_832.png "olap_neu_transparent") | |
| | Excel Export<br>![Excel\_32](../../../ImagesExt/image8_833.png "Excel_32") | |
| | Druck Kurzliste<br>![print\_kurzliste32x32](../../../ImagesExt/image8_834.png "print_kurzliste32x32") | |
| | Barcode zuweisen<br>![barcode\_into](../../../ImagesExt/image8_835.png "barcode_into") | Archiv – Barcode zuweisen<br> |
| **Anzeige**<br>**(Funktionsgruppe)** | Ansicht aktualisieren<br>![refresh](../../../ImagesExt/image8_836.png "refresh") | Aktualisiert die Datentabelle, führt also eine neue Datenerhebung durch. |
| | Vorschau<br>![document\_view](../../../ImagesExt/image8_837.png "document_view") | Stellt PDF-Dokumente und Bildelemente des A.eins-Archivs im Vorschau-Bereich dar.<br> <br>Eine detaillierte Aufstellung welche Mimetypen für die Vorschau vorgesehen bzw. implementiert sind findet sich unter [Mimetypen in A.eins](../../technisches_zum_formulararchiv/mimetypen_in_a_eins.md) |

| | Drucken [F10]<br>![printer2](../../../ImagesExt/image8_840.png "printer2") | Formulararchiv drucken<br>Übergibt das Dokument an Windows zum Drucken. | | Ableitung [SF2]<br>![environment](../../../ImagesExt/image8_841.png "environment") | Siehe Ableitung |
| | Ableitung Export<br>![environment\_into](../../../ImagesExt/image8_842.png "environment_into") | Ableitung Export |
| | Zeige Sql<br>![code\_view](../../../ImagesExt/image8_843.png "code_view") | Datengewinnung bzw. Datenbasis für die Datentabelle |
| | Speichern unter | Speichert die selektierten Dokumente in einen vorgebbaren Ordner. Die Dokumente erhalten die Standard-Namen bei Archiv-Export – außer es ist ein Dateiname in den Stammdaten vorgegeben. |
| | Einrichterparameter<br>![gear](../../../ImagesExt/image8_844.png "gear") | z.Z. keine vorgesehen |
| | Dieses Menü<br>![text\_tree](../../../ImagesExt/image8_845.png "text_tree") | Rollen „Dieses Menü“ |
| Allgemein | Hilfe [F1]<br>![help2](../../../ImagesExt/image8_846.png "help2") | Diese Hilfe |

Eine private Icon-Zuordnung ist für spätere A.eins-Releases vorgesehen, momentan werden private Funktionen unter Abschnitt „Funktion“ mit Icon ![user\_new](../../../ImagesExt/image8_847.png "user_new") eingegliedert.
