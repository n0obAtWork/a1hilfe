# Felder bei Gutschein in der Marktkasse (SPA 863)

Hier wird festgelegt welche Felder bei Gutscheinbehandlung in der Marktkasse abgefragt werden sollen.

| Wert | Bedeutung |
| --- | --- |
| 0 | Keine Felder |
| 1 | Nur Gutscheinnummer |
| 2 | Gutscheinnummer und Bemerkungen |
