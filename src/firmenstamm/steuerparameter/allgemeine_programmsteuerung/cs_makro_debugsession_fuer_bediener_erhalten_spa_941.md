# CS-Makro Debugsession für Bediener erhalten (SPA 941)

<!-- source: https://amic.de/hilfe/_SPA_941.htm -->

In diesem SPA können mehrere Bediener definiert werden, deren C#-Makros abhängig vom Bedienerkürzel statt mit der Datenbanksession mit der hinterlegten Debug-Einstellung kompiliert werden. So können Entwickler von Makros nach einem Neustart von A.eins auch weiter eine externe DLL oder einen externen Code debuggen, ohne dies jeweils auf der Makromaske einstellen zu müssen.
