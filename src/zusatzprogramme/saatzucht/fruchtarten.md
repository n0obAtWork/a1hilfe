# Fruchtarten

<!-- source: https://amic.de/hilfe/_pflegerfruchtarten.htm -->

Hauptmenü > Saatzucht > Fruchtarten

Direktsprung **[SGF]**

In diesem Stammdatenpfleger werden die Daten über Fruchtarten gepflegt.

<p class="just-emphasize">Erfassungsmaske</p>

Es stehen folgende Eingabefelder und Eingabemöglichkeiten zur Verfügung.

| Name | Bedeutung |
| --- | --- |
| Fruchtart-Nummer | Hier wird die Identifikationsnummer der Fruchtart eingetragen. Bei Neuanlage wird die nächste freie Nummer vorgeschlagen<br> |
| Kurz-Bezeichnung | Hier wird die Kurzbezeichnung für die Fruchtart eingetragen. Die Kurzbezeichnung darf maximal aus vier Zeichen bestehen.<br> |
| BSS-Fruchtart | **Dieses Maskenfeld enthält die namentliche Bezeichnung für diese Fruchtart. Zu diesem Feld kann mit F3 die Bezeichnung in einer anderen Sprache erfasst werden.**<br> |
| Art + Unterart botanisch | **Die botanische Art + Unterart Bezeichnung dieser Fruchtart kann hier eingetragen werden.**<br> |
| Art / Familie botanisch | **Die botanische Art / Familie Bezeichnung dieser Fruchtart kann hier eingetragen werden.**<br> |
| Lizenz-Abrechnungsart | Mit **F3** stehen folgende Auswahlmöglichkeiten zur Verfügung:<br><br><br>• sortenspezifisch<br>• Betrag pro Anzahl Mengeneinheiten<br>• % vom Nettobetrag<br> |
| Basis-Zuschlag-Abrechnungsart | Mit **F3** stehen folgende Auswahlmöglichkeiten zur Verfügung:<br><br><br>• sortenspezifisch<br>• Betrag pro Anzahl Mengeneinheiten<br>• % vom Nettobetrag<br> |
| Gattung | **Die Gattung dieser Fruchtart kann hier eingetragen werden.**<br> |
| Gewicht in kg/hl | **Das Gewicht in kg/hl dieser Fruchtart kann hier eingetragen werden.**<br> |
| Sortiermerkmal | Hier kann die Sortierreihenfolge für diese Fruchtart innerhalb der Ausdrucke festgelegt werden.<br> |
| Anerkennungspflichtig | **Die Fruchtart kann hier als anerkennungspflichtig gekennzeichnet werden.**<br> |
| Fließeigenschaft | **Die Fließeigenschaft dieser Fruchtart kann hier eingetragen werden. Vorbelegung aus Anwendungsformat AF_FRUFLIES.**<br> |
| Standardmenge | **Die Standardmenge bei Laboranalysen**<br> |
| Standardunters. Menge | **Die Standardmenge bei Laboranalysen für den Bereich Reinheit/Besatz.**<br> |
| Periode AV-Liste | **Die Periode zur AV-Liste dieser Fruchtart kann hier eingetragen werden.**<br> |
| Ist Mischung | **Ist Mischung zu dieser Fruchtart kann hier eingetragen werden.**<br> |

<p class="just-emphasize">Felder auf der Registerkarte Merkmale</p>

Auf dieser Registerkarte können Qualitätsmerkmale dieser Fruchtart gepflegt werden.

| Name | Bedeutung |
| --- | --- |
| Merkmal | Mit der Taste **F3** kann hier eine Auswahl aus den [Qualitätsmerkmalen](./qualitaetsmerkmale.md) abgerufen werden.<br> |

<p class="just-emphasize">Felder auf der Registerkarte Besatzarten</p>

Auf dieser Registerkarte können Besatzarten zur Fruchtart gepflegt werden.

| Name | Bedeutung |
| --- | --- |
| Besatzart | Mit der Taste **F3** kann hier eine Auswahl aus den [Besatzarten](./besatzarten.md) abgerufen werden.<br> |

<p class="just-emphasize">Felder auf der Registerkarte BBCH</p>

Auf dieser Registerkarte kann die BBCH-Codierung und Bezeichnung zur Fruchtart gepflegt werden. Die Codierung kann aus der BBCH-Skala entnommen werden. Eine BBCH-Skala wird unter folgenden Link [Julius Kühn-Institut](https://www.julius-kuehn.de/publikationsreihen-des-jki/bbch-skala/) zur Verfügung gestellt.

| Name | Bedeutung |
| --- | --- |
| Stadium 2 | Hier kann die BBCH-Codierung 2-stellig eingetragen werden. Mit dem Einrichterparameter „Anzeige Stadium 2-stellig“ kann festgelegt werden, ob dieses Feld angezeigt wird. Vorbelegung: **Ja**<br> |
| Stadium 3 | Hier kann die BBCH-Codierung 3-stellig eingetragen werden. Mit dem Einrichterparameter „Anzeige Stadium 3-stellig“ kann festgelegt werden, ob dieses Feld angezeigt wird. Vorbelegung: **Nein**<br> |
| Stadium 4 | Hier kann die BBCH-Codierung 4-stellig eingetragen werden. Mit dem Einrichterparameter „Anzeige Stadium 4-stellig“ kann festgelegt werden, ob dieses Feld angezeigt wird. Vorbelegung: **Nein**<br> |
| Bezeichnung | Hier kann die Bezeichnung entsprechend der BBCH-Codierung eingetragen werden. Mit der Taste **F3** kann hier eine Auswahl aus den bereits verwendeten Bezeichnungen abgerufen werden.<br> |

<p class="just-emphasize">Felder auf der Registerkarte Artenverzeichnis</p>

Auf dieser Registerkarte kann global für die ausgewählte Fruchtart das Kennzeichen Artenverzeichnis für alle Länder die in der [Saatsorten](./anmeldung_zur_feldbesichtigung/vermehrungsvertrag.md) Zuordnung existieren gesetzt werden.

| Name | Bedeutung |
| --- | --- |
| Ländercode | Code des jeweiligen Landes<br> |
| Land | Bezeichnung des Landes<br> |
| Artenverzeichnis | Ist die Fruchtart in dem dazugehörigen Land im Artenverzeichnis<br> |
