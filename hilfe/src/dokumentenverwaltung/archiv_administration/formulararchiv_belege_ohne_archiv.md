# Formulararchiv ( Belege ohne Archiv )

<!-- source: https://amic.de/hilfe/_formulararchivbelege1.htm -->

Hauptmenü > Administration > Archiv > Administration > Formulararchiv ( Belege ohne Archiv )

Direktsprung **[FAAD]**

In dieser Variante werden Archiv-Belege ohne entsprechend auffindbares Dokument aufgelistet.

In System ohne [Container](../archiv_manager/container.md) – Anbindung ist das Indiz dafür dass die Funktion ***Archiv anzeigen*** nicht erfolgreich durchgeführt werden kann!

Bei Systemen mit Container-Anbindung erfolgt die Beschaffung des Dokumentes über private Anpassungen. Es wird in solchen System daher eher die Regel sein, das dann in dieser Variante mehr Einträge aufgelistet werden.

Folgende Möglichkeiten stehen auch hier zur Verfügung: Funktionen der Variante.
