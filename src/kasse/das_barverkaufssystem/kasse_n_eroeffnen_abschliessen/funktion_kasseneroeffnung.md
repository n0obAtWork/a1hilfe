# Funktion: Kasseneröffnung

<!-- source: https://amic.de/hilfe/funktionkassenerffnung.htm -->

Dabei werden die Vortragswerte der Bargeldbestände der letzten Sitzung an dieser Kasse übernommen und die Kasse bekommt eine eigene neue Sitzungsnummer.  
Ebenso wird eine Verbindung zum Display hergestellt, falls es vorhanden ist.

Innerhalb eines Barverkaufssystems können an einer Kasse mehrere Sitzungen abgehalten werden, jedoch ist höchstens eine aktiv. Dies ist auch durch die dem Zustand entsprechenden möglichen Funktionalitäten sichergestellt, es kann keine zweite Sitzung an dieser Kasse aktiv geschaltet werden.
