# Umpacken

<!-- source: https://amic.de/hilfe/_lvs20_umpacken.htm -->

Grundsätzlich wird im A.eins-LVS2.0 nur sortenrein auf einem Ladeträger gelagert. Ausnahme bilden hier nur Ladeträger, die zum Versand an einen Kunden bestimmt sind. Somit ist auf jedem Ladeträger nur Material EINES Artikels und EINER Partie vorhanden.

Nun kann es notwendig sein, Material aus Platzgründen zusammenzufassen und von einem Ladeträger auf einen anderen umzupacken.

**Empfohlener Arbeitsablauf Scanner:**

• Scan „UMPACK“

• Scan der Quell-NVE

o Anzeige der NVE-Info

• Scan der Ziel-NVE

o Prüfung auf Kompatibilität der Inhalte

o Erzeugen einer Umpackung im VIMP
