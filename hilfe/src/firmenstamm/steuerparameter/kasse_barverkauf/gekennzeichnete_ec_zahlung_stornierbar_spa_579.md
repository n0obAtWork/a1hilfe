# Gekennzeichnete EC Zahlung stornierbar(SPA 579)

<!-- source: https://amic.de/hilfe/_SPA_579.htm -->

Gilt in Zusammenhang mit Steuerparameter „Manuelle Erfassung von EC-Karten“, wenn dort „nur Kennzeichnung der Zahlungsart“ eingestellt ist. In diesem Modus wird die EC Karte von A.eins weder eingelesen noch werden die Daten eingegeben. Benutzt man diese Einstellung, um die EC Zahlung per PIN an einem separaten Bankterminal abzuwickeln, so muss die Zahlung gegen Stornierung gesperrt werden, wenn man keine Möglichkeit hat, die online-Zahlung selbst ebenfalls zu rückgängig machen zu können.
