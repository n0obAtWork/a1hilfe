# Default-Mandant

<!-- source: https://amic.de/hilfe/_defaultmandant.htm -->

Damit auf Systemen mit Mandant-Herkunft „Sektion“ eine Archivierung per Datenbank-Funktion amic_fa_set möglich ist, muss dem System hier mitgeteilt werden, welcher Mandant in die Relation Formulararchiv geschrieben wird.
