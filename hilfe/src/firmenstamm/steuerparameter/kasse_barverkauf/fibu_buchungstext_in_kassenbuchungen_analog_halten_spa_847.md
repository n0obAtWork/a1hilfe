# FiBu-Buchungstext in Kassenbuchungen analog halten (SPA 847)

<!-- source: https://amic.de/hilfe/_SPA_847.htm -->

Wird in der Kasse eine Zahlung vorgenommen, so wird der Buchungstext der FiBu mit dem Namen der Kasse identisch sein. Wird dieser Steuerparameter eingeschaltet, so wird der Buchungstext aus dem angegebenen Buchungstext der Kasse übernommen.
