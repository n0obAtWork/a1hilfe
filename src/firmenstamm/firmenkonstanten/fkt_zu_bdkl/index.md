# Zuordnung von Funktionen zu Bedienerklassen (Rollen)

Hauptmenü > Administration > Firmenkonstanten > Zugriffsrechte Rolle

oder Direktsprung **[ROLLE]**

Das Schutzsystem für Funktionen wird über die Zuordnung von Bedienerklassen zu Funktionen geregelt. So steht für jede Funktion fest, ob eine Bedienerklasse diese ausführen darf oder nicht. Da eine Funktion in mehreren Kontexten innerhalb von A.eins verwendet werden kann, ist jeden solchen Vorkommen ein Schutz zugeordnet.
