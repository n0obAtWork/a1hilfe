# Name Sicherheit Login aktivieren (SPA 769)

<!-- source: https://amic.de/hilfe/_SPA_769.htm -->

Hier wird angegeben ob das A.eins-System das Standard-Login-verfahren von A.eins durch die Vorgaben in Name Sicherheit im Bedienerstamm übersteuert. Mit Einstellung dieses SPA’s wird keinem Bediener mehr ein A.eins-Login gestattet, der nicht auch über einen Eintrag in „Name Sicherheit“ verfügt.
