# Listennummer

<!-- source: https://amic.de/hilfe/listennummer.htm -->

Jedem Artikelstapel wird eine Nummer zugeordnet, diese Nummer kann frei vergeben werden, es ist aber sinnvoll, bei kundenspezifischen Artikelstapellisten die Listennummer und die Kundennummer parallel laufen zu lassen, in diesem Fall unterstützt eine ITEM-BOX und eine Automatikfunktion zur Kundenzuordnung die Arbeit.

Die Listennummer 0 hat eine spezielle Bedeutung, diese Liste kann zu Vererbungszwecken herangezogen werden.
