# Mindestanzahl Vorstart Elara für Autostart (SPA 706)

<!-- source: https://amic.de/hilfe/_SPA_706.htm -->

Mit diesem Steuerparameter kann eingestellt werden, wie oft ein Bediener das Hilfsprogramm Elara aufgerufen haben muss, bevor dies beim A.eins-Start automatisch durch eine Veränderung des Eintrags .Net Vorstart im Bedienerstamm vorgestartet wird, um lange Vorlaufzeiten beim ersten Start eines Elara-AddIns zu vermeiden.
