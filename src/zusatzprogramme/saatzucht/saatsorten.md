# Saatsorten

<!-- source: https://amic.de/hilfe/_pflegersaatsorten.htm -->

Hauptmenü > Saatzucht > Saatgutstammdaten > Saatsorten

Direktsprung **[SAATS]**

In diesem Stammdatenpfleger werden die Daten über Saatsorten gepflegt, diese werden einer Fruchtart zugeordnet. Es ist möglich mehrere Saatsorten einer Fruchtart zuzuordnen.

<p class="just-emphasize">Erfassungsmaske</p>

Es stehen folgende Eingabefelder und Eingabemöglichkeiten zur Verfügung.

| Name | Bedeutung |
| --- | --- |
| BSS-Fruchtartnummer<br> | **Hier wird die Identifikationsnummer der Fruchtart eingetragen.** Mit der Taste **F3** kann hier eine Auswahl aus den [Fruchtarten](./fruchtarten.md) abgerufen werden.<br> |
| Sortennummer<br>. | Hier wird die Sortennummer eingetragen. Die Sortennummer ist die numerische Identifikationsnummer der Saatsorte.<br> |
| Sortencode<br> | Hier wird der Kennzeichnungscode der Saatsorte eingetragen.<br> |
| Sortenname<br> | **Der Sortenname enthält die namentliche Bezeichnung für diese Saatsorte.**<br> |
| Sortentyp<br> | Der Sortentyp wird über das Anwenderformat AF_SORTENTYP gepflegt. Eine Auswahl ist mit **F3** möglich.<br> |
| Zu erwartender Ertrag<br> | Ernteschätzung. In der [Feldbearbeitung](./anmeldung_zur_feldbesichtigung/vermehrungsvertrag.md#Feldbearbeitung) wird dieser Wert bei Angabe der Saatsorte als Vorbelegung verwendet. |
| Archiv-Referenz<br> | Formulararchiv Belegreferenz für das Archiv.<br> |
| Bemerkung | Das Bemerkungsfeld für die Saatsorte.<br> |

<p class="just-emphasize">Felder auf der Registerkarte Allgemein</p>

Das Register Allgemein enthält mindestens das Land. Weitere Spalten können individuell in der Anwendung „Griddefinitionen pflegen“ (Direktsprung [GDS]) unter dem Namen „SortenAnmeldedaten_Laender“ eingerichtet werden.

Der Doppelklick auf die markierte Zeile öffnet einen weiteren Dialog, in dem alle Informationen, die auch auf der Registerkarte Allgemein eingerichtet werden können, eingegeben werden können. Hier werden die für das jeweilige Land zutreffenden Daten eingepflegt
