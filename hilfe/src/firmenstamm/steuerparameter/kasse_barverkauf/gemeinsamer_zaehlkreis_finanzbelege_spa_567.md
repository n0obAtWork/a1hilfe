# Gemeinsamer Zählkreis Finanzbelege(SPA 567)

<!-- source: https://amic.de/hilfe/_SPA_567.htm -->

Nein (Standard): Finanzbelege Kasse erhalten je Belegart (Einzahlung, Auszahlung, Einreichung, ...) einen eigenen aufsteigenden Zählkreis. Die Nummer wird fortlaufend je Kasse bestimmt (ohne NK Zuordnung!).

Ja: Es wird nur ein gemeinsamer fortlaufender Zählkreis für alle Finanzbelege einer Kasse geführt.
