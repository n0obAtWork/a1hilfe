# Soll EKZ-Nummer geprüft werden ?(SPA 485)

<!-- source: https://amic.de/hilfe/_SPA_485.htm -->

Hier wird eingestellt, ob während der Artikelerfassung überprüft werden soll ob für den gewählten Artikel eine EKZ-Nummer &lt;> 0 hinterlegt ist.

Wenn die Überprüfung auf „Ja“ gesetzt wurde, werden Artikel ohne Erlöskennziffer schon bei der Erfassung abgewiesen.
