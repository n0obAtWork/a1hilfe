# Start

<!-- source: https://amic.de/hilfe/_dokumenthilfe_start.htm -->

Der A.eins Dokumenten Editor bietet vielfältige Möglichkeiten das Dokument anzupassen.

<details>
<summary>Zwischenablage</summary>

| Funktion | Beschreibung |
| --- | --- |
| Einfügen (STRG +V) | Fügt den Zwischenspeicher in das Dokument ein |
| Ausschneiden (STRG + X) | Schneidet aus dem Dokument aus und schreibt die Daten in den Zwischenspeicher |
| Kopieren (STRG + C) | Kopiert aus dem Dokument und schreibt die Daten in den Zwischenspeicher |

</details>

<details>
<summary>Schriftart:</summary>

| Funktion | Beschreibung |
| --- | --- |
| Schriftart | Setzt die Schriftart des markierten Textes (und dem folgenden) |
| Schriftgröße | Setzt die Schriftgröße des markierten Textes (und dem folgenden) |
| Schriftfarbe | Setzt die Schriftfarbe des markierten Textes (und dem folgenden) |
| Fett | Setzt die Schrifteigenschaft des markierten Textes auf FETT (und dem folgenden) |
| Kursiv | Setzt die Schrifteigenschaft des markierten Textes auf KURSIV (und dem folgenden) |
| Unterstrichen | Setzt die Schrifteigenschaft des markierten Textes auf Unterstrichen (und dem folgenden) |
| Durchgestrichen | Setzt die Schrifteigenschaft des markierten Textes auf Durchgestrichen (und dem folgenden) |
| Hochstellen | Setzt die Schrifteigenschaft des markierten Textes auf Untertext (und dem folgenden) |
| Tiefstellen | Setzt die Schrifteigenschaft des markierten Textes auf Kopftext (und dem folgenden) |
| Text-Typ | Formatiert den Text in das angegebene Textlayout |
| Farbe (Hintergrund) | Setzt die |
| Farbe (Text) | |

</details>

<summary>Bearbeiten:</summary>

| Funktion | Beschreibung |
| --- | --- |
| Suchen… | Sucht den angegeben Text im Dokument |
| Alles markieren | Markiert alle sim Dokument |
| Ersetzen… | Ersetzt den angegebenen Text mit einem neuen |
| Objekt markieren | Ermöglicht die multible selektion von Objekten im Dokument |
