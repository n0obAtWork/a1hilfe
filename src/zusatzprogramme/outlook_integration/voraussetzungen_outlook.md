# Voraussetzungen Outlook

<!-- source: https://amic.de/hilfe/voraussetzungenoutlook.htm -->

Um dieses Modul nutzen zu können, ist ein Outlook 2000 System oder höher notwendig, Mindestvoraussetzung an den Arbeitsplatzrechner ist Windows 2000/2003 oder Windows XP. Das Windows Scripting Host System muss installiert sein (Standard in Windows 2000/2003 und XP).

Um die Rückübermittlung von Daten (z.B. Besuchsberichte) nutzen zu können, sollte auch ein Exchange Server in Betrieb sein, es ist aber auch möglich über externe e-Mail Dienstanbieter wie z.B. gmx oder hotmail eine Datenrückübermittlung zu gewährleisten. In jedem Falle ist ein Internetanschluss notwendig.
