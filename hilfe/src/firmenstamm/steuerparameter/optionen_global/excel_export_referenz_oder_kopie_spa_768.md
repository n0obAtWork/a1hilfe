# Excel-Export Referenz oder Kopie (SPA 768)

<!-- source: https://amic.de/hilfe/_SPA_768.htm -->

Wird dieser Steuerparameter auf Kopie gestellt, so wird aus dem Tabellenblatt „Data01“ in das Tabellenblatt „Tabelle1“ per Kopie in der Zwischenablage kopiert. Bei der Einstellung Referenz werden hier (diese Standardeinstellung geht schneller) Datenreferenzen gesetzt.
