# Löschen von Labordaten

<!-- source: https://amic.de/hilfe/lschenvonlabordaten.htm -->

Wählen Sie einen Laborsatz einer Partie aus, deren Qualitätsdatensatz Sie löschen möchten und wählen Sie die Funktion „Löschen“ an.

Ihnen werden die Werte des Datensatzes angezeigt.

Wenn Sie diesen Datensatz löschen möchten, so beantworten Sie den Dialog mit „Ja“.
