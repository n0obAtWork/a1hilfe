# ImportVorgPosiAddon

<!-- source: https://amic.de/hilfe/importvorgposiaddon.htm -->

Addon Daten

In dieser Relation werden Daten gespeichert, die später in der Tabelle WarenbewegungAddon zur Position hinterlegt werden sollen.

Der Name des gegebenen AddOn-Feldes muss mit dem Feldnamen in der Tabelle übereinstimmen, da sonst keine Daten gespeichert werden können.

Die Verknüpfung zur Position wird über das Feld IVP_GUID hergestellt.

| Feld | Bedeutung |
| --- | --- |
| IVP_GUID | Guid der Position |
| AddonTyp | AddonTyp |
| AddonName | Name des Addonfeldes |
| AddonWert | Inhalt des Feldes |
