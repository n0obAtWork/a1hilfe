# Workflow Verbuchungsregeln Pfleger

<!-- source: https://amic.de/hilfe/_workflowverbuchungsregeln_pfleger.htm -->

| Name | Beschreibung |
| --- | --- |
| Kunde | Kundennummer des Kunden und Name des Kunden |
| Fremdartikel | Alternative Artikelnummer zu einem eigenen Artikel und Bezeichnung des Fremdartikels |
| Gültig ab | Zeitpunkt ab dem die Verbuchungsregel gültig ist. |
| Sachkonto | Eigene Kontonummer (Quelle ist der Kontenplan aus dem Mandantenstamm) |
| Kostenstelle | Nummer der Kostenstelle |
| Kostenobjekt | Nummer des Kostenstellenobjektes |
| Kostenträger | Nummer des Kostenträgers |

Dieser Pfleger ermöglicht die Abbildung eines Fremdartikels, angegeben durch eine Fremdartikelnummer, auf interne Sachkonten, Kostenstellen, Kostenobjekte und Kostenträger.

Diese Abbildungsvorschrift kann durch die Angabe eines Datums im Zeitverlauf geändert werden.
