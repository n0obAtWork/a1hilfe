# DSGVO-Ansicht / DSGVO-Liste

<!-- source: https://amic.de/hilfe/dsgvo_ansicht_liste.htm -->

In den Stammdatenpflegern der DSGVO-Objekte existiert eine Funktion „DSGVO-Liste“. Diese verzweigt in eine Auswahlliste, in der alle relevanten Daten dieses DSGVO-Objekts angezeigt werden. Hier hat man die Möglichkeit über die Funktion „Report bearbeiten“ eine eigene Liste zu erstellen. Dieser Report kann wie alle Reporte [archiviert](../../zusatzprogramme/auswahlliste_2_0/reporte_verwalten.md#ReportArchivieren) werden. Die Anwendung ist so eingerichtet, dass beim Archivieren die Belegklasse, der Belegtyptext, die Kontierung(dsgvo_ident) und ggf. die Kundennummer zum Archiveintrag vermerkt werden.

Nach Beendigung eines Druckes wird in den Anschriften dieses Objekts automatisch vermerkt, dass die Informationen gedruckt wurden.
