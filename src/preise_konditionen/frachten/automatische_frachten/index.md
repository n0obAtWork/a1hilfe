# Automatische Frachten

<!-- source: https://amic.de/hilfe/automatischefrachten.htm -->

Administration > Steuerung > Steuerungsparameter zeigen [SPA]

Eine Fracht errechnet sich aus einer Kombination aus Artikel und Kunden.

Damit automatische Berechnung der Rabatte jedoch erfolgt, muss diese im Steuerungsparameter [184 – automatische Frachten](../../../firmenstamm/steuerparameter/frachten_und_frachtwesen/automatische_frachten_spa_184.md) eingeschaltet sein.
