# Outlook Integration

<!-- source: https://amic.de/hilfe/outlookintegration.htm -->

Neben der Möglichkeit, direkt einen Vorgang von A.eins heraus in eine e-Mail zu verpacken, und diese dann zu versenden, gibt es noch die Außendienstanbindung mit Outlook als "Frontend" Programm.

Es wird per Übergabeprogramm die Kontakttabelle von Outlook mit Daten versorgt, so dass der Außendienstmitarbeiter in seinem Laptop oder seinem Pocket PC die Informationen über seine Kunden ohne direkte Anbindung an die Datenbank, also ohne direkte Verbindung zum A.eins Server, zur Verfügung hat. Es können dem Außendienst alle Informationen der Datenbank zur Verfügung gestellt werden.

Ist auch ein AIP System angeschlossen, so können auch direkt über den e-Mail Weg Daten, wie z.B. Besuchsberichte an den A.eins Server zurück übermittelt werden.
