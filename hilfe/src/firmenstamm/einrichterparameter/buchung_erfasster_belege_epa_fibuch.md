# Buchung erfasster Belege (EPA FIBUCH)

<!-- source: https://amic.de/hilfe/_EPA_FIBUCH.htm -->

| Bezeichnung | Standardwert | Erklärung |
| --- | --- | --- |
| Buchen ohne Mandantenserver | Nein | Steht hier ein „Ja“, so werden die Belege direkt an diesem Arbeitsplatz gebucht, ohne dass der Mandantenserver gestartet sein muss. |
| Bediener darf nur eigene Belege buchen | Nein | Bei Ja kann man nur die Belege Buchen, die man selber erfasst hat. Die Auswahlmöglichkeit auf dem Bildschirm entfällt. |
