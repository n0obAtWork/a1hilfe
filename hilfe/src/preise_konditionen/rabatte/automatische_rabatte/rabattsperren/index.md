# Rabattsperren

<!-- source: https://amic.de/hilfe/rabattsperren.htm -->

Es gibt mehrere Gründe, die dazu führen können, dass ein Rabatt abweichend vom Standard nicht berechnet wird. Zumeist wird dies die geringe Marge sein, die im Einzelfall nicht durch Zugabe bestimmter Rabatte geschmälert werden soll.
