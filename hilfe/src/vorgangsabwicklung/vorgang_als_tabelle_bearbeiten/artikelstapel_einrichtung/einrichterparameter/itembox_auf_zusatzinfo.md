# Itembox auf Zusatzinfo

<!-- source: https://amic.de/hilfe/itemboxaufzusatzinfo.htm -->

Das Zusatzinfofeld kann mit einer Itembox versehen werden, die Itembox kann hier angegeben werden, z.B. kann es sein, dass das Zusatzinfofeld immer das Herkunftsland einer Ware festlegt, dann kann an dieser Stelle die Itembox des Staatstammes IB_STAASTAMM eingetragen werden.
