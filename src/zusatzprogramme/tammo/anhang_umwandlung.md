# Anhang Umwandlung

<!-- source: https://amic.de/hilfe/_tammo_attachmentConversion.htm -->

In den [Steuerparametern](../../firmenstamm/steuerparameter/optionen_global/tammo_einstellungen_spa_933.md) kann eingestellt werden, dass eine Umwandlung der Anhänge in das Format PDF/A (*Format zur Langzeitarchivierung digitaler Dokumente*) erfolgen soll.

Die Dokumente werden zusätzlich zu den ursprünglichen Dokumenten im Archiv gespeichert.

Aktuell werden nur Excel-Dokumente in das Format PDF/A umgewandelt.
