# Ablauf einer permanenten Inventur

<!-- source: https://amic.de/hilfe/ablaufeinerpermanenteninventur.htm -->

- Artikel, die das Kennzeichen „permanente Inventur“ tragen, werden im Verlauf des Jahres gezählt.
- Ist die Zählung abgeschlossen, so wird ein Inventurbestandbeleg (5055) erstellt.
  - Die Warenbestände werden korrigiert
  - Eine Bewertung solle hier erfolgen
  - Im Anschluss erfolgt eine Bearbeitungssperre der Belege
- Die Bestandsfortschreibung findet bis zum Wirtschaftsjahresende mit geeigneten Mitteln statt.
- Die Inventur wird eröffnet.
- Die Bestände der Artikel ohne PIV werden zum Erhebungstag erfasst.
- Die Bestände der Artikel mit PIV werden zum Erhebungstag vorgetragen ([Inventurende-Funktion](../inventur/inventurende.md))
- Am Jahresende wird sichergestellt,
  - dass alle Artikel gezählt wurden
  - dass alle Lagerplätze aufgesucht wurden
  - alle Inventurbelege (5055) gesperrt und in die FiBu übertragen wurden
