# Automatismus, was passiert?

<!-- source: https://amic.de/hilfe/automatismuswaspassiert.htm -->

Bei der automatisierten Vorgangserstellung werden 3 Vorgänge hintereinander generiert bzw. verändert

Es wird ein neuer Vorgang generiert

Daten aus der XML-Datei lesen

Daten prüfen

Vorgangsbearbeitung anstoßen (start_bestellung in bestellung.vbs)

Dem Vorgang wird eine neue Warenposition hinzugefügt

Daten aus der XML-Datei lesen

Daten prüfen

Vorgangsbearbeitung anstoßen (start_bestellung in bestellung.vbs)

Einer Warenposition des Vorgangs wir eine neue Partie hinzugefügt

Daten aus der XML-Datei lesen

Daten prüfen

Vorgangsbearbeitung anstoßen (start_bestellung in bestellung.vbs)
