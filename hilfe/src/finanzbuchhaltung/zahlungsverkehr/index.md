# Zahlungsverkehr

<!-- source: https://amic.de/hilfe/zahlungsverkehr1.htm -->

Der Zahlungsverkehr im Ein- und Ausgang basiert auf verschiedenen Parametern, die unter "Stammdaten" im Abschnitt "Zahlungsverkehr" zu finden sind. Der Zahlungsverkehr selber ist eine Auswertung dieser Parameter (Zahlungsabstand, Anschreiben, etc.) sowie der in den offenen Posten hinterlegten Informationen (Fälligkeit, Betrag, Auslandskennzeichen etc.).

Zum Erstellen von Zahlungen über den automatischen Zahlungsverkehr sind folgende Schritte notwendig:

- Zahlungsvorschläge erstellen über Automatik, OP-Verwaltung oder Zahlmappe
- Zahlungsvorschläge bearbeiten
- Zahlungen bearbeiten

Oder

- Zahlungen erstellen
- Zahlungen bearbeiten
