# Gutschrift aus Rechnung / Sammel- GU aus Rechnung

<!-- source: https://amic.de/hilfe/gutschriftausrechnungsammelgua.htm -->

Erstellt aus selektierten Rechnungen Einzel- oder Sammelgutschriften, die dann in der Anwendung „Gutschriftbearbeitung“ zu finden sind. In der Finanzbuchhaltung erfolgt eine Buchung auf der Gegenseite des Originalbeleges oder, falls über die Buchungsklasse umgesteuert, eine Buchung auf anderen Konten (z.B. Erlösschmälerung).
