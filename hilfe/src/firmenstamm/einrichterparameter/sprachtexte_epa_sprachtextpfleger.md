# Sprachtexte (EPA SPRACHTEXTPFLEGER)

<!-- source: https://amic.de/hilfe/_EPA_SPRACHTEXTPFLEGER.htm -->

| Bezeichnung | Standardwert | Erklärung |
| --- | --- | --- |
| zweite anzuzeigende Sprache | 0 | Es können gleichzeitig mehrere Sprechen angezeigt werden. Sprache 0 ist immer Deutsch. Wenn man nun für eine weitere Sprache (z.B. Polnisch) die Daten erfassen will, so kann es hilfreich sein den Text zusätzlich noch in Englisch oder einer andere Sprache zu sehen |
| dritte anzuzeigende Sprache | 0 | S.o.<br> |
