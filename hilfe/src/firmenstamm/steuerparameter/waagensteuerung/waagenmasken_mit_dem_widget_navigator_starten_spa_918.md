# Waagenmasken mit dem Widget Navigator starten(SPA 918)

<!-- source: https://amic.de/hilfe/_SPA_918.htm -->

Mit diesem Steuerparameter kann eingestellt werden, ob die Waagenmasken mit dem geänderten Layout per Widget Navigator gestartet werden soll. Dazu wird der Steuerparameter 918 auf ja gestellt. Die Waagenmaske wird nun mit dem neuen Überarbeitenden Standard Layout gestartet.
