# Rechnungstrennung durch Sprache(SPA 84)

<!-- source: https://amic.de/hilfe/_SPA_84.htm -->

Trennen: sind mehrere Lieferscheine mit verschiedenen Sprachen markiert und man will diese in eine Sammelrechnung umwandeln, so wird für jede Sprache eine Rechnung erstellt.

Neu: sind mehrere Lieferscheine mit verschiedenen Sprachen markiert und man will diese in eine Sammelrechnung umwandeln, so wird eine Sammelrechnung erstellt und die Warenbewegungen werden der Sprache der Rechnung zugeordnet.

Nein: sind mehrere Lieferscheine mit verschiedenen Sprachen markiert und man will diese in eine Sammelrechnung umwandeln, so wird eine Sammelrechnung erzeugt und die Warenbewegungen werden der Sprache der Lieferscheine zugeordnet.
