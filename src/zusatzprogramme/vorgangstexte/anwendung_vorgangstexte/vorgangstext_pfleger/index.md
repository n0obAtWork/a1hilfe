# Vorgangstext-Pfleger

<!-- source: https://amic.de/hilfe/_vorgtext_pfleger.htm -->

<details>
<summary>Felder des Vorgangstext-Pflegers:</summary>

| Feld | Beschreibung |
| --- | --- |
| Vorgangstextklasse | Wo soll der Text eingesetzt werden:<br><ul><li><b><u>Allgemein verwendbarer Text</u></b>:<br>immer auswählbar</li><li><b><u>Textbaustein</u></b>:<br>z.B. als Textbaustein im Vorgangshauptteil</li><li><b><u>Kopkommentar 1 – 10</u></b>:<br>als Kommentartext im Formularkopf</li><li><b><u>Fußkommentar 1 – 10</u></b>:<br>als Kommentartext im Formularfuß</li><li><b><u>Rohwarentext</u></b>:<br>Text für Rohwarenabrechnung</li><li><b><u>Ihr Zeichen</u></b>:</li><li><b><u>Unser Zeichen</u></b>:</li><li><b><u>Bestelltext</u></b>:</li></ul> |
| Textnummer | ID in der Textklasse (muss eindeutig sein) |
| Bezeichnung | Bezeichnung des Vorgangstextes |
| Textersetzung | Hier wird festgelegt, wie der Text später bearbeitbar ist:<br><ul><li><b><u>Zu ersetzender Text</u></b>:<br>Dieser Text enthält Platzhalter, die im Vorgang durch manuelle Eingabe gefüllt werden</li><li><b><u>Text kopieren</u></b>:<br>Der Text wird vollständig übernommen</li><li><b><u>Nur als Verweis</u></b>:<br>speichert den Text nur einmal und zwar immer den letzten Änderungsstand</li></ul> |
| Maximale Vorgangsstufe | Zu welchem Zeitpunkt in der Vorgangsverabeitung soll der Vorgangstext übernommen werden |
| Vorgangsklasse | Für welche Vorgangsklasse soll der Text anwendbar sein |
| Vorgangsunterklasse | Für welche Vorgangsunterklasse soll der Text anwendbar sein |

</details>

<details>
<summary>Funktionen des Vorgangstext-Pflegers:</summary>

| Funktion | Beschreibung |
| --- | --- |
| Speichern (F9) | Speichert den aktuellen Datensatz ab |
| Text-Zuordnung (F10) | Öffnet den Pfleger der Textbausteine |

</details>
