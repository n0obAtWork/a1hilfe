# Kontraktpartie

<!-- source: https://amic.de/hilfe/_kontraktpartie.htm -->

Ein Kontrakt kann zum Zweck der Erstellung von Andienungen bzw. Freistellungen in Kontraktpartien aufgeteilt werden. Eine solche ist eine Teil- oder Obermenge des oder mehrerer Kontrakte, die im Stück geliefert (bzw. abgerufen) werden soll — eine Liefereinheit sozusagen.
