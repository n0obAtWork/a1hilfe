# Preisfindung mit Objekt-Anfangsdatum(SPA 208)

<!-- source: https://amic.de/hilfe/_SPA_208.htm -->

Bei ‚Ja‘ bezieht sich die Preisfindung auf das Anfangsdatum des Objektes. Unabhängig von diesem Steuerungsparameter wird das Preisbezugsdatum ebenfalls von der Preisgültigkeit im Objektstamm übernommen, sofern dies dort so eingestellt wird.
