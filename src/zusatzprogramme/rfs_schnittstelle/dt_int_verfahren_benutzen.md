# DT- Int Verfahren benutzen

<!-- source: https://amic.de/hilfe/dtintverfahrenbenutzen.htm -->

Das DT-Int Verfahren bietet beim DTA-Austausch erweiterte Möglichkeiten zur Steuerung der Valuta. Es ist jedoch nur auf bankinterne Konten begrenzt. Bei der Erzeugung der DTA Datenträger können keine bankfremden Bewegungen ( Lastschriften / Bankeinzüge ) außerhalb der hauseigenen Bank berücksichtigt werden.
