# Kundenmapping

<!-- source: https://amic.de/hilfe/_kundenmapping.htm -->

Hauptmenü > Stammdatenpflege > Kunden-/Lieferanten > Kundenmapping

In der Kommunikation zwischen Datenaustauschpartnern kann es notwendig sein, ein Mapping für Kundennummern zu machen. So wird im Austausch mit dem ausgewählten Kunden von diesem eine Fremdkundennummer verwendet, die hier eigenen Kunden/Lieferantennummer zugeordnet werden kann.

Diese Daten können zum Beispiel im Vorgangsimport verwendet werden.
