# Kasse/n eröffnen/abschließen

<!-- source: https://amic.de/hilfe/kassenerffnenabschlieen.htm -->

Bevor an einer Kasse einzelne Barvorgänge erfasst werden können, muss diese eröffnet werden (F9 in der Anwendung Gesamtbarverkauf, dann F8).

Für die Kasseneröffnung / Abschluss ist unter Vorgang / Barvorgänge / Kasse Eröffnung/Abschluss ein Pulldown-Menu eingerichtet.

Über das Hauptauswahlmenu gelangt man über Warenwirtschaft / Barvorgänge / Kasseneröffnung Abschluss.

Je nach Status der Kasse stehen die Abschluss- oder die Eröffnungsfunktionen zur Verfügung.

So wird der Status der Kasse (eröffnet / abgeschlossen / unterbrochen) leicht erkennbar.
