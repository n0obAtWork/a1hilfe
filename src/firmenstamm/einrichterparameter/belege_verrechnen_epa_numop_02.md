# Belege verrechnen (EPA NUMOP_02)

<!-- source: https://amic.de/hilfe/_EPA_NUMOP_02.htm -->

| Bezeichnung | Standardwert | Erklärung |
| --- | --- | --- |
| Belegdatum und Periode prüfen | Warnung | <ul><li>&nbsp;&nbsp;&nbsp; <u>Ignorieren</u>. Es findet kein Test statt.</li><li>&nbsp;&nbsp;&nbsp; <u>Fehler</u>. Man kann nur weiterarbeiten, wenn Datum und Periode übereinstimmen.</li><li>&nbsp;&nbsp;&nbsp; <u>Warnung</u>. Datum und Periode werden gegeneinander geprüft und es wird eine Warnung ausgegeben. Es kann trotzdem weitergearbeitet werden.</li></ul> |
| Bei Ausbuchung Steuerschüssel aus Sachkontenstamm vorbelegen? | Nein | Steht hier ein „Ja“, so wird der Steuerschlüssel vorbelegt. |
| Vorbelegung Bankkonto | 0 | Bei Zahlung Bank wird dieses Konto vorbelegt. |
| Vorbelegung Kassenkonto | 0 | Bei Zahlung Kasse wird dieses Konto vorbelegt. |
