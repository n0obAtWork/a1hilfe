# Schritt für Schritt

<!-- source: https://amic.de/hilfe/_vorgangstexte_sfs.htm -->

#### Textbausteine im Vorgang:

<details>
<summary>Schritt 1: Textbaustein anlegen</summary>

Direktsprung [TBS] -> Neu (F8) -> Feld: Vorgangstextklasse -> F3-Auswahl -> „Textbaustein“ -> Feld: Vorgangstextnummer -> neue ID, welche noch nicht vergeben ist eintragen-> Text-Zuordnung (F10) -> hier kann nun der gewünschte Text eingetragen werden -> Speichern (F9) -> Speichern (F9)

</details>

<details>
<summary>Schritt 2: Textbaustein im Vorgang hinzufügen</summary>

[REE] -> Feld: Kunde -> F3-Auswahl -> Kunde auswählen -> Positionen (F5) -> Textbaustein hinzufügen (Shift + F8) -> in der F3 Auswahl den in Schritt 1 erstellten Textbaustein auswählen -> Verlassen (ESC) -> Abschluss

</details>

<details>
<summary>Schritt 3: Vorgang drucken</summary>

[REB] -> erstellten Vorgang suchen -> Formulardruck (F10) oder Vorschau (F11)

</details>
