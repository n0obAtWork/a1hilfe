# Waagenterminal Übersicht

<!-- source: https://amic.de/hilfe/_waageprofil_uebersicht.htm -->

Folgende Meilensteine sind verwirklicht:

• COM-Port-IO-Routinen

• Anbindung von Systemen über UDP-Protokoll

• Anbindung von Systemen über Hersteller-DLL’s

• Ein Waagenterminal beschreibt alle zum Betrieb notwendigen technischen Daten zum Betrieb an einem Aeins-Host.

• AMIC bekannte Waagenprofile werden mit Aeins ausgeliefert. Durch einen Kopiervorgang können diese Schablonen angepasst und aktiviert werden

Die Waagenterminals werden über die Anwendung „Waagenterminal“ per Direktsprung „WAM“ administriert.
