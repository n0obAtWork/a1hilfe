# Liefernummer auf Position eingeben (SPA 826)

<!-- source: https://amic.de/hilfe/_SPA_826.htm -->

| Einstellungen |
| --- |
| Nein | Diese Einstellung bietet keinerlei Erfassungsmöglichkeiten für Lieferscheinnummern auf der Warenposition |
| Immer | Bei Einstellung „Immer“ bietet sich auf der Warenpositionsmaske die Möglichkeit die Nummer des zugehörigen Lieferscheins (sofern dies in der Unterklasse zugelassen ist manuell zu erfassen.  
 |
| Nur bei fehlendem Lieferschein | Bei dieser Einstellung wird die Möglichkeit zur Erfassung einer Liefernummer nur dann freigegeben, wenn dies in der Unterklasse freigegeben wurde und es zu dieser Positionszeile keine Positionszeile eines Lieferscheins in A.eins gibt. So sind Manipulationen bestehender belege ausgeschlossen. Das Eingabefeld wird nur bei direkter Erfassung eines Beleges freigegeben. |
