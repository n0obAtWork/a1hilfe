# EK Gutschrift

<!-- source: https://amic.de/hilfe/ekgutschrift.htm -->

EK-Gutschriften entstehen durch Neuerfassung oder Umwandlung aus EK-Rechnungen. EK-Gutschriften werden als Vorgang gespeichert; die Menge wird verbucht und vermindert den Bestand; der EK-Umsatz vermindert sich. A.eins stellt folgende Bearbeitungsfunktionen zur Verfügung:

| Funktionsname | Funktion |
| --- | --- |
| ***Erfassen F 8*** | Erfassung einer neuen EK-Gutschrift<br> |
| ***Stapelverarbeitung*** | Übernahme einer oder mehrerer EK-Gutschriften in einen Bearbeitungsstapel<br> |
| ***Erstdruck F9*** | Erstdruck einer EK-Gutschrift<br> |
| ***Formulardruck F10*** | Wiederholungsdruck<br> |
| ***Lieblingsdruckerdruck*** | Auswahl eines anderen Druckers / Zuordnung eines anderen Formulars<br> |
| ***Korrektur F5*** | Korrektur einer EK-Gutschrift<br> |
| ***Ansicht F6*** | EK-Gutschrift im Ansicht-Modus öffnen<br> |
| ***Kopieren CF8*** | Kopieren der EK-Gutschrift für einen auszuwählenden Kunden<br> |
| ***Vorschau F11*** | Druckvorschau<br> |
| ***Stornieren F7*** | Stornieren (Löschen) der EK-Gutschrift<br> |
| ***Freigabe/Sperren*** | Freigabe / Sperren für weitere Bearbeitung<br> |
| ***Archiv ansehen*** | Anzeige archivierter Vorgänge<br> |
| ***Partien nachtragen*** | nachträgliche Zuordnung von Partien<br> |
| ***Wiedervorlage CF9*** | Vorgang mit einem Wiedervorlagevermerk versehen<br> |
| ***Arbeitsregel ändern*** | manuelle Änderung von Weiterverarbeitungsparametern<br> |
| ***FiBu Übertrag*** | Übergabe an die Finanzbuchhaltung<br> |
