# Kontrakt-Report-Zu-/Abschlag: Bezugsdatum (SPA 1002)

<!-- source: https://amic.de/hilfe/_SPA_1002.htm -->

Hier wird das Bezugsdatum zur Berechnung der im Kontrakt festgelegten zeitabhängigen Reports festgelegt. Dieses kann das Preisdatum oder das Lieferdatum, bei Aufträgen entsprechend das Planlieferdatum, sein.

Hinweis: Diese Einstellung gilt nicht für Rohwarebelege, hier gilt IMMER das Lieferdatum als Bezugsdatum!
