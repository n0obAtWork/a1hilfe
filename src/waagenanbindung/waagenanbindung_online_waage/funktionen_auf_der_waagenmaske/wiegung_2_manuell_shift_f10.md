# Wiegung 2 manuell SHIFT + F10

<!-- source: https://amic.de/hilfe/_waage_2tewiegung_manuell.htm -->

Man kann die 2. Wiegung manuell eingeben.  
Das ist dann hilfreich, wenn die Waage intakt ist, die Verbindung zum Computer aber Probleme bereitet, so dass die Wiegung nicht automatisch ausgeführt werden kann.

Wurde im Terminal auf der Registerkarte Vorgang / Zugang die Maximal-, Minimal- und das Taktungsgewicht angegeben, so wird das hier erfasste Gewicht auf Plausibilität geprüft.

D.h. Es kann nur ein Gewicht eingegeben, welches auch von der Waage geliefert wird.

z.B.

Maximal Gewicht 40 t

Minimal Gewicht 1 t

Taktgewicht 0,002 t

So wäre eine Eingabe von 1,233 Tonnen, sowie 41Tonnen und 0,5 Tonnen nicht möglich.
