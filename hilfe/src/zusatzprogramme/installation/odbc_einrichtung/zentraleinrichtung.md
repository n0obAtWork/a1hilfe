# Zentraleinrichtung

<!-- source: https://amic.de/hilfe/zentraleinrichtung.htm -->

Der Setup Prozess prüft bei der Installation, ob eine ODBC-Setup der Microsoft Basisroutinen notwendig ist, oder nicht. Ist so ein Basissetup notwendig, dann wird die Microsoft ODBC Setup Routine gestartet, die nur einmal einfach durchlaufen werden muss.
