# Hofliste-Details

<!-- source: https://amic.de/hilfe/_silohoflistedetails.htm -->

Diese Funktion kann nur ausgeführt werden, wenn ein Datensatz in der jeweiligen Auswahlliste markiert worden ist. Die Funktion ruft die Auswahlliste [Hofliste-Details](../../../waagenanbindung/hofliste_details.md) auf.

Wird diese Funktion in der Variante Silo ausgewählt, so werden zu der angewählten Siloposition alle dazu gehörigen Waagenbelege angezeigt.

Wird diese Funktion in der Variante Silobestand ausgewählt, so werden alle Waagenbelege zu diesem Silo angezeigt.
