# Lieferscheinnummer

<!-- source: https://amic.de/hilfe/_liefnoinwapos.htm -->

Ist der Steuerparameter [826 – Liefernummer auf Position eingeben](../../../firmenstamm/steuerparameter/vorgangsbearbeitung_positionen/liefernummer_auf_position_eingeben_spa_826.md) eingeschaltet und das Eingabefeld in der [Formularzuordnung](../../formularzuordnung/eingabe_eingabefelder.md) freigeschaltet, so kann auf der Warenposition ab Stufe Rechnung eine Lieferscheinnummer manuell erfasst werden.

Es erfolgt keine Prüfung der Lieferscheinnummer auf Plausibilität oder Existenz.
