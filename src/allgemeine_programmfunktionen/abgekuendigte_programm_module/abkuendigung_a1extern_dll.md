# Abkündigung: A1extern.dll

<!-- source: https://amic.de/hilfe/_ab32038.htm -->

Die A1extern.dll ist ein Programmierinterface, welches hauptsächlich von unseren Systemhäusern verwendet wurden. Dies betrifft auch Entwicklungen, die seinerzeit von der GS Computersysteme GmbH in Pleidelsheim mit Programmierwerkzeugen wie Delphi getätigt wurden. Vielen Anwendern kennen solche Lösungen unter dem Begriff Kratochwil-Anwendungen. Der Einsatz dieser Lösungen ist mit der 64-Bit Version nicht mehr möglich. Sollten Sie wissentlich solche Lösungen einsetzen, sprechen Sie uns an. Im Rahmen des Updates werden wir solche Funktionalitäten prüfen. Ein Update auf die 64-Bit Version ist nur möglich, wenn diese Individualentwicklungen auf neuere Technologien umgesetzt wurden.Zu erkennen ist dies, in dem Sie unter MAKRO nach der A1extern.dll suchen:[MAKRO] -> (F2) -> Makroname: "%a1extern.dll%"Wenn hier einige MAKROS erscheinen, besteht das Risiko, dass Sie von der Abkündigung betroffen sind. In dem Fall vereinbaren Sie bitte einen Termin mit einem Servicemitarbeiter, der mit Ihnen gemeinsam erörtert, ob diese Funktionalitäten von Ihnen verwendet werden.
