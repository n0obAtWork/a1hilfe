# Verbotslisten-Übersicht

<!-- source: https://amic.de/hilfe/complianceuebersicht.htm -->

Hauptmenü > Externe Kommunikation > Webservice > Verbotsliste

Diese Auswahlliste dient zur Anzeige der Prüfaufträge, die aus Anschriften, Kunden und Vorgängen entstehen können.

### Felder

| Felder | Beschreibung |
| --- | --- |
| Prüfung | <ul><li><strong><em>Vorgang:</em></strong><br>Die Prüfung ist aus einem Vorgang initiiert worden.</li><li><strong><em>Kunde:</em></strong><br>Die Prüfung ist aus einem Kunden initiiert worden</li><li><strong><em>Anschrift</em></strong><br>Die Prüfung ist aus der Anschrift initiiert worden</li></ul> |
| Kunde | ***Kundennummer*** |
| Vorgang | Vorgangsnummer (wenn vorhanden) |
| Zeitstempel | ***Zeitstempel des Eintrags*** |
| Status | Ergebnis der Prüfung<br> <br><ul><li><strong><em>ungeprüft:</em></strong><br>Die Prüfung ist noch nicht erfolgt</li><li><strong><em>zulässig</em></strong><br>Die Prüfung ist erfolgt und die Anschrift ist als zulässig gewertet worden</li><li><strong><em>unzulässig</em></strong><br>Die Prüfung ist erfolgt und die Anschrift ist als unzulässig gewertet worden</li></ul> |
| Quelle | ***Anschrift aus dem Vorgang***<br> <br><ul><li><strong><em>Kunde:</em></strong><br>Kundenanschrift</li><li><strong><em>Rechnung</em></strong><br>Rechnungsanschrift aus dem Vorgang</li><li><strong><em>Zahlungspflichtiger</em></strong><br>Zahlungspflichtiger aus dem Vorgang</li><li><strong><em>Versandanschrift</em></strong><br>Versandanschrift aus dem Vorgang</li><li><strong><em>-&nbsp;&nbsp;&nbsp;&nbsp; Andere</em></strong><br><strong><em>Weitere Anschriften aus einer </em></strong><a href="./schritt_fuer_schritt_anleitung/schritt_1_setup.md#SPA1063">Datenbankprozedur</a><strong><em>, die Anschriften aus dem Vorgang bzw. in dessen Umfeld hinzuliest.</em></strong><strong></strong></li></ul> |
| AdressId | AdressId der Anschrift |
| Name | ***Name*** |
| Straße | Straße |
| PLZ | ***Postleitzahl*** |
| Ort | Ort |
| Prüfauftrag | ***GUID des Prüfauftrags für Datenbankrecherchen*** |
