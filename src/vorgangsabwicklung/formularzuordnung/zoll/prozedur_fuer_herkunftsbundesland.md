# Prozedur für Herkunftsbundesland

<!-- source: https://amic.de/hilfe/prozedurfrherkunftsbundesland.htm -->

In dieses Feld kann eine private Prozedur eingetragen werden, welche die Herkunft für die, in einer Ausfuhr enthaltenen, Positionen vorbelegt. Als Standard wird, über die Funktion Zoll_Herkunftsbundesland.sql, die beim Anlegen, des zur Ausfuhr zugehörigen Lieferscheins, definierte Herkunft übernommen.
