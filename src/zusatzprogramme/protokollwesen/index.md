# Protokollwesen

<!-- source: https://amic.de/hilfe/protokollwesen.htm -->

Direktsprung [PW]

Im Fehlerprotokoll werden Systemhinweise gesammelt. Im Zusammenhang zum Beispiel mit Archiv-Import-Problemen wird auf weitere Informationen auf Protokolleinträge in dieser Anwendung verwiesen.
