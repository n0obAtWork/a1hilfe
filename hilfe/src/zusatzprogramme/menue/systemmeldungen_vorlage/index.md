# Systemmeldungen - Vorlage

<!-- source: https://amic.de/hilfe/systemmeldungenvorlage.htm -->

Administration > Menü > Systemmeldungen (Vorlagen)

oder Direktsprung [**MENUD**]

Die Systemmeldungs-Vorlagen können für die Erstellung einer Systemmeldung dienen.

(siehe [Neu aus Vorlage](../systemmeldungen/index.md#NeuAusTemplate))

| **Felder** | |
| --- | --- |
| Name | Eindeutiger technischer Name der Systemmeldung. |
| Auslieferung | Ja/Nein<br>Kennzeichen, ob die Systemmeldungs-Vorlage in der Auslieferung ist. |
| Beschriftung | Die explizite Beschriftung der Systemmeldung.<br>Hinweis: Es handelt sich hierbei nicht um die Beschriftung der Funktion. |
| Funktion | Die Funktion, die ausgeführt werden soll, wenn ein User die Systemmeldung klickt. |
| Funktionsart | Informatorisch die Funktionsart. |
| Statement | Das zugrundeliegende Systemmeldungs-Statement.<br>(siehe auch [Systemmeldungstatement](../systemmeldungen/systemmeldungspfleger.md#Systemmeldungstatement))<br>Achtung das tatsächlich gespeicherte Statement kann wesentlich länger als die hier angezeigten maximalen 255 Zeichen sein! |

| **Funktionen** | |
| --- | --- |
| Pflege-Funktionen<br>[**F8**],[**F5**],[**F6**],[**F7**] | Neu, Ändern, Ansehen, Löschen<br>Außer „Ansehen“ ausschließlich für Entwicklung. |
| Funktionen ansehen/bearbeiten [**F11**] | Ruft den Anseh- bzw. Pflegedialog für die angegebene Funktion der Systemmeldung direkt auf. |
| Funktion Informationen [**F9**] | Ruft den Funktions-Informationsdialog für die angegebene Funktion der Systemmeldung auf. |

| **Suchen** | |
| --- | --- |
| Name | Name like |
| Funktion | Funktion like |
| Beschriftung | Beschriftung like |
| Statement | Statement like |
| Auslieferung | Ja,Nein;Egal |
