# Rabattsperre durch Aktionspreis (SPA 310)

<!-- source: https://amic.de/hilfe/rabattsperredurchaktionspreiss.htm -->

Administration > Steuerung > Steuerungsparameter zeigen

Oder Direktsprung **[SPA]**

Wird ein Aktionspreis verwendet, so ist unter Umständen die Marge so gering, dass ein zusätzlicher Rabatt hier nicht gewünscht wird.

Aus diesem Grunde gibt es mit Steuerparameter [310 - Zu-/Abschläge/Rabatte auf Aktionspreise](../../../../firmenstamm/steuerparameter/preisfindung/zu_abschlaege_rabatte_auf_aktionspreise_spa_310.md) die Möglichkeit, dies zu unterbinden.
