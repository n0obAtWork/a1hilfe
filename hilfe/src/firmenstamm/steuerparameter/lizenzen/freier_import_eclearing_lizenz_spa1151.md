# Freier Import Eclearing Lizenz(SPA1151)

<!-- source: https://amic.de/hilfe/_SPA_1151.htm -->

Lizenz, um den freien Import im eClearing-Modul nutzen zu können. Der freie Import ermöglicht es dem Anwender, Dateien per privater Datenbankprozedur einzuspielen und alle Funktionalitäten des eClearing-Moduls anzuwenden.
