# Farbgestaltung der Auswahlliste

<!-- source: https://amic.de/hilfe/farbgestaltungderauswahlliste.htm -->

Die Spalten der Auswahllisten können farblich gestaltet werden. Die Farbgestaltung kann auf zwei verschiedene Arten hinterlegt werden. Farbgestaltung über den Gestaltungsdialog ist nur dann möglich, wenn das zugrundeliegende SQL-Statement kein UNION und kein GROUP BY enthält. Es wird dann ggf. der Gestaltungsdialog für Farben ausgeblendet.
