# Serienbrief

<!-- source: https://amic.de/hilfe/serienbrief.htm -->

Die Serienbrieffunktionalität wurde für die Auswahlliste 2.0 neu entwickelt. Sie beruht auf einer Excel-Datei (\*.xls) als Datenquelle und arbeitet ohne Word-Vorlagen. Die vorherigen Methoden zum Erstellen von Serienbriefen findet man weiterhin unter „Druck / Quickreport“.
