# Sachkontenstamm (EPA Sachkontenstamm)

<!-- source: https://amic.de/hilfe/_EPA_Sachkontenstamm.htm -->

| Bezeichnung | Standardwert | Erklärung |
| --- | --- | --- |
| Nummernkreiszuordnung ignorieren | Nein | Kontonummern sollten für die unterschiedlichen Kontoarten (Sach-, Personen- und Oberkonten ) in verschiedenen Bereichen liegen. Diese kann man in A.eins selber über Nummernkreise festlegen. Diese werden in der „Allgemeinen Nummernkreiszuordnung“ **[MNDNK]** festgelegt. Ist dort kein Nummernkreis hinterlegt, so kann man auch kein neues Konto erfassen. Will man diesen Test ausschalten, so kann man hier **Ja** eintragen.<br> |
