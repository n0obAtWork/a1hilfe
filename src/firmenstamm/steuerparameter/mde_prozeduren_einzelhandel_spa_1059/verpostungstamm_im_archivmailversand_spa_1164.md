# Verpostungstamm im Archivmailversand (SPA 1164)

<!-- source: https://amic.de/hilfe/_SPA_1164.htm -->

Standard ist Nein

Wird dieser SPA auf „Ja“ gestellt, so wird im Formulararchiv-Mailversand der Absender zur Auswahl freigeschaltet. Im Standard wird dieses Feld mit dem Wert aus dem Einrichterparameter (EPA) vorbelegt. Sollte dieser Wert ungültig sein, so wird der ersetzt aus dem Standard des Verpostungsstamms.

Wird der Steuerparameter auf „Ja“ eingestellt, kann dieser Wert furch Auswahl aus dem Versandprofilstamm überschrieben werden.
