# EAN8 Code wird als solcher Erkannt, auch wenn dieser nicht gültig ist. (SPA 727)

<!-- source: https://amic.de/hilfe/_SPA_727.htm -->

Muss eingestellt werden, wenn ein eigener EAN 8 Codes gescannt werden sollen, der keine korrekte Prüfkennziffer hat. Oder wenn der Windows Scanner benutzt wird, da dieser keine Scaneinheit enthält.
