# Hauptmenü-Tastenkombinationen

<!-- source: https://amic.de/hilfe/hauptmentastenkombinationen.htm -->

| Tastenkombination | |
| --- | --- |
| SHIFT+Strg+F3 | Telefonnummerneingang |
| ALT+1 | Telefonnummerneingang |
| SHIFT+F4 | Öffnet den A.eins-Direktsprung-Dialog |
| F3 | Öffnet den A.eins-Direktsprung-Dialog |
| a-z bzw.A-Z | Öffnet den A.eins-Direktsprung-Dialog und belegt die gedrückte Taste dort vor. Der Tastendruck gibt somit den ersten „Buchstaben“ des Direktsprunges vor.<br>Somit kann man „einfach“ im Hauptmenü einen Direktsprung eingeben, ohne vorher SHIFT-F4 oder F3 tätigen zu müssen.<br>Zusätzlich zu den „Buchstaben“ sind die Sonderzeichen +, - und das Ausrufezeichen als erstes Zeichen zugelassen.<br>Wenn Sie Direktsprünge eingerichtet haben, die andere Zeichen als als erstes Zeichen haben als die hier beschriebenen, dann müssen Sie SHIFT-F4 bzw. F3 verwenden! |
| F1 | Hauptmenü-Hilfe |

Weitere Tastaturbelegungen sind kontextabhängig, siehe [Kontextmenüs der Menüpunkte](./hauptmenue_menuepunkte/kontextmenues_der_menuepunkte.md).
