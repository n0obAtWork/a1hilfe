# Rechnungstrennung durch Fakturiergruppe(SPA 93)

<!-- source: https://amic.de/hilfe/_SPA_93.htm -->

Trennen: sind mehrere Lieferscheine mit verschiedenen Fakturiergruppen markiert und man will diese in eine Sammelrechnung umwandeln, so wird für jede Fakturiergruppe eine Rechnung erstellt.

Neu: sind mehrere Lieferscheine mit verschiedenen Fakturiergruppen markiert und man will diese in eine Sammelrechnung umwandeln, so wird eine Sammelrechnung erstellt und die Warenbewegungen werden der Fakturiergruppe der Rechnung zugeordnet.

Nein: sind mehrere Lieferscheine mit verschiedenen Fakturiergruppen markiert und man will diese in eine Sammelrechnung umwandeln, so wird eine Sammelrechnung erzeugt und die Warenbewegungen werden den Fakturiergruppen der Lieferscheine zugeordnet.
