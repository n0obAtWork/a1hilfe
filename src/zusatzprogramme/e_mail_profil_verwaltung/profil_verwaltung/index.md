# Profil Verwaltung

<!-- source: https://amic.de/hilfe/_profil_verwaltung.htm -->

Direktsprung **[EMAIL]**

<details>
<summary>Felder der Profil Verwaltung</summary>

| **Feld** | **Beschreibung** |
| --- | --- |
| Id | Die Id des Profils. |
| E-Mail-Adresse | Die Adresse des Postfachs. |
| Domäne | Die Domäne des Dienstes über den das Postfach ausgelesen werden kann. |
| Webservice-Url | Die vollständige Adresse des Webservices über den das Postfach ausgelesen werden kann. |

</details>
