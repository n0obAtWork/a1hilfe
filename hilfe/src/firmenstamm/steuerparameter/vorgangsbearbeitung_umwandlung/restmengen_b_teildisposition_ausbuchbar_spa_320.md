# Restmengen b. Teildisposition ausbuchbar(SPA 320)

<!-- source: https://amic.de/hilfe/_SPA_320.htm -->

Bei „Ja“ wird auf der Teilumwandlungsmaske ein Knopf angeboten mit dem der Rest bei einer echten Teilumwandlung auf dem Originalbeleg ausgebucht wird. Dieser Knopf ist steht nur dann zur Verfügung, wenn der Quellbeleg die Stufe Auftrag hat.

Ferner gibt es noch einen Einrichterparameter Freischaltung dieses Knopfes auf der Maske.
