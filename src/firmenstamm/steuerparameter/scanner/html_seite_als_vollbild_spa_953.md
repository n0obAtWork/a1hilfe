# HTML-Seite als Vollbild(SPA 953)

<!-- source: https://amic.de/hilfe/_SPA_953.htm -->

Mit diesem Steuerparameter kann eingestellt werden, ob die HTML-Seite die drei Kopfzeilen überlagert.

| Einstellung | Bedeutung |
| --- | --- |
| Nein | Die Anzeige bleibt wie bisher |
| Ja | Die drei Kopfzeilen werden ausgeblendet und die HTML-Seite wird als Vollbild dargestellt. |
