# Zeiterfassung (EPA Zeiterfassung)

<!-- source: https://amic.de/hilfe/_epa_ zeiterfassung.htm -->

| Bezeichnung | Standardwert | Erklärung |
| --- | --- | --- |
| Bediener_Zeiterfassung | | Es kann eine Funktion hinterlegt werden, welche eine KundId sucht, falls diese nicht mitgegeben wird. So könnte beispielsweise bei einem Direktsprung auf die Maske eine KundId in Abhängigkeit des Users gezogen werden.<br>Beispielfunktion: „Zeiterfassung Bediener“<br> |
