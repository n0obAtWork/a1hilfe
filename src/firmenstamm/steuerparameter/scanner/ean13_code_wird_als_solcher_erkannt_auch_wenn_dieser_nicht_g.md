# EAN13 Code wird als solcher Erkannt, auch wenn dieser nicht gültig ist. (SPA 728)

<!-- source: https://amic.de/hilfe/_SPA_728.htm -->

Muss eingestellt werden, wenn ein eigener EAN 13 Codes gescannt werden sollen, der keine korrekte Prüfkennziffer hat. Oder wenn der Windows Scanner benutzt wird, da dieser keine Scaneinheit enthält.
