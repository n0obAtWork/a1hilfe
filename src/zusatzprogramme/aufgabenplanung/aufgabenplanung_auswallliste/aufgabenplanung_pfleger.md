# Aufgabenplanung Pfleger

<!-- source: https://amic.de/hilfe/_aufgabenplanungpfleger.htm -->

<details>
<summary>Felder des Aufgabenplanung Pflegers</summary>

| Feld | Beschreibung |
| --- | --- |
| Aufgenommen von | Kürzel des Erstellers |
| Aufgenommen am | Datum an dem der Datensatz erstellt wurde |
| Verantwortlicher | Kürzel des ToDo Inhabers |
| Status | 0: offen  
1: erledigt |
| Termin von | Datum  
Uhrzeit |
| Termin bis | Datum  
Uhrzeit |
| Für Kunde | Kunden ID (F3) |
| Betreff | Betreff des ToDos |
| Beschreibung | Text zum ToDo |

</details>

<details>
<summary>Funktionen des Aufgabenplanung Pflegers</summary>

| Funktion | Beschreibung |
| --- | --- |
| Neu (F8),  
Speichern (F9) | Erstellt neuen Datensatz  
Speichert aktuellen Datensatz |

</details>
