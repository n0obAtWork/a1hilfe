# Vertreterstamm: Pfleger

<p class="just-emphasize pseudo-header">Kopfdaten:</p>

| Feld | Bedeutung |
| :--- | :--- |
| Vert. Nr. | Gibt die Vertreternummer an |
| Klasse | Zeigt die Vertreternummer an und gibt den dazugehörigen Namen der Klasse aus |

<p class="just-emphasize pseudo-header">Register:</p>

<details>
  <summary>Allgemein</summary>

  | Feld | Bedeutung |
  | :--- | :--- |
  | Verkaufsgebiet | Verkaufsgebiet des Vertreters |
  | Abrechart(*) | Periode der Provisionsabrechnung als numerischer Schlüssel:<table><tbody><tr><td>0</td><td>keine Vertreterabrechnung</td></tr><tr><td>1</td><td>monatlich</td></tr><tr><td>2</td><td>zweimonatlich</td></tr><tr><td>12</td><td>jährlich</td></tr><tr><td>14</td><td>wöchentlich</td></tr><tr><td>15</td><td>vierzehntägig</td></tr></tbody></table> |
  | Anrede | Anrede des Vertreters |
  | Vorname | Vorname des Vertreters |
  | Name | Nachname des Vertreters |
  | Zusatz1 | Zusatz zum Vertreter Namen |
  | Straße | Straße des Vertreters |
  | Land -PLZ | Land + Postleitzahl des Vertreters |
  | Ort | Ort (bezogen auf Land + Postleitzahl) des Vertreters |
  | Ortsteil | Ortsteil (bezogen auf den Ort) des Vertreters |
  | Telefon | Telefonnummer des Vertreters |
  | Fax | Faxnummer des Vertreters |
  | Matchcode |  |
  | Partner 1 | Erster Partner des Vertreters |
  | Partner 2 | Zweiter Partner des Vertreters |
</details>

<details>
  <summary>Bemerkung</summary>

  Hier können Bemerkungen für den Vertreter eingetragen und gespeichert werden.
</details>

<p class="just-emphasize pseudo-header">Funktionen:</p>

<details>
  <summary>Funktionen des Vertreterstamm Pfleger:</summary>

  | Feld | Bedeutung |
  | :--- | :--- |
  | DSGVO-Liste | Zeigt die DSGVO Liste zu dem ausgewählten Vertreter an. |
  | Gruppenanteile (F6) | Öffnet eine Liste in der dem Vertreter ein, oder mehrere Vertretergruppen, zugeordnet werden können. |
  | Provisionsmerkmale (F10) | Öffnet Vertreterprovisionstabelle |
  | Anschriften (Shift + F7) | Ruft den Anschriften-Pfleger für den Vertreter auf. |
</details>
