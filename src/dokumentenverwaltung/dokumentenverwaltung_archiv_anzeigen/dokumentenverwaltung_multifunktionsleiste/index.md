# Dokumentenverwaltung- Multifunktionsleiste

<!-- source: https://amic.de/hilfe/dokumentenverwaltungmultifunkt.htm -->

Das Kontext-Menü und die Multifunktionsleiste ersetzen die sonst übliche Funktionsübersicht in A:eins.

Die Multifunktionsleiste ist in Abschnitte unterteilt, denen im Standard Funktionen wie folgt zugeordnet sind:

| Abschnitt | Funktion/[Tastatur]/Icon | Beschreibung |
| --- | --- | --- |
| **Dokument** | Rücktaste [ESC]  
![nav\_left\_blue](../../../ImagesExt/image8_819.png "nav_left_blue") | Unterstützung verlassen der Maske auf Touchscreens |
| | Anzeigen [CF12]  
![view](../../../ImagesExt/image8_820.png "view") | Ruft den externen Viewer auf, den Windows für die Anzeige vorgesehen hat. |
| | Anhänge … [CF11]  
![](../../../ImagesExt/image8_821.png) | Sucht die zugehörigen Anhänge heraus, und stellt diese in einem extra Fenster dar. |
| | Senden an… [F9]  
![mail\_forward](../../../ImagesExt/image8_822.png "mail_forward") | Formulararchiv Senden An-Dialog  
[Archiv Mail Versand](../../archiv_manager/archiv_mail_versand/index.md) |
| | PDF Signieren  
![](../../../ImagesExt/image8_823.png) | Unterstützt PDF-Signierung durch Signotect-System. |
| | Hinzufügen [F8]  
![add](../../../ImagesExt/image8_824.png "add") | Formulararchiv-Stammdatenpfleger-Funktion Neu/Einfügen  
[Archiv – Dokumente hinzufügen](../../archiv_dokumente_hinzufuegen.md) |
| | Drucken  
![printer2\_pdf](../../../ImagesExt/image8_825.png "printer2_pdf") | PDF-Dokument drucken  
Übergibt die selektierten Dokumente zum [PDF-Drucken](./pdf_drucken.md). |

![edit](../../../ImagesExt/image8_826.png "edit") | Formulararchiv-Stammdatenpfleger-Funktion Ändern  
[Archiv-Stammdatenpfleger](../../archiv_stammdatenpfleger.md) |

![notebook\_view](../../../ImagesExt/image8_827.png "notebook_view") | Formulararchiv-Stammdatenpfleger-Funktion Ansehen

![text\_delete](../../../ImagesExt/image8_831.png "text_delete") | Löscht den Archivtext

**(Funktionsgruppe)** | Ansicht aktualisieren

![document\_view](../../../ImagesExt/image8_837.png "document_view") | Stellt PDF-Dokumente und Bildelemente des A.eins-Archivs im Vorschau-Bereich dar.

![magazine\_folder](../../../ImagesExt/image8_838.png "magazine_folder") | Ordner-Ansicht aktivieren bzw. deaktivieren |
| **Allgemein**  
**(Funktionsgruppe „Administration“)** | Informationen  
![information](../../../ImagesExt/image8_839.png "information") | Zeigt Informationen zum selektierten Archiv-Eintrag und dem Archiv im generellen im Notepad an.  
Diese Informationen dienen hauptsächlich administrativen Zwecken, so kann es zum Beispiel nützlich sein die Guid für Copy&Paste zur Verfügung zu haben. |
| | Drucken [F10]  
![printer2](../../../ImagesExt/image8_840.png "printer2") | Formulararchiv drucken  
Übergibt das Dokument an Windows zum Drucken. So wird beispielsweise für PDF-Dokumente der Adobe-Reader aufgerufen der wiederum das Dokument an den Windows-Standarddrucker delegiert.  
Für PDF-Dokumente gibt es die Alternative auf der PDF-Vorschau via Kontext-Menü (rechte Maustaste) und der Funktion „Print“ das PDF an einen auswählbaren Drucker zu übergeben.  
   
(Diese Funktion ist vorerst unter Administration eingegliedert worden) |
| | Ableitung [SF2]

| | Speichern unter | Speichert die selektierten Dokumente in einen vorgebbaren Ordner. Die Dokumente erhalten die Standard-Namen bei Archiv-Export – außer es ist ein Dateiname in den Stammdaten vorgegeben. |
