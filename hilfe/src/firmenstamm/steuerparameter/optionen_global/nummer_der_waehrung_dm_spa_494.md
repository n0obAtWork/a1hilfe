# Nummer der Währung DM(SPA 494)

<!-- source: https://amic.de/hilfe/_SPA_494.htm -->

Nummer der Währung, in der DM ( deutsche Mark) eingerichtet ist. Die Formularsteuerung zieht diesen Steuerparameter, wenn Beträge in DM ausgewiesen werden soll ( z.B. nach einer Umstellung auf EURO vor dem 1.1.2002 )
