# Dateisystem-Pfad

<!-- source: https://amic.de/hilfe/_dateisystempfad.htm -->

Hier ist der zentrale Speicherort für die Archivierung der Belege anzugeben. Achten Sie bei Mehrplatz-Installation vornehmlich darauf, dass es sich um einen „allgemein“ zugänglichen Speicherort – am besten auf einem Fileserver – handeln muss.

Vermeiden Sie am besten „lokale“ Pfade, da jedes A.eins dann diesen Archivierungspfad lokal auffassen würde, und ein zentrales Formulararchiv sich so nur schwer realisieren lässt.

Die Standardvorgabe „..\\archiv“ ist eine geeignete Stelle um die Belege im A.eins-Verzeichnis des Servers zu sichern.

Sie müssen dann im Falle, dass Sie das komplette A.eins sichern, meistens auch keine Extra-Einstellungen im Datensicherungs-Programm machen.
