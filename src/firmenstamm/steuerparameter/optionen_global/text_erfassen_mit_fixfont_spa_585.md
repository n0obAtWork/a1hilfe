# Text erfassen mit Fixfont(SPA 585)

<!-- source: https://amic.de/hilfe/_SPA_585.htm -->

Bei „Ja“ werden Texte wie Bemerkungen / Textbausteine etc. wieder wie bis zur 6. Version mit einem fixen Font erfasst (gleichmäßige Breite der Zeichen). Bei „Nein“ wird eine proportionale Schriftart verwendet.
