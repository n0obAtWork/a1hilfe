# Bearbeiten der Streckentexte

<!-- source: https://amic.de/hilfe/bearbeitenderstreckentexte.htm -->

Über den Button mit dem Stift-Symbol neben dem jeweiligen Report kann man eine Maske zur Bearbeitung der Streckentexte öffnen.  
Die Felder die angezeigt werden stammen aus der Anwendung [Vorgangsklammer](../../../vorgangsabwicklung/vorgangsklammer.md#uebVorgangsklammer) und dort aus den Registern [Drucktexte](../../../vorgangsabwicklung/vorgangsklammer.md#Register_Drucktexte) und [Bemerkungen](../../../vorgangsabwicklung/vorgangsklammer.md#Register_Bemerkungen).  
Es werden hier im Register Allgemein immer nur die Felder angezeigt, die vom jeweiligen Report verwendet werden.
