# Qualitätsmerkmale

<!-- source: https://amic.de/hilfe/_qualitaetsmerkmale.htm -->

Hauptmenü > Saatzucht > Saatenlabor > Qualitätsmerkmale

Direktsprung **[SAATR]**

In diesem Stammdatenpfleger werden die Qualitätsmerkmale gepflegt. Sie werden in [Laborverfahren](./laborverfahren.md#Laborverfahren_Merkmale), [Labordaten](./labordaten/index.md#Labordaten_Kontrollanbau) und [Fruchtarten](./fruchtarten.md) verwendet. Der Einrichterparameter „[Erweiterte Einstellungen](../../firmenstamm/einrichterparameter/qualitaetsmerkmale_epa_qualitaetsmerkmale.md)“ erlaubt weitere Eingabemöglichkeiten auf der Maske.

<p class="just-emphasize">Erfassungsmaske</p>

Es stehen folgende Eingabefelder und Eingabemöglichkeiten zur Verfügung.

| Name | Bedeutung |
| --- | --- |
| Merkmalsnummer | Eindeutige Nummer des Merkmals.<br> |
| Merkmalstyp | Hier wird der Typ des Merkmals angezeigt. Mit **F3** kann eine Auswahl getroffen werden. Zurzeit stehen folgende Typen zur Verfügung:<br><ul><li>- &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; kein Typ</li><li>Fremdbesatz</li><li>Mutterkorn</li><li>Genotyp&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Merkmale dieses Typs können in den Laborverfahren in den Merkmalen für Markeranalyse verwendet werden.</li><li>Phänotyp&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Merkmale dieses Typs können in den Laborverfahren in den Merkmalen für Feldversuche verwendet werden.<br>&nbsp;</li></ul> |
| Bezeichnung | Die Bezeichnung des Merkmals.<br> |
| Formatauswahl | Mithilfe der Taste **F3** können hier Eingabeformate ausgewählt werden. Dieses Format wird verwendet, um in den Labordaten die Ausprägung des Merkmals anzuzeigen.<br> |
| Einheit | Die Einheit des Merkmals.<br>Steht der Einrichterparameter „Erweiterte Einstellungen“ auf „Ja“, so kann mit der Taste **F3** eine Auswahl über die Mengeneinheiten aufgerufen werden.<br> |
| Druckkennzeichen | Hier kann das Druckkennzeichen angegeben werden.<br> |
