# Rechnungstrennung durch Steuergruppe(SPA 85)

<!-- source: https://amic.de/hilfe/_SPA_85.htm -->

Trennen: sind mehrere Lieferscheine mit verschiedenen Steuergruppen markiert und man will diese in eine Sammelrechnung umwandeln, so wird für jede Steuergruppe eine Rechnung erstellt.

Neu: sind mehrere Lieferscheine mit verschiedenen Steuergruppen markiert und man will diese in eine Sammelrechnung umwandeln, so wird eine Sammelrechnung erstellt und die Warenbewegungen werden der Steuergruppe der Rechnung zugeordnet.

Nein: sind mehrere Lieferscheine mit verschiedenen Steuergruppen markiert und man will diese in eine Sammelrechnung umwandeln, so wird eine Sammelrechnung erzeugt und die Warenbewegungen werden der Steuergruppe der Lieferscheine zugeordnet.
