# Kostenobjekte: Pfleger

<!-- source: https://amic.de/hilfe/_kostenobjekte_pfleger.htm -->

### Felder des Kostenobjekt-Pflegers

| **Feld** | |
| --- | --- |
| Nummer | Die Nummer des Kostenobjektes.<br>Ein Kostenobjekt mit der Nummer „0“ wird ausgeliefert. Es wird jedoch nicht als Kostenobjekt ausgewertet, denn „0“ bedeutet immer „kein Kostenobjekt“.<br>Wurde in der allgemeinen Nummernkreiszuordnung <strong>[MNDNK]</strong> den Kostenobjekten ein Nummernkreis zugeordnet, so findet hier nach der Eingabe der Kostenobjekt-Nummer eine Bereichsüberprüfung statt.<br> |
| Bezeichnung | Die Bezeichnung des Kostenobjektes.<br>Ist der SPA 34 (Mehrsprachigkeit aktiv) gesetzt, so kann hier mithilfe der **F3**\-Taste eine [sprachabhängige Bezeichnung](../../../firmenstamm/a_eins_sprache/sprachabhaengige_bezeichnung_in_den_stammdaten.md) eingepflegt werden.<br> |
| Matchcode | Die Kurzbezeichnung des Kostenobjektes.<br>Im ***Neu***\-Fall wird der Matchcode nach dem Setzen der Bezeichnung mit den ersten 20 Zeichen der Bezeichnung vorbelegt, solange kein Matchcode eingetragen war.<br> |
| Erfassungssperre | Diese Sperre gilt für die Belegerfassung in der Finanzbuchhaltung. Steht diese auf **Ja**, so kann das Kostenobjekt dort nicht mehr verwendet werden.<br>Bedingung: Die Erfassungssperre kann auf **Ja** gesetzt werden, wenn kein Eintrag für das Kostenobjekt im [Sachkontenstamm](../../stammdaten_der_fibu/sachkonten.md) existiert. Existiert ein Eintrag, so muss zunächst das Kostenobjekt aus den entsprechenden Sachkonten entfernt werden, um die Erfassungssperre auf **Ja** setzen zu können.<br> |
| Bemerkung | Hier kann ein wahlfreier Text zu dem jeweiligen Kostenobjekt erfasst werden.<br>Mit einem **Doppelklick** auf das Bemerkungs-Feld öffnet sich der Text-Editor.<br> |
