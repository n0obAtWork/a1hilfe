# Formulararchiv

<!-- source: https://amic.de/hilfe/_vorgangsmappe_formulararchiv.htm -->

Die Funktionalität „[Formulararchivgruppe](../../dokumentenverwaltung/formulararchiv_gruppen.md)“ findet in der Streckenerfassung Anwendung. Im Allgemeinen gilt dabei, wird etwas innerhalb der Streckenerfassung archiviert, bekommt der Archiveintrag eine Gruppenzuordnung zur Strecke.

Das findet zurzeit bei folgenden Funktionen statt:

- Druck von Reporten
- Druck von Vorgängen
- Import aus der Formulararchivanzeige (Strg + F12)
