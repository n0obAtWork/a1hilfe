# Rechnungstrennung durch Empfängergebiet(SPA 89)

<!-- source: https://amic.de/hilfe/_SPA_89.htm -->

Trennen: sind mehrere Lieferscheine mit verschiedenen Empfängergebieten markiert und man will diese in eine Sammelrechnung umwandeln, so wird für jedes Empfängergebiet eine Rechnung erstellt.

Neu: sind mehrere Lieferscheine mit verschiedenen Empfängergebieten markiert und man will diese in eine Sammelrechnung umwandeln, so wird eine Sammelrechnung erstellt und die Warenbewegungen werden dem Empfängergebiet der Rechnung zugeordnet.

Nein: sind mehrere Lieferscheine mit verschiedenen Empfängergebieten markiert und man will diese in eine Sammelrechnung umwandeln, so wird eine Sammelrechnung erzeugt und die Warenbewegungen werden den Empfängergebieten der Lieferscheine zugeordnet.
