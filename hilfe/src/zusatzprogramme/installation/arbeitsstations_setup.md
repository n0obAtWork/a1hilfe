# Arbeitsstations- Setup

<!-- source: https://amic.de/hilfe/arbeitsstationssetup.htm -->

Die einzelnen Arbeitsstationen, welche auf das A.eins System zugreifen, benötigen auch ein Versionsupdate, wenn eine neue Version in das System eingespielt worden ist. Dieser Umstellungsprozess muss unter Administratorrechten direkt auf dem Arbeitsplatz durchgeführt werden, dazu ist das Standard Setup Programm aus dem Setup Verzeichnis aufzurufen.
