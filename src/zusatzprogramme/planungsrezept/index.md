# Planungsrezept

<!-- source: https://amic.de/hilfe/planungsrezept.htm -->

Administration > Werkzeuge > Elara Konfiguration

Mit dem Planungsrezept können Sie Ressourcen planen, die Sie für die Erstellung eines oder mehrerer Artikel benötigen. So können Sie beispielsweise Rohwaren einem oder mehreren Zielartikeln zuordnen und so eine Datengrundlage für die Auswertung Ihrer Bedarfsplanung schaffen.

**Achtung:**

Trotz des Namensanteils „Rezept“ hat diese Anwendung keinen direkten Zusammenhang mit den in der Produktion verwendeten Rezepten !
