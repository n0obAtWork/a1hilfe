# Mindestanforderung für die Scanner-Hardware

Die Spefikationswerte unten in der Tabelle sind aus den Spezifikationsdaten des Datalogic Memor Scanner entnommen worden. Die sind als Referenz Daten zu sehen.

| --- |
| Expansion Slots: Secure Digital (user accessible) |

| Operating System: Windows CE 5.0 |
| Microprocessor: Intel® XScale PXA255 @ 200 MHz |
| System Flash Memory: 128 MB |
| Local Area Network: IEEE 802.11b/g (WLAN) |
| Interfaces: USB Connector |
