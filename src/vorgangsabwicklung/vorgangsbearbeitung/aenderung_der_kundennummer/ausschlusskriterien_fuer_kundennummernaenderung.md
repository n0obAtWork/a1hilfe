# Ausschlusskriterien für Kundennummernänderung

<!-- source: https://amic.de/hilfe/ausschlusskriterienfrkundennum.htm -->

Unter folgenden Umständen sind Änderungen der Kundennummer nicht möglich

| Kontraktbeteiligungen |
| --- |
| **Kriterium** | **Verfahrensweise** |
| Position hat erledigten Kontrakt | |
| Beleg enthält Stückliste oder Folgeartikel | |
| Beleg wurde aus Teilumwandlung erstellt | |
| Warenposition(en) wurde(n) weiterdisponiert | |
| | |
| Je nach Behandlungsschema | Beleg stornieren und neu erfassen oder Behandlungsschema anpassen |

| Partiebeteiligung |
| --- |
| **Kriterium** | **Verfahrensweise** |
| Position hat kundenspezifische Partie und Partiezwang. Es ist keine Partie für den Zielkunden vorhanden | Position stornieren |
| | |
| | |
| Je nach Behandlungsschema | Beleg stornieren und neu erfassen oder Behandlungsschema anpassen |
