# Kontextmenüs der Menü-Hauptpunkte

<!-- source: https://amic.de/hilfe/kontextmensdermenhauptpunkte.htm -->

| Kontextmenü Menü-Hauptpunkt | Tastatur | Beschreibung |
| --- | --- | --- |
| Zugriffsrechte | Umschalttaste + F2 | Ermöglicht die Rollenpflege des Menü-Hauptpunktes.<br>Für die Pflege der Funktion „Zugriffsrechte“ selber siehe [Rollenpflege](../hauptmenue_rollenpflege.md) |
| Zugriffsrechte Hauptmenü | | Ermöglicht die Rollenpflege aller Menü-Hauptpunkte.<br>Für die Pflege der Funktion „Zugriffsrechte Hauptmenü“ selber siehe [Rollenpflege](../hauptmenue_rollenpflege.md) |
| Direktsprung | Umschalttaste + F4 | „Aeins-Direktsprung“ |
| Funktionspfleger (nur für Entwickler) | | Direkt-Aufruf des Funktionspflegers |
| Icon zuordnen (vorerst nur für Entwickler) | | Direkt-Aufruf des Icon-Zuordnungspflegers |
