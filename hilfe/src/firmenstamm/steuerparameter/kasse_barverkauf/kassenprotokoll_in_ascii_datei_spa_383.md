# Kassenprotokoll in ASCII-Datei(SPA 383)

<!-- source: https://amic.de/hilfe/_SPA_383.htm -->

Hier wird entschieden, ob über die POS-Kasse erfasste Vorgänge in einer Datei mitprotokolliert werden sollen, um auf einem Ausfallsystem erfasste Vorgänge aus dieser Datei nachträglich einzuspielen. Die Datei befindet sich in der A.eins-Root im Verzeichnis Export/kassprot. Dabei wird pro Kasse, Bediener und Tag eine eigene Datei angelegt.
