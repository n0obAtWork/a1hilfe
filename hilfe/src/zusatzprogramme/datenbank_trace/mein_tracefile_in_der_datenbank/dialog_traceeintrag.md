# Dialog „Traceeintrag“

<!-- source: https://amic.de/hilfe/dialogtraceeintrag.htm -->

Mit Hilfe des Dialoges können Trace-Einträge gesondert in Augenschein genommen werden, insbesondere lange „SQL-Ausdrücke“ können so inspiziert werden.

| Maskenfeld |
| --- |
| SQL Ausdruck |
| Plan |
| Wer |
| Zeit |
| Maske |
| Verbrauch |
| Err |
| Status |
| CurNo |
| Id |

Die Maskenfelder entsprechen den Auswahllisten-Feldern.
