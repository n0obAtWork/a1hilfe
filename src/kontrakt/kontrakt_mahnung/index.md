# Kontrakt „Mahnung“

<!-- source: https://amic.de/hilfe/_kontrakt_kbmahnung.htm -->

Hauptmenü > Kontraktverwaltung > Kontraktstammdaten

oder Direktsprung **[KTR]**

In der Kontraktverwaltung besteht die Möglichkeit, dass für Kontrakte, deren Mengenzeiträume abgelaufen sind, eine sogenannte Kontraktmahnung gedruckt werden kann. 

Hierzu gibt es zuerst eine „Kontraktmahnung Vorschlagsliste“. In dieser werden nur Kontrakte angezeigt, die unerledigt, ungelöscht und mindestens ein Mengenzeitraum des Kontraktes zum „Stichtag“ eine offene Restmenge > 0 aufweist.
