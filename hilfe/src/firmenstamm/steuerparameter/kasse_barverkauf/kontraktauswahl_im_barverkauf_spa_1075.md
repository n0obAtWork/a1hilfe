# Kontraktauswahl im Barverkauf (SPA 1075)

<!-- source: https://amic.de/hilfe/_SPA_1075.htm -->

| Einstellung | Bedeutung |
| --- | --- |
| Ja | Aktiviert die Kontraktauswahl im Barverkauf. |
| Nein | Keine Kontraktauswahl im Barverkauf (Standard). |
| Passende Kontrakte direkt zuordnen | Passende Kontrakte werden ohne Abfrage direkt zugeordnet. |
