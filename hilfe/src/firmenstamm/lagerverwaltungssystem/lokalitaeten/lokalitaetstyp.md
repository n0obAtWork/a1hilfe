# Lokalitätstyp

<!-- source: https://amic.de/hilfe/_lokalittstyp.htm -->

Der Lokalitätstyp stellt den Typ der Lokalität dar. Zum Anlegen von Typen gibt es das Anwendungsformat „AF_LVS_LOKTY“.

In diesem können die einzelnen Typen angelegt werden. Eine Voraussetzung ist, dass mindestens das Format „0“ „--“ angelegt sein muss. Dieses ist jedoch bei der Auslieferung von A.eins bereits angelegt.
