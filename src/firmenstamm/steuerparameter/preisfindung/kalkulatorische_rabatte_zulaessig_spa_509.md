# Kalkulatorische Rabatte zulässig(SPA 509)

<!-- source: https://amic.de/hilfe/_SPA_509.htm -->

Hierdurch können kalkulatorische Rabatte freigeschaltet werden. Bisher konnten nur Zu- / Abschläge kalkulatorisch behandelt werden.

| Einstellungen |
| --- |
| Nicht behandeln | (Voreinstellung) kalkulatorische Rabatte werden wie gewöhnliche Rabatte behandelt. |
| verwenden | Kalkulatorische Rabatte werden verwendet (kalkulatorische Rabatte sind freigeschaltet) |
| ignorieren | Kalkulatorische Rabatte, so sie denn eingerichtet sind (z.B. als Vorbereitung auf eine Umstellung) werden nicht berücksichtigt und auch im Gegensatz zur Voreinstellung nicht als gewöhnliche Rabatte berechnet. |
