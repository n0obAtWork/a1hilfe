# EPAS

<!-- source: https://amic.de/hilfe/epas2.htm -->

Unter dem Punkt können auf Masken für einzelne Bedienerklassen Felder versteckt, angezeigt oder gegen Eingabe geschützt werden.

| Feldname | Bedeutung |
| --- | --- |
| Maskenname | In diesem Feld wird der Name angezeigt |
| Bedienerklasse | Für welche Bedienerklasse soll die Einstellungen gelten. |
| Feldname | Name des Feldes |
| Standard | Standard des Auslieferungszustandes |
| Nicht abfragen | Das Feld wird für die Eingabe gesperrt. |
| Nicht zeigen | Feld wird für die gewählte Bedienerklasse ausgeblendet. |
| Nicht verbergen | Feld wird für die gewählte Bedienerklasse eingeblendet |
