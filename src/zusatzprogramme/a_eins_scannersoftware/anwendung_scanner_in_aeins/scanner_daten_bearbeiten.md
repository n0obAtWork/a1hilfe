# Scanner Daten bearbeiten

<!-- source: https://amic.de/hilfe/_scanner_daten_bearbeiten.htm -->

In dieser Variante werden alle erfassten Positionen zu einem Scanvorgang angezeigt. Wird im Feld ein D ein \* angezeigt wird, so wird diese Position beim nächsten Scann mit dem dazugehörigen Scanner gelöscht.
