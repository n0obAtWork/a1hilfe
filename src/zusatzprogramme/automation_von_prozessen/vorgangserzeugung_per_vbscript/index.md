# Vorgangserzeugung per VBScript

<!-- source: https://amic.de/hilfe/vorgangserzeugungpervbscript.htm -->

Zur automatisierten Erzeugung von Vorgängen per VBScript werden die folgenden Dateien

benötigt:

• bestellung.vbs

• bestellung_include.vbs

• bestellung_start.vbs

• autom_bestellung.xml

• auftrag.xml

Die Namen sowie der Speicherort sind nicht zu verändern.
