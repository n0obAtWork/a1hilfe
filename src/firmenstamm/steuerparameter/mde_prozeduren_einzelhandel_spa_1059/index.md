# MDE Prozeduren Einzelhandel (SPA 1059)

<!-- source: https://amic.de/hilfe/_SPA_1059.htm -->

Hier werden/ können alternative Prozeduren hinterlegt werden, die im MDE-Workflowprozess Einzelhandel die Funktionalität der Standard-Prozeduren überschreiben.

Für jede Art von Abarbeitungsart kann hier eine eigene private Prozedur hinterlegt werden.
