# Beleganlage trotz fehlerhafte Position im Vorgangsimport(SPA 965)

<!-- source: https://amic.de/hilfe/_SPA_965.htm -->

Mit diesem Steuerparameter kann eingestellt werden, ob ein Beleg aus dem Vorgangsimport heraus erzeugt werden soll, obwohl eine Position nicht in den Beleg übernommen wurde.

| Einstellung | Bedeutung |
| --- | --- |
| Ja | Steht der Steuerparameter auf „Ja“ so wird der Beleg erstellt ohne die Fehlerhafte Position. Dies ist der Standard Fall. |
| Nein | Steht der Steuerparameter auf „Nein“ so wird der Beleg nicht erzeugt. Dieser wird dann im Vorgangsimport auf „Fehlerhaft“ gestellt. |
