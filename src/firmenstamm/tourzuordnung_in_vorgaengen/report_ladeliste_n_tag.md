# Report Ladeliste n. Tag

<!-- source: https://amic.de/hilfe/reportladelistentag.htm -->

Dieser Report zeigt Vorgänge für ausgewählte Touren an. Dafür ist ein bestimmtes Lieferdatum auszuwählen. In der Bereichsauswahl kann man außerdem festlegen, ob statt der Artikelbezeichnung der Warenbewegungstext angezeigt werden soll. In der Standardeinstellung für den Vorgangstyp werden Lieferscheine angezeigt; dieser kann beliebig mit Hilfe der F3-Auswahl geändert werden.
