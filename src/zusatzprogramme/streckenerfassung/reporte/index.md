# Reporte

<!-- source: https://amic.de/hilfe/_vorgangsmappe_reporte.htm -->

Im System lassen sich „Crystal Report“ und AMIC Etikettendruck einbinden. Diese können über das [Profile aktiviert](../profile/index.md#registerAuswertungen) werden. Die Reporte lassen sich dann über das Kontextmenü aufrufen.

Für die eingebundenen Reporte gibt es auf der Maske verschiedene Buttons.

Die Vorschau (Button Symbol Lupe) zeigt den Inhalt des jeweiligen Reports an.

Die [Bearbeitung der Streckentexte](./bearbeiten_der_streckentexte.md) für den jeweiligen Report erfolgt über den Button mit dem Symbol Stift.

Drucken der Reports / Archiv Mail Versand erfolgt über den Button mit dem Drucker
