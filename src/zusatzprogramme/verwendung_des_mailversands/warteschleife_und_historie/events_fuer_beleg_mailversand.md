# Events für Beleg-Mailversand

<!-- source: https://amic.de/hilfe/eventsfrbelegmailversand.htm -->

Systempflege > Sonstige > [Event-Manager](../../events/einrichten_bearbeiten_eines_events.md#amic_ueb_rk_allgemein)

Je nachdem wann Sie die zu versendenden Rechnungen an den Mailserver schicken wollen, können Sie mit dem Direktsprung **[EVT]** – Events ein Event einrichten, das eine der unten aufgeführten Eventprozeduren aufruft.

Erstellen Sie einen Zeitplan für dieses Event, das je nach gewünschtem Intervall anliegende Rechnungen per Mail versendet.

- „AMIC_EVT_E_BELEGMAILER()“ bzw. eine private Ableitung davon
