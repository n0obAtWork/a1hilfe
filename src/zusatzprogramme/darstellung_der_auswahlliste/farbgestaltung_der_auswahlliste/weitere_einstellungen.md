# Weitere Einstellungen

<!-- source: https://amic.de/hilfe/weitereeinstellungen.htm -->

Man kann die Farbgestaltung für A.eins per Aufrufparameter ein- bzw. ausschalten:

**FARBE = FALSE**

Dies bewirkt, dass die Farbauswahl nicht mehr zu Verfügung steht. Sämtliche Farbeinstellungen werden ignoriert. Der Reiter zur Farbeinstellung erscheint nicht im Gestaltungsdialog.

**FARBBALKEN =TRUE**

Bisher werden markierte Zeilen immer mit einem „#“ versehen, um sie als ausgewählt zu kennzeichnen. Setzt man den Parameter auf TRUE, so werden die Zeilen zusätzlich in blauer Schrift dargestellt.
