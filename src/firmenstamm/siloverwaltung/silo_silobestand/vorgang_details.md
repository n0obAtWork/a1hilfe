# Vorgang-Details

<!-- source: https://amic.de/hilfe/_silovorgangdetails.htm -->

Diese Funktion kann nur ausgeführt werden, wenn ein Datensatz in der jeweiligen Auswahlliste markiert worden ist. Die Funktion ruft die Auswahlliste [Vorgangsübersicht](../../../vorgangsabwicklung/vorgangsbearbeitung/vorgang_detailansicht.md) auf

Wird diese Funktion in der Variante Silo ausgewählt, so werden zu der angewählten Siloposition die dazugehörigen Vorgänge angezeigt.

Wird diese Funktion in der Variante Silobestand ausgewählt, so werden alle Vorgänge zu diesem Silo angezeigt.
