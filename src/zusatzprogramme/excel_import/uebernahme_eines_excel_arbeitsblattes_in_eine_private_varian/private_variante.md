# Private Variante

<!-- source: https://amic.de/hilfe/_EXCLEI_PRIVATE_VARIANTE.htm -->

Herkunft und Aufbau dieser Variante basieren auf [Excel-Import](../index.md). Hat noch keine Aktualisierung der Variante stattgefunden, verfügt die Variante über folgende Funktionalitäten:

| **Felder** | |
| --- | --- |
| Information | „Excelimport starten mit F10 …“ |

| **Funktionen** | |
| --- | --- |
| Variante aktualisieren [**F10**] | Führt den Excel-Import, der dieser Variante zugrunde liegt durch und aktualisiert diese Variante.  
 |
