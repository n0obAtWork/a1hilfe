# A.eins Scannersoftware

<!-- source: https://amic.de/hilfe/_cescanner.htm -->

Mit der A.eins Scannersoftware können Prozesse abgearbeitet werden.

Folgende

Eine Mindestvoraussetzung an die Hardware finden Sie in [Technische Daten](./technische_daten/index.md).

| Unterstützte Prozesse |
| --- |
| Erfassung von Eingangsbelegen |
| Erfassung von Labordaten |

| Erfassung der Inventur inkl. Partiezuordnung (Online wie Offline ) |
| Erfassung von Aufträgen |
| **Abarbeitung von Lagerumbuchungen** |
| Abarbeitung von Bestellungen |
