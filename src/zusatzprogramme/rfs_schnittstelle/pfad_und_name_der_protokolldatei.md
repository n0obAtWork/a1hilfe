# Pfad und Name der Protokolldatei

<!-- source: https://amic.de/hilfe/pfadundnamederprotokolldatei.htm -->

Geben Sie hier bitte einen gültigen Pfad und Dateinamen an. Der Pfad muss existieren, die Datei wird, sofern sie noch nicht vorhanden ist, neu angelegt. In dieser Datei werden sämtliche Ausnahmesituation ( Fehlermeldungen / falsche Kontenzuordnungen etc ) sowie jede erfolgreiche Datenerstellung protokolliert.
