# Bei Gutscheinannahme Nummer verproben? (SPA 992)

<!-- source: https://amic.de/hilfe/_SPA_992.htm -->

Hier wird festgelegt wie die Verprobung der Gutscheinnummer geschehen soll.

| Wert | Bedeutung |
| --- | --- |
| 0 | Keine Verprobung (Standard) |
| 1 | Warnmeldung bei bereits eingelöstem Gutschein |
| 2 | Die Gutscheinnummer wird abgelehnt bei bereits eingelöstem Gutschein |
