# Folgeartikel automatisch erfassen (SPA 1133)

<!-- source: https://amic.de/hilfe/_SPA_1133.htm -->

| Einstellung | Bedeutung |
| --- | --- |
| Ja | Aktiviert automatische Erfassung von Folgeartikeln in der Marktkasse außer Leergutartikel. |
| Nein | Keine automatische Erfassung von Folgeartikeln in der Marktkasse. |
