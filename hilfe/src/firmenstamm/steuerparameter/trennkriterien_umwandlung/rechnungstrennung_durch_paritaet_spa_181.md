# Rechnungstrennung durch Parität(SPA 181)

<!-- source: https://amic.de/hilfe/_SPA_181.htm -->

Trennen: sind mehrere Lieferscheine mit verschiedenen Paritäten markiert und man will diese in eine Sammelrechnung umwandeln, so wird für jede Parität eine Rechnung erstellt.

Neu: sind mehrere Lieferscheine mit verschiedenen Paritäten markiert und man will diese in eine Sammelrechnung umwandeln, so wird eine Sammelrechnung erstellt und die Warenbewegungen werden der Parität der Rechnung zugeordnet.

Nein: sind mehrere Lieferscheine mit verschiedenen Paritäten markiert und man will diese in eine Sammelrechnung umwandeln, so wird eine Sammelrechnung erzeugt und die Warenbewegungen werden den Paritäten der Lieferscheine zugeordnet.
