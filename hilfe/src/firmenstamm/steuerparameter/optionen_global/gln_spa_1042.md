# GLN (SPA 1042)

<!-- source: https://amic.de/hilfe/_SPA_1042.htm -->

In diesem Steuerparameter können Einstellungen für das Switchboard vorgenommen werden.

| Wert | Bedeutung |
| --- | --- |
| GLN | Die GLN des Unternehmens aus der die NVEs generiert werden |
| Basisnummer | Die Basisnummer der GLN |
| Reserveziffer NVE | Die erste Ziffer der NVE ist die Reserveziffer |
| | |
