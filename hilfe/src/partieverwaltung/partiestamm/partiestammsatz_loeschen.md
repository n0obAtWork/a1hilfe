# Partiestammsatz löschen

<!-- source: https://amic.de/hilfe/_partienstammsatzlsch.htm -->

Hauptmenü > Partieverwaltung > Chargen / Partien > Partie-Stammdaten

oder Direktsprung **[PAR]**

Über die Funktion ***Löschen*** **F7** wird das Löschen einer Partie eingeleitet. In dem Anschlussmenü wird damit die Funktion ***Löschen*** **F7** frei geschaltet und kann somit für das endgültige Löschen verwendet werden.

Grundsätzlich wird über diese Funktion eine Partie nicht vollständig aus dem System gelöscht. Gelöschte Partien können weiterhin zur Anzeige gebracht werden, die Nutzung einer gelöschten Partie wird aber nicht mehr möglich sein.
