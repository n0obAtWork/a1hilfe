# Druckeinstellungen(SPA 907)

<!-- source: https://amic.de/hilfe/_SPA_907.htm -->

Hier können allgemeine Druckeinstellungen für A.eins festgelegt werden. Folgende Typen stehen zur Verfügung.

| Typ | Wert |
| --- | --- |
| Lieblingsdrucker VRGD Makro ausführen | Hier kann eingestellt werden, wie die Vorbelegung für das Ausführen der [Vorgangsdruckklassen](../../druckereinrichtung/vorgangsdruckklassen.md) Makros auf der [Lieblingsdrucker](../../../vorgangsabwicklung/vorgangsbearbeitung/lieblingsdruckerdruck.md) Maske ist. |
