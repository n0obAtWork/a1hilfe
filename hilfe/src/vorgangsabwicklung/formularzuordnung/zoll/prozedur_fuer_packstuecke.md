# Prozedur für Packstücke

<!-- source: https://amic.de/hilfe/prozedurfrpackstcke.htm -->

In dieses Feld kann eine private Prozedur eingetragen werden, welche für alle Positionen einer Ausfuhranmeldung die dazugehörigen Packstücke definiert. Dieses Feld muss gefüllt werden, da sonst eine Platzhalterfunktion verwendet wird. Die Vorbelegung durch die Platzhalterfunktion Zoll_Packstuecke.sql ist also nicht aussagekräftig.
