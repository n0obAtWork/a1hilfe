# eRechnung editieren (SPA 1153)

<!-- source: https://amic.de/hilfe/_SPA_1153.htm -->

Mit diesem SPA wird das verhalten eingestellt, das A.eins zeigt, wenn eine Rechnung geöffnet werden soll, für die ein eRechnungs-Export erstellt wurde.

| Einstellung | Bedeutung |
| --- | --- |
| Nein | Editiersperre - Die Editiersperre kann durch Rücksetzen des Exports aufgehoben werden. |
| Warnung | Vor dem Öffnen des Belegs wird gewarnt, dass der Beleg bereits zu einer eRechnung exportiert wurde. |
| Ja | Der Beleg darf beliebig editiert werden |
| | |
