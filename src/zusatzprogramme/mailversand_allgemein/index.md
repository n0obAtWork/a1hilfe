# Mailversand allgemein

<!-- source: https://amic.de/hilfe/_vpstmailversandallgemein.htm -->

Es gibt viele Quellen, die E-Mails aus A.eins versenden möchten. Alle diese Mailquellen sollten im Idealfall die gemeinsame Schnittstelle zum Versand von E-Mails verwenden.

Voraussetzung für den Versand von Belegen ist,

• dass diese ins Archiv gedruckt wurden / werden

• dass der Mailempfänger bekannt ist oder im Rahmen der Abarbeitung bekannt gemacht wird

• dass der Zugang zum Mailserver im Versandprofilstamm hinterlegt ist oder die Daten an die Schnittstelle übertragen werden.
