# Restmenge über Gesamtzeit

<!-- source: https://amic.de/hilfe/_restmengeuebergesamtzeit.htm -->

| Restmenge über Gesamtzeit |
| --- |
| Nein | Die Restmenge wird immer über die jeweiligen Zeiträume geprüft. |
| Ja | Die Restmenge wird nur über den gesamten Kontraktzeitraum geprüft. Eine Buchung ist also unabhängig von den eingetragenen Teil-Zeiträumen möglich. |
