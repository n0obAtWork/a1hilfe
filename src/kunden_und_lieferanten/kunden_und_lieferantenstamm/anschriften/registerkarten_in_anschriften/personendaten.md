# Personendaten

<!-- source: https://amic.de/hilfe/_anschperso.htm -->

An dieser Stelle werden für den Fall, dass diese Anschrift eine Person (z.B. einen Ansprechpartner) identifiziert die Möglichkeit angeboten personenbezogene Daten für die [Verbotslistenprüfung](../../verbotslisten/stammdatenpflege.md) zu hinterlegen.

| Feld | Beschreibung |
| --- | --- |
| IN-Number | Sozialversicherungsnummer |
| Nationalität | Nationalität |
| Geburtsort | Geburtsort |
| Geburtsland | Geburtsland |
| Reisepassdaten | Reisepassdaten (Nummer, Ausstellungsdatum und -Ort) |
