# Lagerumb. Bei Lieferung Voreink.-Vorverk. (603)

<!-- source: https://amic.de/hilfe/lagerumbbeilieferungvoreinkvor.htm -->

Ja: Bei Lieferungen für einen Voreinkaufs- oder Vorverkaufskontrakt, die nicht auf oder von dem Lager erfolgen, auf dem die Voreinkaufs- oder Vorverkaufsrechnung erfasst wurde, wird automatisch eine Lagerumbuchung auf dieses Lager erstellt.

 Nein: Bei Lieferungen für einen Voreinkaufs- oder Vorverkaufskontrakt, die nicht auf oder von dem Lager erfolgen, auf dem die Voreinkaufs- oder Vorverkaufsrechnung erfasst wurde, wird keine Lagerumbuchung auf dieses Lager erstellt. Dieses muss im Bedarfsfall per Hand erfolgen.
