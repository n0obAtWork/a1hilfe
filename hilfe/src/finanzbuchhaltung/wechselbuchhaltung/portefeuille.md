# Portefeuille

<!-- source: https://amic.de/hilfe/portefeuille.htm -->

Hauptmenü \> Mahn-/Zahl-/Zinswesen \> Wechselbuchhaltung \> Portefeuille

Direktsprung **[WEP]**

Der Bereich Portefeuille dient dazu, bei großen Wechselbeständen die Wechsel in Bereiche einzuteilen.  
Hier kann festgelegt werden, ob der Wechsel diskontierfähig ist.

| | Beschreibung |
| --- | --- |
| Portefeuille Nr | Laufende Nummer, die als Verweis in anderen Tabellen verwendet wird.<br> |
| Diskontierbar | Ob für einen Wechsel dieser Gruppe eine Diskontabrechnung erstellt werden darf.<br> |
| Bezeichnung | Ausführliche Bezeichnung<br> |
