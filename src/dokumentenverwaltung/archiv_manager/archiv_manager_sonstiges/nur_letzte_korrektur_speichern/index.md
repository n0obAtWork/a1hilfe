# Nur letzte Korrektur speichern

<!-- source: https://amic.de/hilfe/_nurletztekorrektursp.htm -->

Bei Vorgangsdrucken wird geprüft ob sich im Formulararchiv schon Einträge mit **gleicher Belegnummer, gleicher Vorgangsklasse**, **gleicher Vorgangs-Unternummer** und **gleicher Jahrnummer** befinden und wenn dies der Fall sein sollte, dann wird kein neuer Formulararchiv-Eintrag angelegt, sondern der jüngste dieser Belege erfährt ein Update.

Somit wird sichergestellt, dass sich immer nur die letzte Korrektur eines Beleges im Archiv befindet.
