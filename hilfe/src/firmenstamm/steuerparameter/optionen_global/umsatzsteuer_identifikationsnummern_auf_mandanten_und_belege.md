# Umsatzsteuer-Identifikationsnummern auf Mandanten und Belegebene(SPA 703)

<!-- source: https://amic.de/hilfe/_SPA_703.htm -->

Dieser Steuerparameter sorgt dafür, dass man in der Belegerfassung der Finanzbuchhaltung für die Belegarten AR, AG, ER, EG und EB eine abweichende Umsatzsteueridentifikationsnummer (USt.-IdNr.) für den Kunden/Lieferanten bzw. den Mandanten erfassen kann. Es können in der Belegerfassung nur Umsatzsteueridentifikationsnummern erfasst werden, die im Kundenstammpfleger bzw. im Mandantenstammpfleger erfasst wurden. Diese können über eine F3-Auswahl ausgewählt werden.

Der Steuerparameter hat drei Ausprägungen:

**Nein:** Die Felder werden nicht abgefragt.

<strong>Ja:</strong> Die Felder werden normal abgefragt.

**Mit Vorbelegung:** Wenn nur eine USt.-IdNr existiert, so wird dieses sofort vorbelegt, ansonsten öffnet sich, sobald man das noch leere Feld betritt, die F3-Auswahl.
