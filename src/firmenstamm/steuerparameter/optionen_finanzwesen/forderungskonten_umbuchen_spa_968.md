# Forderungskonten umbuchen(SPA 968)

<!-- source: https://amic.de/hilfe/_SPA_968.htm -->

Wenn für Personenkonten im laufenden Betrieb die Forderungsgruppe geändert wird, muss der aus diesem Personenkonto resultierende Forderungs- bzw. Verbindlichkeitsbetrag von den Konten der alten Forderungsgruppe auf die Konten der neuen Forderungsgruppe umgebucht werden. Dies geschieht beim Jahreswechsel, wenn dieser Parameter auf **Ja** steht.
