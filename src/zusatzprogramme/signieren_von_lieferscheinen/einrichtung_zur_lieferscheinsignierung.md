# Einrichtung zur Lieferscheinsignierung

<!-- source: https://amic.de/hilfe/einrichtungzurlieferscheinsign.htm -->

Auf einer Einrichtungsmaske kann die Unterschriftsposition auf dem Lieferschein festgelegt werden. Innerhalb der Formulararchivabwicklung wird noch festgelegt, in welchem Importverfahren die Signatur abgearbeitet werden soll. Die Unterschrift wird IMMER auf einer festen Seite (z.B. 1) eingetragen, die Seitennummer kann eingerichtet werden.
