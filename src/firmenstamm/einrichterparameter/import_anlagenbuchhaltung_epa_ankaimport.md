# Import Anlagenbuchhaltung (EPA ANKAIMPORT)

<!-- source: https://amic.de/hilfe/_EPA_ANKAIMPORT.htm -->

| Bezeichnung | Standardwert | Erklärung |
| --- | --- | --- |
| Importprozedur ist änderbar? | Nein | Im Normalfall ist die Prozedur zum Einspielen der Anlagedaten völlig ausreichend und muss nicht geändert werden. Wenn jedoch von fachkundiger Seite eine eigene Prozedur erstellt wurde, so kann man mit diesem Schalter es ermöglichen, die Datenbankprozedur änderbar zu machen. |
