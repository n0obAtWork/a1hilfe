# Automatische Rabatte

<!-- source: https://amic.de/hilfe/automatischerabatte.htm -->

Administration > Steuerung > Steuerungsparameter zeigen

Oder Direktsprung **[SPA]**

Ein Rabatt errechnet sich aus einer Kombination aus Artikel und Kunden.

Damit automatische Berechnung der Rabatte jedoch erfolgt, muss diese im Steuerungsparameter [60 - Automatische Rabatte](../../../firmenstamm/steuerparameter/preisfindung/automatische_rabatte_spa_60.md) eingeschaltet sein.
