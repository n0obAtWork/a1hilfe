# Neuer Fremdlagerkontrakt je Voreinkauf (580)

<!-- source: https://amic.de/hilfe/neuerfremdlagerkontraktjevorei.htm -->

Ja: Bei jeder Erfassung einer Fremdlagerposition wird ein neuer Fremdlagerkontrakt für diese Position angelegt.

 Nein: (außer Rohwareartikel) Falls eine Fremdlagerposition mit einer bereits erfassten Fremdlagerposition in Kunde, Artikel, Lager und Lagerplatz übereinstimmt, so wird der Kontrakt zu dieser Position um die Menge der neuen Position erhöht und für die neue Position wird kein eigener Kontrakt angelegt.  
Für Rohwareartikel wird immer ein neuer Fremdlagerkontrakt angelegt.
