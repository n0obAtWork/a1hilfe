# Lieferbelegstamm

<!-- source: https://amic.de/hilfe/lieferbelegstamm.htm -->

Lieferbelege erhalten beim Anlegen eine eindeutige Nummer, die vom Bediener nicht zu ändern ist.  
Lieferbelegdatensätze können nicht geändert werden. Wenn ein Datensatz fehlerhaft ist, muss er gelöscht werden. Er erhält dann ein Löschkennzeichen und die dazugehörigen Positionen werden gelöscht.
