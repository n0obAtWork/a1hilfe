# DrillDown

<!-- source: https://amic.de/hilfe/drilldown.htm -->

Drilldown ist die Möglichkeit mit einem Click mit der rechten Maustaste auf einem Datenfeld in der Tabelle Details anzusehen. So können Sie sich in einer zusammengefassten Ansicht die Herkunft der Summen auf einen Blick anzeigen lassen oder Felder sehen, die Sie der Übersichtlichkeit halber nicht als Datenfelder bestimmt haben.
