# Bedingungen zum Hedging

<!-- source: https://amic.de/hilfe/_bedingungenzumhedgin.htm -->

Absicherungen durch Hedging sind anzuraten, wenn die Kontraktwährung eine Fremdwährung ist.

Die AMIC-Vorgabe-Prozedur AMIC_HEDGE_GETORDERSTRING stellt folgende Bedingungen:

Die Kontraktwährung ist eine Fremdwährung

Die Kontraktwährung hat einen Hedge-Teiler-Eintrag im Währungsstamm, der größer als 0 ist, sowie ein Eintrag, dass die Fremdwährung stärker als die Buchwährung ist.

In das Anwender Format af_HedgeLoca müssen gültige Hedge Lokationen eingetragen werden.

Im Artikelstamm muss eine gültige Hedge-Lokation angegeben werden.
