# Wechselkosten

<!-- source: https://amic.de/hilfe/wechselkosten.htm -->

Hauptmenü \> Mahn-/Zahl-/Zinswesen \> Wechselbuchhaltung \> Wechselkosten

Direktsprung **[WEKO]**

![](../../ImagesExt/image8_744.png)

| | Beschreibung |
| --- | --- |
| Abr. Gruppe | Laufende Nummer<br> |
| Bezeichnung | Ausführliche Bezeichnung<br> |
| Sprache | Auswahl mit **F3** aus vorher eingerichteten Sprachen<br> |
| Formularid | Auswahl mit **F3** aus vorher eingerichteten Wechselformularen<br> |
| Zinsgruppe | Auswahl mit **F3** aus vorher eingerichteten Zinsgruppen<br> |
| Pos | Laufende Nummer der Position.<br> |
| Text | Text der Position<br> |
| Vorbelegung | Betrag in Buchwährung oder Prozent<br> |
| Bezug | Kann **Betr** sein, wenn es sich bei der Vorbelegung um einen festen Betrag handelt oder<br>**PRZ** wenn unter Vorbelegung die Prozentzahl steht, mit der später gerechnet werden soll.<br> |
| Konto | Auswahl mit **F3** aus dem [Sachkontenstamm](../stammdaten_der_fibu/sachkonten.md) um den Betrag auf ein Sachkonto verbuchen zu können<br> |
| Kostenstelle | Auswahl mit **F3** aus den Kostenstellen um den Betrag auf eine [Kostenstelle](../kostenrechnung/kostenstellen.md) verbuchen zu können<br> |
| Kostenträger | Steht der SPA „Kostenträgerrechnung angeschlossen“ auf **Ja**, dann erscheint diese Spalte. Dort können die eingerichteten [Kostenträger](../kostenrechnung/kostentraeger.md) mit **F3** ausgewählt werden.<br> |
| Kostenobjekt | Mit einer gültigen Kostenobjekt-Lizenz können hier die [Kostenobjekte](../kostenrechnung/kostenobjekte/index.md) erfasst werden.<br> |
| StSchl | Auswahl mit **F3** aus vorher eingerichteten Steuerschlüsseln um evtl. den Betrag um die gesetzliche Umsatzsteuer zu erhöhen<br> |
