# Anbindung an das A.eins

<!-- source: https://amic.de/hilfe/anbindungandasaeins.htm -->

Per Ereignissteuerung kann eine automatisierte Verarbeitung der Ausgehenden wie auch eingehenden Datenstrukturen gewährleistet werden. Hierzu steht die Funktion Bitzer_Vorgang zur Verfügung. Diese Funktion bildet über ein Importverzeichnis die Übernahme der Daten in die Offline- Waagedaten Tabelle ab.
