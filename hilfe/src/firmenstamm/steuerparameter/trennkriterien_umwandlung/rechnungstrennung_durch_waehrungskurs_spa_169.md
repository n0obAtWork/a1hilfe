# Rechnungstrennung durch Währungskurs(SPA 169)

<!-- source: https://amic.de/hilfe/_SPA_169.htm -->

Trennen: sind mehrere Lieferscheine mit verschiedenen Währungskursen markiert und man will diese in eine Sammelrechnung umwandeln, so wird für jeden Währungskurs eine Rechnung erstellt.

Neu: sind mehrere Lieferscheine mit verschiedenen Währungskursen markiert und man will diese in eine Sammelrechnung umwandeln, so wird eine Sammelrechnung erstellt und die Warenbewegungen werden dem Währungskurs der Rechnung zugeordnet.

Nein: sind mehrere Lieferscheine mit verschiedenen Währungskursen markiert und man will diese in eine Sammelrechnung umwandeln, so wird eine Sammelrechnung erzeugt und die Warenbewegungen werden dem Währungskurs der Lieferscheine zugeordnet.
