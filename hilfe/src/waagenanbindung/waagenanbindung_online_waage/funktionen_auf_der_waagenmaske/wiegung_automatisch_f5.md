# Wiegung automatisch F5

<!-- source: https://amic.de/hilfe/_waage_wiegung_automatisch.htm -->

Die Wiegung wird automatisch durchgeführt. Die Werte der Waage werden an das System übertragen.  
Es wird vom System erkannt, ob es sich um die 1. Wiegung handelt oder ob schon eine Wiegung existiert und somit die 2. Wiegung stattfindet.
