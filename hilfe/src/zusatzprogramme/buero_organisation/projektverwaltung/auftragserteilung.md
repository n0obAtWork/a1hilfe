# Auftragserteilung

<!-- source: https://amic.de/hilfe/auftragserteilung.htm -->

Aus dem Projekt heraus kann jederzeit ein Auftrag erzeugt werden, dabei wird die eingegebene Zeit als Menge dem im Einrichtungsbereich zugeordneten Artikels hinzugefügt.

Für die drei verschiedenen Zeitangaben werden drei Artikelzeilen erzeugt.
