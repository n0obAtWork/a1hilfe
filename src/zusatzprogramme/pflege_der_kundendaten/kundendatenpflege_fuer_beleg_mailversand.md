# Kundendatenpflege für Beleg-Mailversand

<!-- source: https://amic.de/hilfe/kundendatenpflegefrbelegmailve.htm -->

Zu guter Letzt sollten Sie beachten, dass eine E-Mail nur an einen Empfänger gesendet werden kann, wenn der E-Mail-Empfänger auch bekannt ist.

Im Kundenstamm des Kunden finden Sie das Feld E-Mail-Adresse. Dort muss für den Kunden eine gültige E-Mail Adresse eingetragen sein, unter der die Belege zugestellt werden können.

Im Kundenstamm [KU] pflegen Sie auf der [Registerkarte Kennzeichen](../../kunden_und_lieferanten/kunden_und_lieferantenstamm/kennzeichen/index.md) das Merkmal [Belegversand](../../kunden_und_lieferanten/kunden_und_lieferantenstamm/kennzeichen/index.md#Kundenstamm_Kennzeichen_Belegversand).
