# Beispiel für eine Private Anwendung

<!-- source: https://amic.de/hilfe/_cescannerprivateanw.htm -->

In diesem Beispiel erklären wir, wie Sie die Möglichkeit haben, eine eigene Anwendung für unser Aeins Scanner System zu entwickeln. In diesem einfachen Beispiel zeigen wir, wie Artikel Information unten in die Anzeige geschrieben werden.

| **Was wird für eine Private Anwendung benötigt.** |
| --- |
| Eine IB_Box, die die Anzeige auf dem Scanner steuert |
| Eine Private Prozedur, die die Daten zusammensammelt und an den Scanner überträgt |
| Die Private Anwendung muss unter sctcp eingetragen sein |
