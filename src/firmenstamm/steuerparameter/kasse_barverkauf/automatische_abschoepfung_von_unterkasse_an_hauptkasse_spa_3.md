# Automatische Abschöpfung von Unterkasse an Hauptkasse(SPA 356)

<!-- source: https://amic.de/hilfe/_SPA_356.htm -->

Hier wird entschieden, ob beim Kassenabschluss einer Unterkasse eine Zählung durchgeführt wird („Nein“) oder ob alle kassenseitig erfassten Zahlungen der zugehörigen Hauptkasse automatisch gutgeschrieben werden und so nur eine Zählung nötig ist („Ja“).
