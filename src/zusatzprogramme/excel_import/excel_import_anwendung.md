# Excel-Import: Anwendung

<!-- source: https://amic.de/hilfe/excelimportanwendung.htm -->

Stammdatenpflege > Stammdatenpfleger > Excel-Import

oder Direktsprung [**EXCELI**]

| **Feld** | |
| --- | --- |
| Name | Der Name des Excel-Importes und der privaten Variante.<br> |
| Speicherort | Der Pfad der Excel-Datei.<br> |
| Blatt-Name | Der Name des zu importierenden Excel-Arbeitsblattes.<br> |
| Anwendung | Der Name der Anwendung, unter der sich die private Variante befindet.<br> |
| Offset Zeile;Spalte | Siehe [Offset Zeile](./excel_import_pfleger.md#Offset_Zeile) und [Offset Spalte](./excel_import_pfleger.md#Offset_Spalte)<br> |
| Erstellt von | Benutzer, der den Excelimport-Stammsatz angelegt hat.<br> |
| Erstellt am | Zeitpunkt, zu dem der Excelimport-Stammsatz angelegt wurde.<br> |
| Spalten als Text | Siehe [Spalten als Text](./excel_import_pfleger.md#Spalten_als_Text)<br> |

| **Funktionen** | |
| --- | --- |
| Ändern **[F5]**, Ansehen **[F6]**, Neu **[F8]**, Löschen **[F7]** | siehe [Excel-Import: Pfleger](./excel_import_pfleger.md)<br> |
| Variante aktualisieren **[F10]** | Führt den Excel-Import aus und aktualisiert die Variante.<br> |
| Variante starten **[F9]** | Wechselt in die betreffende Variante.<br> |
| Exceldatei öffnen **[SF9]** | Öffnet die Excel-Datei.<br> |

| **F2-Bereichsauswahl** | |
| --- | --- |
| Suchen | Sucht in den Feldern<br>• Name<br>• Speicherort<br>• Blatt-Name<br>• Anwendung<br> |
