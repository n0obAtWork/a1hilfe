# Abweisen ab

<!-- source: https://amic.de/hilfe/_abweisenab.htm -->

Wenn Sie denn nun A.eins so präpariert haben, dass es die externe Darstellung zum Anzeigen der Belege verwendet und sie zu viele Belege zur Ansicht markiert haben, dann laufen Sie Gefahr, den Speicher Ihres Systems zu ruinieren. Mit diesem Parameter können Sie angeben, ab wie viel Einheiten das Anzeigen verweigert werden soll.
