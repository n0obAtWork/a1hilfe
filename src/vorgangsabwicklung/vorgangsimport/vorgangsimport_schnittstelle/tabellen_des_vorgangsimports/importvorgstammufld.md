# ImportVorgStammUFLD

<!-- source: https://amic.de/hilfe/importvorgstammufld.htm -->

UFLD-Felder

In dieser Relation werden die Setzungen von UFLD-Feldern für den Vorgang vorgenommen.

Bitte beachten Sie, dass nur UFLD-Felder gesetzt werden können, die vom importierenden Bediener für die jeweilige Vorgangs(unter-)klasse gemäß Einrichtung gesetzt werden dürfen.

| Feld | Bedeutung |
| --- | --- |
| IVS_GUID | Guid des Stammsatzes |
| UFLDID | Id Nummer des User Feldes |
| UFLDWert | Inhalt des Feldes |
