# Die wesentlichen Einstellungen im Detail

<!-- source: https://amic.de/hilfe/diewesentlicheneinstellungenim.htm -->

**Belegdatum:**

Das Belegdatum ist bei den meisten Umwandlungen eingebbar.

**Unterklasse:**

Bei Belegumwandlung in dem Zielvorgang die Vorgangsunterklasse 0 gewählt. Bei gesetztem Häkchen kann eine abweichende Vorgangsunterklasse angegeben werden. Löscht man das Häkchen, so bleibt bei der Umwandlung die Unterklasse des Quellbelegs erhalten (sofern für die Zielklasse eine entsprechende Unterklasse eingerichtet ist!). Die hierfür früher zuständige Einstellung unter SPA (‚Erhalten der Unterklasse bei Umwandlung’) wird nur noch als Vorbelegung beim ersten Anwenden einer Umwandlungsfunktion herangezogen.

**Nachlauf:**

Bei sehr vielen Umwandlungsfunktionen wird eine Nachlauffunktion angeboten, mit deren Hilfe die umgewandelten Zielbelege bequem weiterverarbeitet werden können. Um Blockierungsprobleme mit dem Mandantenserver zu verhindern kann die Einstellung des [Steuerparameters 800](../../firmenstamm/steuerparameter/vorgangsbearbeitung_allg/bei_umwandlung_von_belegen_auf_mandantenserververarbeitung_w.md) geändert werden.

**Korrektur:**

Nach der Umwandlung werden alle Zielbelege zur manuellen Korrektur aufgerufen.

**Druck:**

Die Zielbelege können nach der Umwandlung gedruckt werden.

**Fibu Übertrag ( + Druck):**

Es erfolgt zunächst der Druck, dann der Fibuübertrag. Hinweis: Per Einrichterparameter kann die Funktionalität auf bestimmte Bedienergruppen eingeschränkt werden.

**Periode Ware / Behandlung einstellen:**

Die Behandlung von Perioden bei den diversen Umwandelfunktionen kann hier beeinflusst werden.

ACHTUNG: Das Häkchen ‚Behandlung einstellen’ wird gespeichert, die darunter angebotenen Alternativen werden stets aus SPA-Einstellungen vorbelegt (die neuen SPA Einstellungen zu Perioden werden weiter unten in einem eigenen Kaptitel beschrieben).

**Periode Ware / Periode einstellen:**

Bei gesetztem Häkchen wird die Einstellung unter ‚Behandlung einstellen’ außer Kraft gesetzt. Die Periode wird fest eingestellt.

**Periode Fibu / Periode einstellen:**

Die Fibu Periode kann hier bei Bedarf eingestellt werden. Wir empfehlen hier jedoch, keine Vorgaben zu machen, da zum Zeitpunkt des Fibuübertrags per SPA-Steuerung eine geeignete Periodenauswahl vorgenommen wird.
