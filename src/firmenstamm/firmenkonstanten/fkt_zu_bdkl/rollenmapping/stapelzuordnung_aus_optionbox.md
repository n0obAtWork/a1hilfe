# Stapelzuordnung aus Optionbox

| Felder |    |
| :----- | :- |
| Die Funktionen aus der Ziel-Optionbox | Die Optionbox in der die „neuen“ Funktionen sind, die die „alten“ Rechte übernehmen sollen |
| erben die rechte aus der Quell-Optionbox | Die Optionbox die die Rechte zum Mapping zur Verfügung stellt |
| Gridanzeige | <p>Listet die in Frage kommenden Funktionen an.</p><p>Das sind genau diejenigen nicht-privaten Funktionen aus der Ziel-Optionbox, die es auch in der Quell-Optionbox gibt und für die es keinen Eintrag in Rollenkontextmapping gibt.</p> |

| Funktionen |    |
| :--------- | :- |
| Rollenmapping durchführen … **(F10)** | Erzeugt für jeden Eintrag aus dem Grid einen Standard-Eintrag im Rollenkontextmapping. |
