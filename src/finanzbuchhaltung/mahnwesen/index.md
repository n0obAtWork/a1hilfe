# Mahnwesen

<!-- source: https://amic.de/hilfe/mahnwesen.htm -->

Das Mahnwesen basiert auf verschiedenen Parametern, die unter **"Stammdaten"** im Abschnitt **"Mahnwesen"** eingerichtet werden können. Das Mahnwesen selber ist lediglich eine Auswertung dieser Parameter **(Mahnabstand, Anschreiben, Gebühren, etc.)** sowie der in den offenen Posten hinterlegten Informationen **(Fälligkeit, Betrag, etc.)**.

Zum Erstellen von Mahnungen sind folgende Schritte notwendig:

- **Mahnvorschläge erstellen**
- **Mahnvorschläge bearbeiten und freigeben**

**•** **Mahnungen bearbeiten**
