# Manuelle Zu-/Abschläge

<!-- source: https://amic.de/hilfe/manuellezuabschlge.htm -->

Zu-/Abschläge können manuell im Anschluss an die Erfassung einer Warenposition erfasst werden.

<p class="just-emphasize">Nummer oder Text</p>

Auswahl eines [Zu-Abschlagsatzes](./automatische_zu_abschlaege/generelle_zu_abschlaege.md). Wählen Sie hier den für diesen Zu-/Abschlag gültigen Satz aus. 0 ist die Standardeinstellung für einen komplett manuell erfassten Zu-/Abschlag.

<p class="just-emphasize">Zu-/Abschlag-Gruppe</p>

Wird nur angezeigt – die Zu-/Abschlaggruppe des Artikels

<p class="just-emphasize">EKZ-Nummer</p>

Erlöskennziffer auf die der Zu-/Abschlag gebucht werden soll – 0 = die gleiche Erlöskennziffer wie die zugehörige Warenposition.

<p class="just-emphasize">Kostenstelle</p>

Hier kann eine von der Warenposition abweichende Kostenstellennummer für den Zu-/Abschlag angegeben werden.  
0 = es wird die Kostenstellennummer der Warenposition übernommen  
Dieses Erfassungsfeld steht nur zur Verfügung, wenn der Steuerparameter **Kostenstellen-Lizenz** aktiviert ist.

<p class="just-emphasize">Kostenträger</p>

Hier kann eine von der Warenposition abweichende Kostenträgernummer für den Zu-/Abschlag angegeben werden.  
0 = es wird die Kostenträgernummer der Warenposition übernommen  
Dieses Erfassungsfeld steht nur zur Verfügung, wenn der Steuerparameter **Kostenträgerrechnung angeschlossen** aktiviert ist.

<p class="just-emphasize">Kostenobjekt</p>

Hier kann eine von der Warenposition abweichende Kostenobjektnummer für den Zu-/Abschlag angegeben werden.  
0 = es wird die Kostenobjektnummer der Warenposition übernommen  
Dieses Erfassungsfeld steht nur zur Verfügung, wenn der Steuerparameter **Kostenobjekt-Lizenz** aktiviert ist.  
Die Bezeichnung dieses Feldes ist in der **OPTION Kostenobjekt_Label** einrichtbar!

<p class="just-emphasize">Wirkt auf Preis</p>

Ja – Zu-/Abschlag wirkt auf den Einzelpreis des Artikels pro Mengeneinheit und wird dann erst mit der Menge multipliziert

<p class="just-emphasize">Formel</p>

Zu-Abschlagsformel siehe auch [in den automatischen Zu-/Abschlägen](./automatische_zu_abschlaege/generelle_zu_abschlaege.md#ZuAbschlagformel).

<p class="just-emphasize">Prozentsatz</p>

Prozentsatz des Zu-/Abschlags (bei prozentualen Zu-/Abschlägen)

<p class="just-emphasize">Preis/Satz</p>

Zu-/Abschlagbetrag (nicht bei prozentualen Zu-/Abschlägen)

<p class="just-emphasize">Preiseinheit</p>

Die zum Zu-/Abschlagbetrag gehörige Preiseinheit

<p class="just-emphasize">Bezugsmenge</p>

Ebendies

<p class="just-emphasize">Bezugswert</p>

Wert auf den sich der Zu-/Abschlag beziehen wird

<p class="just-emphasize">Betrag</p>

Wird nur angezeigt: Zu-/Abschlagbetrag

<p class="just-emphasize">Steuer</p>

Wird nur angezeigt: Steuerbetrag

<p class="just-emphasize">Skontierfähig</p>

Ja/nein

<p class="just-emphasize">Offen</p>

Wird nur angezeigt: wird dieser Zu-/Abschlag auf dem Beleg gedruckt

<p class="just-emphasize">In Zeile</p>

Ja/Nein-Entscheidung, ob dieser Zu-/Abschlag auf dem Ausdruck unterdrückt werden soll (JA), weil er lediglich zur internen Preisermittlung dient oder dem Belegempfänger sichtbar ausgedruckt werden sill (Nein).
