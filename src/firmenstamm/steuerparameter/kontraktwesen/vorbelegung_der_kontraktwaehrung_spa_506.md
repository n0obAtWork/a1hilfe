# Vorbelegung der Kontraktwährung (SPA 506)

<!-- source: https://amic.de/hilfe/_SPA_506.htm -->

Hier wird entschieden, wie die Währung bei Neuanlage von Kontrakten vorbelegt wird.

| Wert | Bedeutung |
| --- | --- |
| 0 (keine) | Es erfolgt keine Vorbelegung, d.h. es wird die Währung 0 verwendet. |
| 1 (aus Kundestamm) | Die Vorbelegung erfolgt mit der Währung des Kunden. |
| 2 (wie Buchwährung) | Die aktuelle Buchwährung wird als Vorbelegung verwendet. |
