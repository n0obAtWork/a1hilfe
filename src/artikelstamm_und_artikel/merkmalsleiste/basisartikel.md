# Basisartikel

<!-- source: https://amic.de/hilfe/_basisartikel.htm -->

Um dieses System nutzen zu können, müssen zunächst die so genannten „Basisartikel“ als Artikel im System eingegeben werden. Diese Basisartikel enthalten die Kerninformationen zu allen weiteren per Merkmalsleiste angelegten Artikel. Während der Vorgangserfassung können dann Artikel auf Basis dieses Artikels angelegt werden, die in folgenden Feldern von dem Basisartikel ggf. abweichen:

• Artikelnummer

• Mengeneinheitsgruppe

• Artikelbezeichnung

Es müssen dann für jedes Lager entsprechende Basisartikel angelegt werden.
