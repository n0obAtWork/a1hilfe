# Anhang Umwandlung

<!-- source: https://amic.de/hilfe/_email_connector_attachmentConversion.htm -->

Im Profil kann eingestellt werden, dass eine Umwandlung der Anhänge in das Format PDF/A (*Format zur Langzeitarchivierung digitaler Dokumente*) erfolgen soll.

Die Dokumente werden zusätzlich zu den ursprünglichen Dokumenten im Archiv gespeichert.

Aktuell werden nur Excel-Dokumente in das Format PDF/A umgewandelt.
