# Zuordnung von Bildern

<!-- source: https://amic.de/hilfe/_artikelbildzuordnung.htm -->

Sowohl einem Artikelstammeintrag wie auch einem Artikeleintrag kann ein Bild in Form einer Bilddatei zugeordnet werden (zum Beispiel Bilddateien vom Typ JPEG, JPG, PNG, GIF, BMP).

Mit der Funktion <strong>Bilddatei…</strong> wird eine Maske geöffnet, die ein bereits zugeordnetes Bild darstellt. Ist noch keine Bilddatei zugeordnet, so wird der Text „es ist keine Graphik zugeordnet…“ auf der Maske ausgegeben.

Mit den Maskenfunktionen <strong>aus Datei laden…</strong> und **bearbeiten…** kann eine Bilddatei zugeordnet beziehungsweise ein bereits zugeordnetes Bild mit dem unter Windows dem jeweiligen Dateityp zugeordneten Bildbearbeitungsprogramm bearbeitet werden.

<strong>Achtung:</strong> Hier vorgenommene Änderungen werden erst beim Speichern des Artikelstamms beziehungsweise Artikels wirksam.
