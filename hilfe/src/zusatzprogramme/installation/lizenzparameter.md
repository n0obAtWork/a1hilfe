# Lizenzparameter

<!-- source: https://amic.de/hilfe/lizenzparameter.htm -->

Während der Installation werden im A.eins System die Lizenzparameter eingetragen, sowie in der Windows Registrierdatenbank. Weiterhin werden die Lizenzparameter beim Setup in der Datenbank hinterleg, sowie bei den einzelnen Mandanten in der AmicConf.INI eingetragen. Die Lizenzparameter müssen vor der Installation bereitliegen, da nur mit gültigem Lizenzschlüssel und gültiger Seriennummer die Software installierbar ist.
