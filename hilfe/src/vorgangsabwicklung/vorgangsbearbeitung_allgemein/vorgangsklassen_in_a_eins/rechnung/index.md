# Rechnung

<!-- source: https://amic.de/hilfe/rechnung.htm -->

Rechnungen entstehen durch Neuerfassung oder Umwandlung aus Angeboten / Aufträgen / Lieferscheinen. Rechnungen werden als Vorgang gespeichert, auf sie kann in Nachfolgevorgängen zugegriffen werden; die Menge wird verbucht und vermindert den Bestand; der Umsatz erhöht sich. A.eins stellt folgende Bearbeitungsfunktionen zur Verfügung:

| Funktionsname | Funktion |
| --- | --- |
| ***Erfassen F 8*** | Erfassung einer neuen Rechnung<br> |
| ***Stapelverarbeitung*** | Übernahme einer oder mehrerer Rechnungen in einen Bearbeitungsstapel<br> |
| ***Erstdruck F9*** | Erstdruck einer Rechnung<br> |
| ***Formulardruck F10*** | Wiederholungsdruck<br> |
| ***Lieblingsdruckerdruck*** | Auswahl eines anderen Druckers / Zuordnung eines anderen Formulars<br> |
| ***Korrektur F5*** | Korrektur einer Rechnung<br> |
| ***Ansicht F6*** | Rechnung im Ansicht-Modus öffnen<br> |
| ***Schnellkorrektur*** | Plandatum und Menge ändern<br> |
| ***Kopieren CF8*** | Kopieren der Rechnung für einen auszuwählenden Kunden<br> |
| ***Vorschau F11*** | Druckvorschau<br> |
| ***Stornieren F7*** | Stornieren (Löschen) der Rechnung<br> |
| ***Storno Rechnung*** | Erfassung eines Stornobeleges<br> |
| ***Freigabe/Sperren*** | Freigabe / Sperren für weitere Bearbeitung<br> |
| ***Gutschrift aus Rechnung*** | Umwandeln in eine Gutschrift<br> |
| ***Sammel-GU aus Rechnungen*** | Umwandeln mehrerer Rechnungen in eine Gutschrift<br> |
| ***Transportauftrag*** | Auftrag einer Spedition zuordnen<br> |
| ***Archiv ansehen*** | Anzeige archivierter Vorgänge<br> |
| ***Partien nachtragen*** | nachträgliche Zuordnung von Partien<br> |
| ***Wiedervorlage CF9*** | Vorgang mit einem Wiedervorlagevermerk versehen<br> |
| ***Arbeitsregel ändern*** | manuelle Änderung von Weiterverarbeitungsparametern<br> |
| ***FiBu Übertrag*** | Übergabe an die Finanzbuchhaltung<br> |
| ***FiBu-Eintrag zurücknehmen*** | Siehe [FiBu – Eintrag zurücknehmen](../../../vorgangsbearbeitung/fibu_eintrag_zuruecknehmen.md)<br> |
| ***EDI-Datentransfer*** | Bearbeitet den ausgewählten Beleg nach dem Profil des im Kunden hinterlegten EDI-Partners. Dabei werden die Felder „EDI-Partner“ und „Export“ in der Auswahlliste bewirtschaftet. Nach der erfolgreichen Bearbeitung ist die EDI-Nachricht im Direktsprung [STDDAT] einsehbar.<br> |
| ***PDF-Signieren SH-F12*** | Die ausgewählte Rechnung [elektronisch unterschreiben](../../../../dokumentenverwaltung/archiv_manager/signature_pad_benutzen/pdf_dokument_signieren.md)<br> |
