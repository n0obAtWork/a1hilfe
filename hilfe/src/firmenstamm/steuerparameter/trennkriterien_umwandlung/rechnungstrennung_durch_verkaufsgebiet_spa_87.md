# Rechnungstrennung durch Verkaufsgebiet(SPA 87)

<!-- source: https://amic.de/hilfe/_SPA_87.htm -->

Trennen: sind mehrere Lieferscheine aus verschiedenen Verkaufsgebieten markiert und man will diese in eine Sammelrechnung umwandeln, so wird für jedes Verkaufsgebiet eine Rechnung erstellt.

Neu: sind mehrere Lieferscheine aus verschiedenen Verkaufsgebieten markiert und man will diese in eine Sammelrechnung umwandeln, so wird eine Sammelrechnung erstellt und die Warenbewegungen werden dem Verkaufsgebiet der Rechnung zugeordnet.

Nein: sind mehrere Lieferscheine aus verschiedenen Verkaufsgebieten markiert und man will diese in eine Sammelrechnung umwandeln, so wird eine Sammelrechnung erzeugt und die Warenbewegungen werden dem Verkaufsgebiet der Lieferscheine zugeordnet.
