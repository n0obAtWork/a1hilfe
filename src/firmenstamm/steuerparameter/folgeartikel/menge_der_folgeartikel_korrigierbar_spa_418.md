# Menge der Folgeartikel korrigierbar(SPA 418)

<!-- source: https://amic.de/hilfe/_SPA_418.htm -->

| Wert | Bedeutung |
| --- | --- |
| Ja | Die Menge in einem Folgeartikel kann abgeändert werden. |
| Nein | Die Menge in einem Folgeartikel kann nicht abgeändert werden. |

Hinweis: Diese SPA-Einstellung kann durch eine Angabe in der [Vorgangsunterkasse](../../../vorgangsabwicklung/formularzuordnung/spa.md) überschrieben werden!
