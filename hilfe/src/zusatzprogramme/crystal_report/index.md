# Crystal Report

<!-- source: https://amic.de/hilfe/crystalreport.htm -->

Crystal Report ist ein voll in A.eins integriertes Analysewerkzeug. Datenbereitstellung und Archivierung werden von A.eins übernommen. Neben den Standard-Reporten können auch eigene Reporte in das System integriert werden.
