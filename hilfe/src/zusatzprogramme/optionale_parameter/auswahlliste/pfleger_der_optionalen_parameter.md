# Pfleger der Optionalen Parameter

<!-- source: https://amic.de/hilfe/_opt_pfl.htm -->

<details>
<summary>Felder des Auswallistenpflegers</summary>

| Feld | Beschreibung |
| --- | --- |
| ID | ID der Option (wird automatisch generiert) |
| Nur global | Legt fest, ob jeder Bediener auf diese Option zugreifen kann |
| Option Name | Name der Option (F3-Auswahl) |
| Beschreibung | Beschreibung der Option |
| Bediener | Legt fest für welchen Bediener diese Option zur Verfügung steht (setzt das „Nur global“-Feld automatisch auf nein, sobald dieses Feld mit einem Bediener gefüllt wurde |
| Wert | Der Wert welcher von A.eins ausgelesen wird |

</details>
