# Belege mit Betrag 0 im Kontenblatt-Druck(SPA 540)

<!-- source: https://amic.de/hilfe/_SPA_540.htm -->

Mit diesem Steuerparameter kann festgelegt werden, ob Belege im KOKORE mit dem Betrag 0 angezeigt werden sollen. Der Steuerparameter wirkt nur bei neuen KOKOREs, bei bereits erstellten hat es keinen Einfluss mehr.

Folgende Werte stehen hier zur Verfügung.

| Wert | Bedeutung |
| --- | --- |
| 0 = Nein | Belege mit dem Betrag 0 werden nicht im KOKORE angezeigt. |
| 1 = Ja | Es werden auch Belege mit dem Betrag 0 angezeigt. |
| 2 = Ja (Keine Lieferungen zu Vorverkauf/-einkauf) | Es werden auch Belege mit dem Betrag 0 angezeigt. Desweiteren werden aber keine Lieferungen zu Vorverkaufs/-einkaufsbelegen angezeigt. |
