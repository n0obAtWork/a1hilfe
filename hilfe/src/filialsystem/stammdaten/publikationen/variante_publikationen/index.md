# Variante Publikationen

<!-- source: https://amic.de/hilfe/_BstFil_Publikation.htm -->

| **Felder** | |
| --- | --- |
| Publikation | Name der Publikationen |
| Eigenschaft | Zeigt die Eigenschaft einer Publikation:<br><ul><li>Amic-Standard</li><li>benutzerdefiniert</li></ul> |
| Anzahl Artikel | Anzahl der in der Publikation enthaltenen Artikel |
| Anzahl aktive Artikel | Anzahl der Artikel in einer Aktiven Publikation |

| **Funktionen** | |
| --- | --- |
| Pflege-Funktionen | Neu |
| Upgrade Tabellen | Weist alle noch nicht zugeordneten Tabellen der Publikation „**AMIC_nicht_zugeordnet**“ zu. |

| **Bereiche/Profile** | |
| --- | --- |
| Publikation wie | Ermöglicht Suche nach Publkationsnamen<br>**F3** ermöglicht die konkrete Auswahl und informiert über den Publikationstypen. |
| Eigenschaft | Ermöglicht Suche nach Publikationseigenschaft<br><ul><li>Amic-Standard</li><li>benutzerdefiniert</li></ul> |
