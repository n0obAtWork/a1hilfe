# Bediener aus Auftrag bei Teildisposition (SPA 811)

<!-- source: https://amic.de/hilfe/_SPA_811.htm -->

Hier kann festgelegt werden, ob bei der Teildisposition eines Lieferscheins aus der Auftragsbearbeitung der Bediener beibehalten werden soll. So soll sichergestellt werden, dass der Ansprechpartner der Auftragsannahme auch auf dem Lieferschein erscheint.
