# Makros bei Ansicht eines Vorgangs ausführen (SPA 862)

<!-- source: https://amic.de/hilfe/_SPA_862.htm -->

| Einstellungen |
| --- |
| Immer | Es werden immer alle Makros ausgeführt |
| Nur AIS-Makros | Es werden im Ansehen-Modus nur AIS-Makros ausgeführt |
| Nur Vorgangsmakros | Es werden im Ansehen-Modus nur Vorgangs-Makros ausgeführt |
| Nie | Es werden im Ansehen-Modus keine Makros ausgeführt |
