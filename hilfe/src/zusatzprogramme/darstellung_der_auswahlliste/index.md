# Darstellung der Auswahlliste

<!-- source: https://amic.de/hilfe/darstellungderauswahlliste.htm -->

Die Darstellung von Auswahllisten lässt sich über private Ableitungen oder über den Gestaltungsdialog festlegen. Der Gestaltungsdialog steht für alle Anwender zur Verfügung, die im Bedienerstamm das Kennzeichen „Auswahllistenadministrator“ auf „Ja“ oder „Temporär“ gesetzt haben. Temporär bedeutet, dass die im Gestaltungsdialog vorgenommenen Einstellungen nicht gespeichert werden und nur für die aktuelle A.eins-Sitzung gelten.

Der Gestaltungsdialog ist den privaten Ableitungen vorzuziehen.
