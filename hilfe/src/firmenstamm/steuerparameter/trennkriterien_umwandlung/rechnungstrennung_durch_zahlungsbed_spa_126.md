# Rechnungstrennung durch Zahlungsbed.(SPA 126)

<!-- source: https://amic.de/hilfe/_SPA_126.htm -->

Ja: sind mehrere Lieferscheine mit verschiedenen Zahlungsbedingungen markiert und man will diese in eine Sammelrechnung umwandeln, so wird für jede Zahlungsbedingung eine Rechnung erstellt.

Nein: sind mehrere Lieferscheine mit verschiedenen Zahlungsbedingungen markiert und man will diese in eine Sammelrechnung umwandeln, so wird eine Sammelrechnung erzeugt und die Warenbewegungen werden den Zahlungsbedingungen der Lieferscheine zugeordnet.
