# Vorgang erz. / edit. SHIFT+F6

<!-- source: https://amic.de/hilfe/_waage_vorgangerzeugen_editieren.htm -->

Diese Funktion erzeugt einen Vorgang und öffnet diesen zum Editieren. Vorgänge können nicht im Stapel erzeugt werden, sondern nur für jeden Waagedatensatz einzeln. Bevor der Vorgang zum Bearbeiten geöffnet wird, muss darauf gewartet werden, dass der Beleg im Mandantenserver verarbeitet worden ist. Während des Wartens wird abgefragt, ob man warten möchte bis der Beleg freigegeben worden ist. Wird diese Frage verneint, so wird der Beleg nicht im bearbeiten Modus geöffnet.

Der Vorgang ist trotzdem erzeugt worden.
