# Hauptmenü-Favoritenbereich

| Element | Tastatur | Beschreibung |
| --- | --- | --- |
| Icon | ![favorit24](../../../ImagesExt/image8_1484.png "favorit24") | |
| Beschriftung | Favoriten(X) | X steht für die Anzahl der Favoriten |

Favoriten sind vom A.eins-Bediener individuell ausgewählte Menü-Punkt zwecks schnelleren Zugriffs auf häufig verwendete Menüpunkte des Arbeitsbereiches.

Zusätzliche Informationen zum Favoriten-Handling finden sich unter [Menüpunkte](./hauptmenue_menuepunkte/index.md) und [Kontextmenüs der Menüpunkte](./hauptmenue_menuepunkte/kontextmenues_der_menuepunkte.md).

Für Pflege und Sortierung der Favoriten siehe [Menü-Favoriten](../menue_favoriten/index.md) und [Menü-Favoriten-Sortierung](../menue_favoriten/menue_favoriten_sortierung.md).
