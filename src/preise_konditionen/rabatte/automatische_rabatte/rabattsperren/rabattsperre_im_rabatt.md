# Rabattsperre im Rabatt

<!-- source: https://amic.de/hilfe/rabattsperreimrabatt.htm -->

Eine Rabattsperre kann auch im Rabatt selbst eingerichtet werden. So lassen sich eingerichtete Rabatte vorübergehend abschalten. [Siehe Allgemeine Rabatte](../allgemeine_rabatte_zuordnung_von_rabattgruppen_und_rabattkla.md).
