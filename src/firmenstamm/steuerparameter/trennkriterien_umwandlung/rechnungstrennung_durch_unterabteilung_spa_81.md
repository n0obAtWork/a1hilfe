# Rechnungstrennung durch Unterabteilung(SPA 81)

<!-- source: https://amic.de/hilfe/_SPA_81.htm -->

Trennen: sind mehrere Lieferscheine mit verschiedenen Unterabteilungen markiert und man will diese in eine Sammelrechnung umwandeln, so wird für jede Unterabteilung eine Rechnung erstellt.

Neu: sind mehrere Lieferscheine mit verschiedenen Unterabteilungen markiert und man will diese in eine Sammelrechnung umwandeln, so wird eine Sammelrechnung erstellt und die Warenbewegungen werden der Unterabteilung der Rechnung zugeordnet.

Nein: sind mehrere Lieferscheine aus verschiedenen Unterabteilungen markiert und man will diese in eine Sammelrechnung umwandeln, so wird eine Sammelrechnung erzeugt und die Warenbewegungen werden den Unterabteilungen der Lieferscheine zugeordnet.
