# Artikelkopie aus Sortimentslager

<!-- source: https://amic.de/hilfe/artikelkopieaussortimentslager.htm -->

Wenn der gewünschte Artikel im aktuellen Lager nicht angelegt ist, jedoch im Sortimentslager definiert wurde, so kann der Artikel von dort kopiert werden. Rufen Sie dazu die Funktion „Kopie von Sortimentslager“ auf. Sie können dann einen Artikel aus dem Sortimentslager auswählen und auf das aktuelle Lager kopieren. Eine Nachbearbeitung des gerade kopierten Artikels können Sie dort vornehmen.

Nach dem Verlassen der Artikelkopie wird der Artikel in die Erfassungsmaske übernommen.
