# Rechnungstrennung durch Versandadresse(SPA 92)

<!-- source: https://amic.de/hilfe/_SPA_92.htm -->

Trennen: sind mehrere Lieferscheine mit verschiedener Versandadresse markiert und man will diese in eine Sammelrechnung umwandeln, so wird für jede Versandadresse eine Rechnung erstellt.

Neu: sind mehrere Lieferscheine mit verschiedenen Versandadressen markiert und man will diese in eine Sammelrechnung umwandeln, so wird eine Sammelrechnung erstellt und die Warenbewegungen werden der Versandadresse der Rechnung zugeordnet.

Nein: sind mehrere Lieferscheine mit verschiedenen Versandadressen markiert und man will diese in eine Sammelrechnung umwandeln, so wird eine Sammelrechnung erzeugt und die Warenbewegungen werden den Versandadressen der Lieferscheine zugeordnet.
