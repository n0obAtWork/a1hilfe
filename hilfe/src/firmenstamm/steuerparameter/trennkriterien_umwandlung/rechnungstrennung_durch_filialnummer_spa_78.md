# Rechnungstrennung durch Filialnummer(SPA 78)

<!-- source: https://amic.de/hilfe/_SPA_78.htm -->

Trennen: sind mehrere Lieferscheine aus verschiedenen Filialen markiert und man will diese in eine Sammelrechnung umwandeln, so wird für jede Filiale eine Rechnung erstellt.

Neu: sind mehrere Lieferscheine aus verschiedenen Filialen markiert und man will diese in eine Sammelrechnung umwandeln, so wird eine Sammelrechnung erstellt und die Warenbewegungen werden der Filiale der Rechnung zugeordnet.

Nein: sind mehrere Lieferscheine aus verschiedenen Filialen markiert und man will diese in eine Sammelrechnung umwandeln, so wird eine Sammelrechnung erzeugt und die Warenbewegungen werden der Filiale der Lieferscheine zugeordnet.
