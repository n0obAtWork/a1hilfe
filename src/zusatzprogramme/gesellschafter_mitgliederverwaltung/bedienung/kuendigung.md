# Kündigung

<!-- source: https://amic.de/hilfe/kndigung.htm -->

Um Anteile zu kündigen wird ein Gesellschafter in der Auswahlliste ausgewählt mit **F5** zum ***Bearbeiten*** geöffnet und die Funktion ***Kündigung*** **F7** ausgewählt.

Die Funktion **Kündigen** **F7** öffnet folgende Eingabefelder:

<div class="table-wrapper">
  <table>
    <tbody>
      <tr>
        <td colspan="2">
          <p><strong>Felder</strong></p>
        </td>
      </tr>
      <tr>
        <td>
          <p>Anzahl</p>
        </td>
        <td>
          <p>Hier wird die Anzahl der Anteile eingetragen.</p>
        </td>
      </tr>
      <tr>
        <td>
          <p>Anteilstyp</p>
        </td>
        <td>
          <p>Hier wird der Anteilstyp <strong>F3</strong> Freiwillig oder Pflicht eingetragen.</p>
        </td>
      </tr>
      <tr>
        <td>
          <p>Zeichnungsdatum</p>
        </td>
        <td>
          <p>Das Zieldatum ist satzungsabhängig das geplante Auszahlungsdatum.</p>
        </td>
      </tr>
      <tr>
        <td>
          <p>Bemerkung zum Vorgang</p>
        </td>
        <td>
          <p>Hier kann eine Bemerkung zum Vorgang eingetragen werden. (60 Zeichen)</p>
        </td>
      </tr>
      <tr>
        <td>
          <p>Belegdatum</p>
        </td>
        <td>
          <p>In diesem Feld wird das Eingangsdatum der Kündigung eingetragen.</p>
        </td>
      </tr>
    </tbody>
  </table>
</div>

Es wird vom Programm verhindert, dass die Anzahl der gekündigten Anteile die der gezeichneten Anteile je Typ überschreitet.
