# Varianten

<!-- source: https://amic.de/hilfe/varianten.htm -->

Es gibt in diesem Bereich vier Varianten.

| Varianten | Name | Bedeutung |
| --- | --- | --- |
| Variante 1 | Mehrmandant Einstamm | Einstellungen für den Server und Empfänger/Alt |
| Variante 2 | Mehrmandant Transfer | Einstellungen für die angebundenen Relationen |
| Variante 3 | XML Transfertabelle | Inhalt der Proxytabelle |
| Variante 4 | Server | Einrichtung und Löschung der Server |
