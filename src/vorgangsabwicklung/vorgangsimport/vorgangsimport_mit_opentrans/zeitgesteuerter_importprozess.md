# Zeitgesteuerter Importprozess

<!-- source: https://amic.de/hilfe/_OTVorgimport_evt.htm -->

Sie können Makros zeitgesteuert über die Einrichtung eines Events starten. So können Sie auch mit einem Importmakro verfahren. Bedingung ist allerdings, dass der Mandantenserver durchgehend oder zumindest während der geplanten Ausführungszeit aktiv ist.

Mehr zum zeitgesteuerten Aufruf eines Makros erfahren Sie unter dem Stichwort „Zyklischer Aufruf“ in der Abteilung C#-Makros.
