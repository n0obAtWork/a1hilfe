# Crw/Vbs

<!-- source: https://amic.de/hilfe/crwvbs.htm -->

Hier hat man in der F3-Auswahl verschiedene Möglichkeiten:

\-------------------

Dies ist der Standardformulardruck.

CRW auf Basis Formular mit CRW Drucker

Man erstellt ein Formular und einen zugehörigen Report. In dem Report werden die Zeilen aus der Formulareinrichtung angedruckt. Man kann durch die Reportbearbeitung Dinge ändern (z.B. die Schrift im Mittelteil der Zeilen) oder zufügen.  
Dieser Report auf Basis eines Formulars wird mit dieser Einstellung auf dem Drucker des Reports ausgedruckt. Unter dem Direktsprung [lst] kann man den Listen/Reports mit Hilfe der Druckerauswahl Drucker zuordnen.

CRW auf Basis Formular mit DRZ/VRGD Drucker

Man erstellt ein Formular und einen zugehörigen Report. In dem Report werden die Zeilen aus der Formulareinrichtung angedruckt. Man kann durch die Reportbearbeitung Dinge ändern (z.B. die Schrift im Mittelteil der Zeilen) oder zufügen.  
Dieser Report auf Basis eines Formulars wird mit dieser Einstellung auf dem Drucker der Druckerzuordnung [DRZ] oder der Vorgangsdruckklasse [VRGD] des Kunden gedruckt.

VBS Ausgabe

Es wird das unter Script angegebene VBS Script verwendet.

AMIC Etikettendruck mit Druckerabfrage

Es wird ein unter dem AMIC Etikettendruck [etidr] erstelltes Formular auf dem abgefragten Drucker gedruckt. Der Inhalt des Feldes Funktionsident für das Etidr Formular muss hier als Name angegeben werden.

AMIC Etikettendruck auf dem im ETIDR-Formular eingerichteten Drucker

Es wird ein unter dem AMIC Etikettendruck [etidr] erstelltes Formular auf dem für dieses Formular dort angegebenem Drucker gedruckt. Der Inhalt des Feldes Funktionsident für das Etidr Formular muss hier als Name angegeben werden.

AMIC Etikettendruck auf dem DRZ/VRGD Drucker

Es wird ein unter dem AMIC Etikettendruck [etidr] erstelltes Formular auf dem Drucker der Druckerzuordnung [DRZ] oder der Vorgangsdruckklasse [VRGD] des Kunden gedruckt.  
Der Inhalt des Feldes Funktionsident für das Etidr Formular muss hier als Name angegeben werden.

CRW mit VDCRYSTA und CRW Drucker

Man legt einen Crystal Report an und entscheidet sich hier, dass dieser auf dem Drucker des Reports gedruckt wird. Unter dem Direktsprung [lst] kann man den Listen/Reports mit Hilfe der Druckerauswahl Drucker zuordnen.

CRW mit VDCRYSTA und DRZ/VRGD Drucker

Man legt einen Crystal Report an und entscheidet sich hier, dass dieser auf dem Drucker der Druckerzuordnung [DRZ] oder der Vorgangsdruckklasse [VRGD] des Kunden gedruckt werden soll.
