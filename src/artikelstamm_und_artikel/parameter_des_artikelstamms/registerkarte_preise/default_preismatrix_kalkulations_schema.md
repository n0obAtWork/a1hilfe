# Default-Preismatrix, Kalkulations-Schema

<!-- source: https://amic.de/hilfe/_defaultpreismatrixka.htm -->

Im Artikel wird hinterlegt, welche Preismatrix aktiv ist; dies könnten in Abhängigkeit vom Lager unterschiedliche sein. Hier im Artikelstamm kann eine Vorbelegung vorgenommen werden, so dass die Eingabe im Artikel nicht mehr erforderlich ist.

Für die Kalkulation von Listenpreisen steht ein mächtiges System zur Verfügung. Die Verfahren werden innerhalb der Preiskalkulation festgelegt; hier kann das für diesen Artikel gewünschte zugeordnet werden.
