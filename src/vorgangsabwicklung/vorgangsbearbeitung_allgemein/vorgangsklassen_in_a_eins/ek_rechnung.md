# EK-Rechnung

<!-- source: https://amic.de/hilfe/ekrechnung.htm -->

EK-Rechnungen entstehen durch Neuerfassung oder Umwandlung aus Anfragen / Bestellungen / EK-Lieferscheinen. EK-Rechnungen werden als Vorgang gespeichert, auf sie kann in Nachfolgevorgängen zugegriffen werden; die Menge wird verbucht und erhöht den Bestand; der EK-Umsatz erhöht sich. A.eins stellt folgende Bearbeitungsfunktionen zur Verfügung:

| Funktionsname | Funktion |
| --- | --- |
| ***Erfassen F 8*** | Erfassung einer neuen EK-Rechnung<br> |
| ***Stapelverarbeitung*** | Übernahme eines oder mehrerer EK-Rechnungen in einen Bearbeitungsstapel<br> |
| ***Erstdruck F9*** | Erstdruck einer EK-Rechnung<br> |
| ***Formulardruck F10*** | Wiederholungsdruck<br> |
| ***Lieblingsdruckerdruck*** | Auswahl eines anderen Druckers / Zuordnung eines anderen Formulars<br> |
| ***Korrektur F5*** | Korrektur einer EK-Rechnung<br> |
| ***Ansicht F6*** | EK-Rechnung im Ansicht-Modus öffnen<br> |
| ***Schnellkorrektur*** | Plandatum und Menge ändern<br> |
| ***Kopieren CF8*** | Kopieren der EK-Rechnung für einen auszuwählenden Kunden<br> |
| ***Vorschau F11*** | Druckvorschau<br> |
| ***Stornieren F7*** | Stornieren (Löschen) der EK-Rechnung<br> |
| ***Storno EK-Rechnung*** | Erfassung eines Stornobeleges<br> |
| ***Freigabe/Sperren*** | Freigabe / Sperren für weitere Bearbeitung<br> |
| ***Gutschrift aus EK-Rechnung*** | Umwandeln in eine Gutschrift<br> |
| ***Sammel-EGU aus E-Rechnungen*** | Umwandeln mehrerer EK-Rechnungen in eine EK-Gutschrift<br> |
| ***Transportauftrag*** | Auftrag einer Spedition zuordnen<br> |
| ***Archiv ansehen*** | Anzeige archivierter Vorgänge<br> |
| ***Partien nachtragen*** | nachträgliche Zuordnung von Partien<br> |
| ***Wiedervorlage CF9*** | Vorgang mit einem Wiedervorlagevermerk versehen<br> |
| ***Arbeitsregel ändern*** | manuelle Änderung von Weiterverarbeitungsparametern<br> |
| ***FiBu Übertrag*** | Übergabe an die Finanzbuchhaltung<br> |
| ***FiBu-Eintrag zurücknehmen***<br> | Siehe [FiBu – Eintrag zurücknehmen](../../vorgangsbearbeitung/fibu_eintrag_zuruecknehmen.md)<br> |
