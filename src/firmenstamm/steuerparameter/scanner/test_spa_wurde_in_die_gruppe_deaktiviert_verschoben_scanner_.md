# Test SPA wurde in die Gruppe deaktiviert verschoben Scanner-Scanner Testmitschrift aktivieren (SPA 789)

<!-- source: https://amic.de/hilfe/_SPA_789.htm -->

Die gescannten Befehle einer Sitzung (eines Scanners) können in eine Testtabelle geschrieben werden, um sie später zu Validierungszwecken wieder aufzurufen. Im Optionswert wird der Name der Mitschrift gesetzt, der Schlüssel steuert die individuelle Scanner-Zuordnung (im Regelfall die IP Adresse). Der Wert steuert den Start der Mitschrift (Ja/Nein).
