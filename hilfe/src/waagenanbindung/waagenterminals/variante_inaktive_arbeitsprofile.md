# Variante Inaktive Arbeitsprofile

<!-- source: https://amic.de/hilfe/_waageprofil_inaktiveprofile.htm -->

Nachdem ein Waagenprofil aus den Mustervorlagen verschoben worden ist, kann dieses an dieser Stelle bearbeitet werden. Des Weiteren können an dieser Stelle eigene Waagenprofile erstellt werden.

Folgende Funktionen stehen zur Verfügung

1. Anlegen eines neuen Waagenprofils

2. Ändern eines Waagenprofils

3. Ansehen eines Waagenprofils

4. Löschen eines Waagenprofils
