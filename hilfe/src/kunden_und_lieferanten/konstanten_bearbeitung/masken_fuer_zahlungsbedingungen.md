# Masken für Zahlungsbedingungen

<!-- source: https://amic.de/hilfe/maskenfrzahlungsbedingungen.htm -->

Direktsprung [ZBM]

Hier wird die Einrichtung der optischen Aufbereitung für den Ausdruck der Zahlungsbedingung vorgenommen. Sie besteht aus Texten und Platzhaltern für Parameter und errechnete Werte.

| Feldname | Beschreibung |
| --- | --- |
| Nummer | Nummer der Aufbereitungsmaske für die Zahlungsbedingung |
| Bezeichnung | Name der Aufbereitungsmaske |
| Text | 4 Zeilen für den Text mit Platzhaltern |

**Aktuell existieren folgende** [Platzhalter](./platzhalter_fuer_zahlungsbedingungen.md)**.**
