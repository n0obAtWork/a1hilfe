# Vieraugenprinzip Zahlungen(SPA 716)

<!-- source: https://amic.de/hilfe/_SPA_716.htm -->

Mit diesem Steuerparameter kann eingestellt werden, ob für das DTA Verfahren das Vieraugenprinzip benutzt werden soll. Wird der Steuerparameter auf **Ja** gestellt so gibt es unter Zahlungen bearbeiten einen neue Variante.
