# Systemmeldungen

<!-- source: https://amic.de/hilfe/systemmeldungen.htm -->

Es wird empfohlen die [Systemmeldung](../menue/systemmeldungen/index.md) ***AMIC_WARNUNG_AEINSFEHLP*** zu aktivieren.

Mit Hilfe von [Protokollkontrolle](../../firmenstamm/firmenkonstanten/bedienerwesen_bediener_bedienerklassen_und_erfasser/bedienerstamm/bedienerstamm_pfleger.md#Protokollkontrolle) können Sie festlegen welcher Bediener über die Systemmeldung benachrichtigt werden soll.
