# Bewertung

<!-- source: https://amic.de/hilfe/_bewertung.htm -->

Für die laufende Bewertung des Warengeschäftes stehen verschiedene Verfahren zur Verfügung. Hierauf wird nachfolgend eingegangen. Daneben spielen organi­satorische Faktoren eine wichtige Rolle; auf die damit verbundenen Prinzipien wird im Anschluss eingegangen.
