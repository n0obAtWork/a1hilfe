# Fiktive Menge bei Frachtermittlung(SPA 530)

<!-- source: https://amic.de/hilfe/_SPA_530.htm -->

In der Warenbewegung kann eine fiktive Menge (für Mengenbezüge) hinterlegt werden. Die fiktive Menge wird anstelle der eigentlichen Menge zur Ermittlung von Zu-/Abschlägen und Rabatten herangezogen.

Ist die Einstellung „Ja“, so wird die fiktive Menge auch bei der Frachtermittlung berücksichtigt!
