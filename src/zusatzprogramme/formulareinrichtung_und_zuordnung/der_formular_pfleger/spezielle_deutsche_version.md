# Spezielle deutsche Version

<!-- source: https://amic.de/hilfe/spezielledeutscheversion.htm -->

Zu Testzwecken wurde eine intern fest verdrahtete deutsche Verbalversion zur Verfügung gestellt. Hierzu kann man im Formularstamm unter ‚[DB Fkt. Num Text](./db_fkt_num_text.md)’ folgendes angeben:

Deutsch,Euro,Cent

Es wird der Betrag ohne Vorzeichen dargestellt. Wird „Cent“ nicht angegeben, werden nur volle Beträge gedruckt.
