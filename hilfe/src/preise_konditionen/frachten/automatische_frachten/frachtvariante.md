# Frachtvariante

<!-- source: https://amic.de/hilfe/_Frachten_FRAV.htm -->

Nebenbuchhaltungen > Frachtverwaltung > Frachttabellen **[FRA]** > Frachtvariante **[FRAV]**

Frachtvarianten geben an, auf welche Weise Waren verschickt werden. Hier kann z.B. Brief, Päckchen, Paket oder Bahn, Schiff, Flugzeug o.ä. eingerichtet sein.
