# Abkündigung: JRCON/GS MDE – Einzelhandelslösung MOBILAR (OFFLINE)

<!-- source: https://amic.de/hilfe/_ab32041.htm -->

Seit letztem Jahr haben wir für A.eins eine Online arbeitende Einzelhandels-MDE Lösung. Die obige Lösung war auf Basis von Windows CE entwickelt worden. Der Support für dieses Betriebssystem wurde von Microsoft komplett eingestellt. Ebenso gibt es keine aktuellen Entwicklungsumgebungen mit denen man Anpassungen an diesen Lösungen vornehmen könnte. Wir kündigen diese Lösung ab. Sofern Sie Interesse an der neuen MDE-Lösung haben, sprechen Sie unseren Vertrieb an.
