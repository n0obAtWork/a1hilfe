# Variante Inhalte von Publikationen

<!-- source: https://amic.de/hilfe/_InhaltPublikation.htm -->

| **Felder** | |
| --- | --- |
| Publikation | Name der Publikationen |
| Eigenschaft | Zeigt die Eigenschaft einer Publikation:<br><ul><li>-&nbsp;&nbsp;&nbsp;&nbsp; Amic-Standard</li><li>-&nbsp;&nbsp;&nbsp;&nbsp; benutzerdefiniert</li></ul> |
| Artikel | Name des in der Publikation angegebenen Artikels.<br>(Name der Tabelle) |
| PK vorhanden | JA/Nein<br>Zeigt an, ob der Artikel ( Tabelle ) über Primary Key-Felder verfügt.<br>Sollte hier ein „Nein“ verzeichnet sein so ist dieser Artikel im Falle der Verwendung in Publikationen, welche in Replikationen verwendet werden sollen, mit einem gültigen Primary Key zu versehen! |
| Publikations-Artikel aktiv | Zeigt an, ob der Artikel in einer aktiven Publikation enthalten ist.<br>Suche nach:<br><ul><li>-&nbsp;&nbsp;&nbsp;&nbsp; ja</li><li>-&nbsp;&nbsp;&nbsp;&nbsp; nein</li><li>-&nbsp;&nbsp;&nbsp;&nbsp; egal</li></ul> |
| Partitionsbedingung | Subscribe by |
| Restriktionsbedingung | where-Klausel |
| Lock-Status | Zeigt an, ob die Tabelle des Artikels gerade gesperrt ist.<br>Suche nach:<br><ul><li>-&nbsp;&nbsp;&nbsp;&nbsp; ja</li><li>-&nbsp;&nbsp;&nbsp;&nbsp; nein</li><li>-&nbsp;&nbsp;&nbsp;&nbsp; egal</li></ul> |
| Lock-Bediener | Zeigt eine Liste der Bediener, welche die Tabelle eines Artikels gerade sperren. |

| **Funktionen** | |
| --- | --- |
| Pflege-Funktionen | Neu, Ändern, Löschen<br>[Pfleger Publikationen](./pfleger_publikationen.md) |
| Zuordnung vergleichen | [Zuordnung vergleichen](./zuordnung_vergleichen.md) |

| **Bereiche/Profile** | |
| --- | --- |
| Publikation wie | Ermöglicht Suche nach Publikationsnamen<br>**F3** ermöglicht die konkrete Auswahl und informiert über den Publikationstypen. |
| Eigenschaft | Ermöglicht Suche nach Publikationseigenschaft<br><ul><li>-&nbsp;&nbsp;&nbsp;&nbsp; Amic-Standard</li><li>-&nbsp;&nbsp;&nbsp;&nbsp; Benutzerdefiniert</li></ul> |
| Artikel | Ermöglicht Suche nach Artikeln |
| Artikel aktiv | Suche nach<br><ul><li>-&nbsp;&nbsp;&nbsp;&nbsp; ja</li><li>-&nbsp;&nbsp;&nbsp;&nbsp; nein</li><li>-&nbsp;&nbsp;&nbsp;&nbsp; egal</li></ul> |
| Lock-Status | Suche nach<br><ul><li>-&nbsp;&nbsp;&nbsp;&nbsp; ja</li><li>-&nbsp;&nbsp;&nbsp;&nbsp; nein</li></ul> |
| Lock-Bedienerliste | Suche nach Bedienerkurzbezeichnungen |
