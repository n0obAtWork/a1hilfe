# Gutschrift

<!-- source: https://amic.de/hilfe/gutschrift.htm -->

Gutschriften entstehen durch Neuerfassung oder Umwandlung aus Rechnungen. Gutschriften werden als Vorgang gespeichert; die Menge wird verbucht und erhöht den Bestand; der Umsatz vermindert sich. A.eins stellt folgende Bearbeitungsfunktionen zur Verfügung:

| Funktionsname | Funktion |
| --- | --- |
| ***Erfassen F 8*** | Erfassung einer neuen Gutschrift<br> |
| ***Stapelverarbeitung*** | Übernahme einer oder mehrerer Gutschriften in einen Bearbeitungsstapel<br> |
| ***Erstdruck F9*** | Erstdruck einer Gutschrift<br> |
| ***Formulardruck F10*** | Wiederholungsdruck<br> |
| ***Lieblingsdruckerdruck*** | Auswahl eines anderen Druckers / Zuordnung eines anderen Formulars<br> |
| ***Korrektur F5*** | Korrektur einer Gutschrift<br> |
| ***Ansicht F6*** | Gutschrift im Ansicht-Modus öffnen<br> |
| ***Kopieren CF8*** | Kopieren der Gutschrift für einen auszuwählenden Kunden<br> |
| ***Vorschau F11*** | Druckvorschau<br> |
| ***Stornieren F7*** | Stornieren (Löschen) der Gutschrift<br> |
| ***Freigabe/Sperren*** | Freigabe / Sperren für weitere Bearbeitung<br> |
| ***Archiv ansehen*** | Anzeige archivierter Vorgänge<br> |
| ***Partien nachtragen*** | nachträgliche Zuordnung von Partien<br> |
| ***Wiedervorlage CF9*** | Vorgang mit einem Wiedervorlagevermerk versehen<br> |
| ***Arbeitsregel ändern*** | manuelle Änderung von Weiterverarbeitungsparametern<br> |
| ***FiBu Übertrag*** | Übergabe an die Finanzbuchhaltung<br> |
| ***FiBu-Eintrag zurücknehmen*** | Siehe [FiBu – Eintrag zurücknehmen](../../vorgangsbearbeitung/fibu_eintrag_zuruecknehmen.md)<br> |
