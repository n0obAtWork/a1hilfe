# Bedienerklassen

Hauptmenü > Administration > Firmenkonstanten > Bedienerklassen 

oder Direktsprung **[BDKL]** 

Bedienerklassen fassen Mitarbeiter mit gleichen Rechten zusammen; sie dienen also somit einer Strukturierung der Rechtevergabe.

<details>
  <summary>Felder der Bedienerklasse:</summary>

  In der Variante „Bedienerklasse“ werden folgende Felder behandelt:

  | Felder | Beschreibung |
  | :----- | :----------- |
  | Bedienerklasse | <p>Im Normalfall eine nicht-negative Zahl die Bediener gruppiert.</p><p>Als Spezialfälle gibt es technische Bedienerklassen Bedienerklasse: Defaultklasse.</p> |
  | Bezeichnung | Bezeichnung der Bedienerklasse |
  | Betriebsstätte | <p>Bei angeschlossenem Filialsystem Zuordnung der Bedienerklasse zur Betriebsstätte.</p><p>Standard: 0, ohne Filiale</p> |
  | Bezeichnung Betriebsstätte | Bezeichnung der Betriebsstätte |
  | Gesperrt | <p>JA/NEIN</p><p>Login-Sperre aller Bediener dieser Bedienerklasse</p> |
  | Sicherheitsklasse | Sicherheitsklasse |
  | Controllerklasse | Controllerklassen werden gelb hervorgehoben. |
  | Bedienerlisten-Info | <p>Liste der zugehörigen Bediener.</p><p>Angezeigt werden nur Bediener, die weder gelöscht oder gesperrt sind, und die in der Datenbank als User verzeichnet sind.</p> |
</details>

<details>
  <summary>Suchmöglichkeiten der Bedienerklasse:</summary>

  |    | Suchkriterien |
  | :- | :------------ |
  | Bedienerklasse | von … bis … |
</details>

<details>
  <summary>Funktionen der Bedienerklasse</summary>

  In der Variante „Bedienerklasse“ stehen folgende Funktionen zur Verfügung.

  | Funktionen | Beschreibung |
  | :--------- | :----------- |
  | Ändern **(F5)**, Ansehen **(F6)**, Löschen **(F7)**, Neu **(F8)** | <p>Ruft den Bedienerklasse: Pfleger auf. Innerhalb dieser Funktion lässt sich mit Speichern unter… eine neue Bedienerklasse erzeugen.</p><p>Dabei werden dann auch sämtliche Schutzeinstellungen der Funktionen (Zugriffsrechte Funktionen) übernommen.</p> |
  | EPAs zeigen **(F10)** | <p>Individuelle Steuerungen von Abläufen können in Anwendungen über Einrichterparameter (EPA) vorgenommen werden.</p><p>Diese Funktion ruft die entsprechende Anwendung zur Ansicht und Pflege der Einrichterparameter auf.</p><p>Hauptmenü  Administration  Steuerung  EPAs zeigen</p><p>Direktsprung [EPAZ]</p> |
</details>
