# Belegkopie (Speichern unter)

<!-- source: https://amic.de/hilfe/belegkopiespeichernunter.htm -->

Die „Speichern unter“ Funktion nutzt die Schnellerfassung, um bequem Belege, die schon einmal erfasst worden sind, als Vorlage für einen anderen Beleg zu nutzen. In den obigen Beschreibungen ist mehrfach darauf hingewiesen worden, dass es sich nicht um eine Vorgangskopie handelt, sondern nur um eine Vorlage für einen neuen Beleg.
