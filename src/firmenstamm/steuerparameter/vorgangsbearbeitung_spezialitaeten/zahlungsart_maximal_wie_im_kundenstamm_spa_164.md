# Zahlungsart maximal wie im Kundenstamm(SPA 164)

<!-- source: https://amic.de/hilfe/_SPA_164.htm -->

Mittels dieses Parameters kann die Zahlungsartvorbelegung frei bei der Belegerfassung überschrieben werden.

Ja: Es kann nur eine kleinere Zahlungsart vorgeschlagen eingetragen werden (Bsp.: Eintrag ist 4, Änderung nur auf 1-3 möglich, nicht auf 5)

Nein: Keine Einschränkung bei der Vergabe.
