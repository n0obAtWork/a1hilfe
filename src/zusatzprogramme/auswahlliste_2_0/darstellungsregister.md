# Darstellungsregister

<!-- source: https://amic.de/hilfe/darstellungsregister.htm -->

Das Darstellungsregister ist in zwei große Blöcke und einen Block mit der Hilfe-Funktion eingeteilt. Der linke Block mit den Funktionen ***Sortierung***, ***Farben***, usw. bezieht sich jeweils auf eine Anwendungsvariante, der zweite Block gilt für die Auswahlliste allgemein. Diese Einstellungen werden gespeichert und stehen immer wieder so zur Verfügung. Die Funktionen zum Sortieren, Gruppieren und Filtern in der [Datentabelle](./datentabelle.md) werden hingegen nicht gespeichert und bleiben nur so lange aktiv, solang man in der Variante steht.

| | **Bedeutung** |
| --- | --- |
| Sortierung | Siehe „[Sortierung der Auswahlliste](../darstellung_der_auswahlliste/sortierung_der_auswahlliste.md)“  
 |
| Farben | Siehe „[Farbgestaltung der Auswahlliste](../darstellung_der_auswahlliste/farbgestaltung_der_auswahlliste/index.md)“  
 |
| Spalten | Siehe „[Feldauswahl der Auswahlliste](../darstellung_der_auswahlliste/feldauswahl_der_auswahlliste.md)“  
 |
| SQL-Variablen | Siehe „[SQL-Variablen in der Auswahlliste verwenden](../darstellung_der_auswahlliste/sql_variablen_in_der_auswahlliste_verwenden.md)“  
 |
| Summen | Siehe „[Summierung in der Auswahlliste](../darstellung_der_auswahlliste/summierung_in_der_auswahlliste/index.md)“  
 |
| Vorbelegungen | Die Funktionalität „[Vorbelegungen](../darstellung_der_auswahlliste/standardvorbelegung_in_der_auswahlliste_definieren_nur_auswa/vorbelegung_im_gestaltungsdialog.md)“ steht bisher nur für das auf der neuen Auswahlliste basierende Archiv und den auf der Auswahlliste basierenden [Reporten](./reporte_verwalten.md) zur Verfügung.  
 |
| Ansichten Verwalten | Dies ist eine Möglichkeit der neuen Auswahlliste für alle Anwender die Darstellung veralten, ohne private Ableitungen zu bilden.   
 |
| Schriftart auswählen | Hier kann die Schriftart, der Schriftschnitt und die Schriftgrößer eingestellt werden. Die Standardeinstellung lautet Verdana-9.  
 |
| NULL-Darstellung auswählen | Datenbankfelder, in die nie ein Wert eingegeben wurde und die keinen Default-Wert haben, haben den Wert NULL, der nicht mit der Zahl 0 verwechselt werden darf. Hier kann man zwischen zwei Werten wählen: &lt;null> oder . (Punkt).  
Für diese Auswahl existiert im Menü die gleichnamige Funktion „NULL-Darstellung auswählen“. Wird diese weggeschützt, so steht diese Auswahl nicht zur Verfügung.  
 |
| Design auswählen  
 | Hier kann zwischen verschiedenen Farbgebungen der Auswahlliste ausgewählt werden.  
Für diese Auswahl existiert im Menü die gleichnamige Funktion „Design auswählen“. Wird diese weggeschützt, so steht diese Auswahl nicht zur Verfügung.  
 |
| Excel-Ausgabe auswählen | Hier kann eingestellt werden, mit welcher Art der Excel-Export ausgeführt werden soll. Dabei ist mit der Einstellung folgendes gemeint:  
    
    
• **Excel xlsx-Format** exportiert die Daten im xlsx-Format, das von allen Excel Versionen ab Excel 2007 gelesen werden kann. Dies ist die Standardeinstellung, wenn nichts anderes gewählt wurde.  
• **Excel xls-Format** exportiert die Daten im xls-Format, das von älteren Excel Versionen gelesen werden kann. Hier ist die Anzahl der Datensätze auf 65536 beschränkt.  
   
Für diese Auswahl existiert im Menü die gleichnamige Funktion „Excel-Ausgabe auswählen“. Wird diese weggeschützt, so steht diese Auswahl nicht zur Verfügung.  
 |
| Gruppieren An/Aus | Blendet den Bereich, der mit „Zur Gruppierung hier ablegen“ bezeichnet ist, zum individuellen Gruppieren aus. Diese Einstellung wird pro Variante und Benutzer gespeichert. Will man für ganze Benutzergruppen diese Funktion ausblenden, so geschieht dies durch Sperren der Funktion „Gruppieren-Bereich“.  
 |
| Filterzeile An/Aus  
   
 | Blendet die Filterzeile aus. Diese Einstellung wird pro Variante und Benutzer gespeichert. Will man für ganze Benutzergruppen diese Funktion ausblenden, so geschieht dies durch Sperren der Funktion „Filter“.  
 |
| Funktionstasten An/Aus  
 | Die Anzeige der Funktionstasten auf dem Menüband kann hier ein- oder ausgeschaltet werden. |
| Zeige SQL  
 | Zeig das SQL-Statement mit zugehörigem Datenzugriffs-Plan an, welches für die aktuelle Variante verwendet wurde.  
 |
| Private Variante  
 | Man kann diese Variante privat ableiten.  
 |
| SQL-Text pflegen  
 | Schnelle Möglichkeit, den zugrundeliegenden SQL-Text zu bearbeiten.  
 |
| Entwickler-Konfiguration  
 | Hier wird eine Maske aufgerufen, in der die Konfiguration der Auswahlliste und deren zugehörige Varianten vorgenommen werden kann.  
 |
| Variante als Favorit dem Menü zuordnen  
 | Diese Variante erscheint für den Benutzer im Hauptmenü auf dem Register „Favoriten“.  
 |
| Hilfe Auswahlliste | Diese Dokumentation.  
 |
