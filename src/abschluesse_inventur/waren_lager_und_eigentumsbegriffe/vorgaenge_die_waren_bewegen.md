# Vorgänge, die Waren bewegen

<!-- source: https://amic.de/hilfe/vorgngediewarenbewegen.htm -->

<div class="table-wrapper">
  <table>
    <tbody>
      <tr>
        <td colspan="2">
          <p><strong>Einkauf / Verkauf</strong></p>
        </td>
      </tr>
      <tr>
        <td>
          <p>Einkauf</p>
        </td>
        <td>
          <p>Beim klassischen Einkauf wird die Ware bei Bezahlung oder gegen Quittung übergeben. Die Ware wird sofort dem Bestand <a href="./begriffsdefinitionen.md#Eigenware">Eigenware</a> hinzugefügt.</p>
        </td>
      </tr>
      <tr>
        <td>
          <p>Verkauf</p>
        </td>
        <td>
          <p>Beim klassischen Verkauf wird die Ware bei Bezahlung übergeben. Die Rechnung wird ebenso zu diesem Zeitpunkt erstellt. Unabhängig von der Art der Bezahlung gilt hier immer:</p>
          <p>Die Ware wird sofort aus dem Bestand <a href="./begriffsdefinitionen.md#Eigenware">Eigenware</a> ausgebucht.</p>
        </td>
      </tr>
    </tbody>
  </table>
</div>

      <tr>
        <td colspan="2">
          <p><strong>Voreinkauf</strong></p>
        </td>
      </tr>
      <tr>
        <td>
          <p>Voreinkauf</p>
        </td>
        <td>
          <p>Ein Voreinkauf wird getätigt, wenn z.B. der Preis günstig ist, die Anlieferung jedoch aus Kapazitäts-, logistischen oder anderen Gründen erst später erfolgt. Bei einem Voreinkauf wird Ware eigenes Eigentum. Es liegt jedoch bis zur Anlieferung auf dem <a href="./begriffsdefinitionen.md#Fremdlager">Fremdlager</a>.</p>
        </td>
      </tr>
      <tr>
        <td>
          <p>Voreinkauf Anlieferung</p>
        </td>
        <td>
          <p>Bei der Anlieferung des Vorverkaufs wird die bereits bezahlte Ware vom fremden Lager zum eigenen Lager transportiert. Die Ware wird also vom <a href="./begriffsdefinitionen.md#Fremdlager">Fremdlager</a> zur <a href="./begriffsdefinitionen.md#Eigenware">Eigenware</a> transferiert.</p>
        </td>
      </tr>
      <tr>
        <td>
          <p>Voreinkauf Rückabwicklung</p>
        </td>
        <td></td>
      </tr>

<div class="table-wrapper">

      <tr>
        <td colspan="2">
          <p><strong>Vorverkauf</strong></p>
        </td>
      </tr>
      <tr>
        <td>
          <p>Vorverkauf</p>
        </td>
        <td>
          <p>Bei einem Vorverkauf bezahlt der Kunde seine Ware sofort, holt diese jedoch erst zu einem späteren Zeitpunkt ab (Gegenstück zu Voreinkauf). Die Ware ist bereits Eigentum des Dritten, liegt jedoch im <a href="./begriffsdefinitionen.md#EigenesLager">eigenen Lager</a> und wird als <a href="./begriffsdefinitionen.md#Fremdware">Fremdware</a> deklariert.</p>
        </td>
      </tr>
      <tr>
        <td>
          <p>Vorverkauf Abholung</p>
        </td>
        <td>
          <p>Der Kunde holt seine bereits bezahlte Ware ab. Diese wird also aus dem <a href="./begriffsdefinitionen.md#Fremdware">Fremdware</a>-Bestand ausgebucht.</p>
        </td>
      </tr>
      <tr>
        <td>
          <p>Vorverkauf Rückabwicklung</p>
        </td>
        <td></td>

<div class="table-wrapper">
  <table>
    <tbody>
      <tr>
        <td colspan="2">
          <p><strong>Einlagerung</strong></p>
        </td>
      </tr>
      <tr>
        <td>
          <p>Einlagerung</p>
        </td>
        <td>
          <p>Bei der Einlagerung wird die Ware eines Dritten in das <a href="./begriffsdefinitionen.md#EigenesLager">eigene Lager</a> eingelagert. Die Ware wird als <a href="./begriffsdefinitionen.md#Fremdware">Fremdware</a> deklariert.</p>
        </td>
      </tr>
      <tr>
        <td>
          <p>Einlagerung Abholung</p>
        </td>
        <td>
          <p>Wird die Ware, die ein Dritter im Lager eingelagert hatte wieder abgeholt, so wird diese aus dem <a href="./begriffsdefinitionen.md#Fremdware">Fremdware</a>bestand ausgebucht.</p>
        </td>
      </tr>
      <tr>
        <td>
          <p>Einlagerung Vereinnahmung</p>
        </td>
        <td>
          <p>Im Bereich Rohware kommt es zuweilen vor, dass eingelagerte Ware zu einem späteren Zeitpunkt aufgekauft wird. Diesen Vorgang nennt man dann Vereinnahmung. Dabei wird der Bestand von <a href="./begriffsdefinitionen.md#Fremdware">Fremdware</a> auf <a href="./begriffsdefinitionen.md#Eigenware">Eigenware</a> umgebucht. Die Ware muss dabei u.U. physikalisch nicht einmal bewegt werden.</p>
        </td>
      </tr>
    </tbody>
  </table>
</div>

<div class="table-wrapper">
  <table>
    <tbody>
      <tr>
        <td colspan="2">
          <p><strong>Kommission</strong></p>
        </td>
      </tr>
      <tr>
        <td>
          <p>Kommission</p>
        </td>
        <td>
          <p>Bei der Kommission wird Ware einem Dritten u.U. zum Zwecke des Verkaufs anvertraut. Diese bleibt <a href="./begriffsdefinitionen.md#EigeneWare">eigenes Eigentum</a>, wird jedoch nicht im <a href="./begriffsdefinitionen.md#EigenesLager">eigenen Lager</a> bzw. in der <a href="./begriffsdefinitionen.md#Eigenware">Eigenware</a> geführt, sondern im <a href="./begriffsdefinitionen.md#Fremdlager">Fremdlager</a>.</p>
        </td>
      </tr>
      <tr>
        <td>
          <p>Kommission Verkauf</p>
        </td>
        <td>
          <p>Ist vereinbart, dass der Dritte die Ware verkaufen darf, so ist dieser verpflichtet über diesen Verkauf zu informieren. In diesem Fall wird die Ware aus dem <a href="./begriffsdefinitionen.md#Fremdlager">Fremdlager</a>-Konto ausgebucht und der Verkäufer erhält eine Rechnung über die Ware.</p>
        </td>
      </tr>
      <tr>
        <td>
          <p>Kommission Rücknahme</p>
        </td>
        <td>
          <p>Je nach Hintergrund des Geschäfts wird hier eine ausgelagerte oder zur Kommission überlassene Ware zurückgenommen und vom <a href="./begriffsdefinitionen.md#Fremdlager">Fremdlager</a> in die <a href="./begriffsdefinitionen.md#Eigenware">Eigenware</a> und damit auch ins <a href="./begriffsdefinitionen.md#EigenesLager">eigene Lager</a> übernommen.</p>
        </td>
      </tr>
    </tbody>
  </table>
</div>
