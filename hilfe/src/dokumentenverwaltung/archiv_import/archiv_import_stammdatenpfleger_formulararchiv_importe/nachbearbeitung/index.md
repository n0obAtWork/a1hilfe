# Nachbearbeitung

<!-- source: https://amic.de/hilfe/_nachbearbeitung.htm -->

Auf Grund der möglichen Komplexität der vorliegenden Daten kann es durchaus Fälle geben, in dem das sehr flexible System der regulären Ausdrücke nicht im ersten Schritt reicht, um die gewünschte Daten-Repräsentation und damit unmittelbar das Kern-Datum zu gewinnen. Deshalb lässt sich jedes Ergebnis noch mal durch einfache reguläre Methoden nach- bzw. aufbereiten.

Hierbei ist es möglich, störende Leerzeichen zu eliminieren, Umstellungen und Umformatierungen zu erreichen. Die Beispiele sollen hinreichend Anschauungsmaterial bieten.
