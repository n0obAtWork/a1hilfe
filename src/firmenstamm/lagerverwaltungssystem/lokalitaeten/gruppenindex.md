# Gruppenindex

<!-- source: https://amic.de/hilfe/_lokalittsgrpidx.htm -->

Legt die Position der Lokalität in der Gruppe fest. Diese Einrichtung ist optional. Es dürfen mehrere Lokalitäten mit Index 0, aber nur jeweils eine mit einem Index > 0 existieren.

Nur Indizes >1 haben einen Effekt auf die Belegung mit Gruppen.

Siehe auch [Anwendung von Lokalitätengruppen](../anwendung_von_lokalitaetengruppe.md)
