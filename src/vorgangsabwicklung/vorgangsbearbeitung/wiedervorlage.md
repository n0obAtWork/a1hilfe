# Wiedervorlage

<!-- source: https://amic.de/hilfe/wiedervorlage.htm -->

Hiermit kann einem ausgewählten Vorgang ein Wiedervorlagevermerk für ein Datum mit einer Bemerkung gegeben werden. Dieser Vorgang erscheint dann automatisch zu diesem Datum beim Programmstart in einem Auswahlfenster. Dieses Fenster kann auch jederzeit mit dem Direktsprung WIEDV oder über die Funktion ***Wiedervorlage bearbeiten*** aufgerufen werden.
