# Aktivität des Mandantenserver

<!-- source: https://amic.de/hilfe/_mandserinfo.htm -->

Hauptmenü > Systempflege > Mandantenserver > Mandantenserver-Info

Direktsprung **[MSI]**

| Dialog-Felder | Mandantenserver |
| --- | --- |
| Zustand | Mögliche Zustände:<br>0 nicht aktiv<br>1 wartet<br>2 arbeitet<br>3 Fehler<br><br> |
| Arbeitsmodus | 0 Alle Aufträge<br>1 Ein Auftrag und Stopp<br>2 Pause<br>3 Stopp<br><br> |
| Anzahl Aufträge | Anzahl der noch zu bearbeitenden Aufträge<br><br> |
| Zähler | Wird mit jeder Aktivierung des MS gezählt<br><br> |
| ltz. Aktion | Zeitstempel der letzten Aktion<br><br> |
| Seit Serverstart | Überschrift<br><br> |
| Erledigte Aufträge | Anzahl erledigter Aufträge seit Serverstart<br><br> |
| Fehlerhafte Aufträge | Anzahl fehlerhafter Aufträge seit Serverstart<br><br> |
| insgesamt | Überschrift<br><br> |
| Erledigte Aufträge | Anzahl erledigter Aufträge insgesamt<br><br> |
| Fehlerhafte Aufträge | Anzahl fehlerhafter Aufträge insgesamt<br><br> |
| Unvollständige Buchungen | Anzahl unvollständiger Buchungen insgesamt<br><br> |
| Automatik eingeschaltet | Der Status, ob das [Event](../../zusatzprogramme/events/index.md) für den Mandantenserver eingeschaltet ist. |
| Wiederholungsintervall | Das Zeitintervall des [Events](../../zusatzprogramme/events/index.md) in dem geprüft wird, ob der Mandantenserver noch aktiv ist.<br><br> |
| Zähler | Wird mit jeder Aktivierung des MS gezählt<br><br> |
| Verarbeitungsroutine | Verarbeitungsroutine des Mandantenservers<br><br> |

| Dialog-Funktionen | Mandantenserver |
| --- | --- |
| Neu Anzeigen | Aktualisiert die Anzeige (refresh)<br><br> |
| Daueranzeige | Funktion zur Dauerhaften Anzeige<br><br> |
| Statistik zurücksetzen | Setzt die Statistik zurück<br><br> |
| unvollständige Aufträge zurücksetzen | Setzt die Unvollständigen Aufträge zurück<br><br> |
| Auf Pause stellen | Pausiert den Mandantenserver‘<br><br> |
| Einzelschritt | Einen einzelnen Schritt durchführen<br><br> |
| Normale Bearbeitung | Stellt den Mandantenserver auf normale Bearbeitung<br><br> |
| Stop Mandantenserver | Stopp den Mandantenserver<br><br> |
| Event Manager | Ruft die Auswahlliste aller [Events](../../zusatzprogramme/events/index.md) auf.<br><br> |
| Event ändern | Ruft den Dialog zum Ändern des [Events](../../zusatzprogramme/events/index.md) auf.<br><br> |
| Event Zeitplan ändern | Ruft den Dialog zum Ändern des Zeitintervalls für den [Event](../../zusatzprogramme/events/index.md) auf.<br><br> |
| Automatik deaktivieren | Deaktiviert den Event für den Mandantenserver.<br><br> |
| Automatik aktivieren | Aktiviert den Event für den Mandantenserver.<br><br> |
